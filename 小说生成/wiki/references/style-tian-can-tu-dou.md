## 二、结构DNA（structure_dna）

### 2.1 开篇模式：bridge_then_push

**含义**：先建立人物关系/情感连接（桥），再推进冲突/升级（推）。

**四部作品均采用此模式**：
- 《斗破苍穹》cliffhanger_ratio: 0.35
- 《大主宰》cliffhanger_ratio: 0.35
- 《元尊》cliffhanger_ratio: 0.35
- 《万相之王》cliffhanger_ratio: 0.25（略低，因连载中）

**操作含义**：
- 章开头先写人物互动/情感张力
- 再推进到核心冲突/战斗/升级事件
- 结尾留钩子，33%的章节以悬念结尾

### 2.2 信息揭示：layered_reveal

**含义**：分层揭示，不是开篇把设定全部抛出，而是随着剧情推进逐步揭露。

**操作含义**：
- 世界观设定不写进开篇介绍
- 通过角色视角/对话/冲突逐步带出设定
- 每章节只揭露1-2个新设定元素

## 三、节奏引擎（pacing_engine）

### 3.1 章节字数统计

| 作品 | 平均字数 | 最低 | 最高 | 对话占比 | 小高潮间隔 |
|------|---------|------|------|---------|----------|
| **斗破苍穹** | 2828.7 | - | - | 24.4% | 2章 |
| **大主宰** | 3304.7 | - | - | 16.5% | 2.67章 |
| **元尊** | 2950.2 | - | - | 20.7% | 5章 |
| **万相之王** | 3824.5 | - | - | 22.5% | 9.5章 |
| **四部均值** | 3178.6 | 313 | 6467 | 20.9% | 4.41章 |

**实用参考**：
- 章节字数目标：**2800-3300字**（主力区间）
- 可接受范围：313字（极短章）到6467字（超长章）
- 但80%的章节应落在2500-4000字区间

### 3.2 节奏核心规则

**小高潮间隔：4-8章**

即：每4-8章必须有一个让读者"爽到"的情节点。

**各作品节奏风格差异**：
- 斗破苍穹：节奏最快，**每2章一个小高潮**，战斗场景密集
- 大主宰：节奏快，2.67章一高潮
- 元尊：节奏中等，5章一高潮
- 万相之王：节奏较慢，9.5章一高潮（但连载中，可能调整）

**对写作的启示**：
- 如果写"打脸爽"，间隔不超过**2-3章**就要出现一次
- 如果写"传承爽/升级爽"，可以拉长到**5-8章**
- 超长待机（10章以上无爽点）会让读者流失

## 四、角色公式（character_formula）

### 4.1 主角模板

**central focalizer with escalating agency**

核心特征：
- **视角集中**：主要用主角视角叙事，其他角色通过主角的眼睛呈现
- **掌控力递增**：主角从被动应对→主动出击→掌控全局，掌控力随剧情推进不断升级
- **金手指存在但不开挂过度**：金手指提供能力，但主角需要努力才能发挥

### 4.2 配角模式

**functional layered cast**

特征：
- 每个配角都有**功能定位**（对手/导师/爱人/小弟/情报源）
- 配角出场服务于特定剧情目的
- 配角层级分明：核心配角→功能配角→一次性配角

### 4.3 高频角色词频（语言指纹）

**Top 10 角色提及**：
| 角色 | 提及次数 | 来自作品 |
|------|---------|---------|
| 牧尘 | 178.0 | 大主宰 |
| 周元 | 176.0 | 元尊 |
| 李洛 | 152.6 | 万相之王 |
| 萧炎 | 142.0 | 斗破苍穹 |

**高频动作词**：
| 动作 | 频次 | 说明 |
|------|------|------|
| 笑 | 636.3 | 最多——角色以"笑"表达复杂情绪（冷笑/苦笑/嘲讽/自信） |
| 说 | 432.7 | 第二——对话驱动叙事 |
| 沉默 | 35.7 | 远低于"笑"，说明天蚕土豆不喜欢让角色"沉默思考" |

**写作启示**：
- 多用"笑"代替"想/认为/感到"
- 对话是叙事主力（对话占比20.9%）
- 不要写大段角色内心独白——用对话和动作替代

## 五、语言指纹（language_fingerprint）

### 5.1 句子统计

| 指标 | 值 |
|------|----|
| 平均句长 | 178.96字 |
| 句子画像 | medium_long（中长句为主） |
| 对话占比 | 20.9% |
| 具体/抽象比 | 84.224 |

**解读**：每100个词里有84个是具体词汇（人名/地名/动作/物品），只有16个是抽象词汇（概念/情感/修饰语）。这是天蚕土豆"具体感"极强的底层原因。

### 5.2 高频词组（Top bigrams）

| 排名 | 词组 | 频次 | 性质 |
|------|------|------|------|
| 1 | 牧尘 | 609 | 角色名 |
| 2 | 周元 | 581 | 角色名 |
| 3 | 了一 | 521.2 | 助词/完成体 |
| 4 | 萧炎 | 519 | 角色名 |
| 5 | 也是 | 486.1 | 转折连接 |
| 6 | 李洛 | 449.4 | 角色名 |
| 7 | 有些 | 400.5 | 弱转折/委婉 |
| 8 | 不过 | 374.1 | 转折连接 |
| 9 | 能够 | 288 | 能力描述 |
| 10 | 有着 | 276 | 状态描述 |

**高频语言特征**：
- `"也是"` `"不过"` 高频 → 喜欢用弱转折，不用强转折（不用"但是"）
- `"有些"` 高频 → 语气克制，不把话说满
- `"能够"` `"有着"` 高频 → 客观陈述能力/状态，不加主观评价

### 5.3 高频三元组（Top trigrams）

| 词组 | 频次 | 说明 |
|------|------|------|
| 姜青娥 | 122.5 | 角色名 |
| 北灵院 | 113.0 | 地点名 |
| 唐芊儿 | 111.0 | 角色名 |
| 洛岚府 | 109.9 | 势力名 |
| 斗之气 | 94.0 | 修炼等级术语 |
| 苏幼微 | 76.0 | 角色名 |

**解读**：天蚕土豆的世界观用**具体名词**构建，角色名/地名/势力名/修炼术语构成语言骨架。

### 5.4 对话挂接原则

**dialogue_grounding_bias: high**

含义：对话必须有动作/环境承接，不能悬浮在半空中。

**反面教材**：
```
"今天的事你不要管。"
"我一定要管。"
```

**正确写法**：
```
"今天的事你不要管。"
他背过身去，手按在门框上。

"我一定要管。"
她跟了两步，在门槛前站住了。
```

## 六、世界框架（world_framework）

### 6.1 揭示模式：relationship_first

**含义**：先让读者知道人物之间的关系（谁是谁的对手/爱人/师父），再慢慢揭示他们所在世界的规则。

**与普通玄幻作者的区别**：
- 普通作者：先写世界观设定，再写人物
- 天蚕土豆：先建立人物关系，再通过关系冲突带出世界观

### 6.2 关键词分布

| 类型 | 频次 | 说明 |
|------|------|------|
| 关系词（relation_terms） | 2119.2 | 最多——人物关系是核心 |
| 战斗词（battle_terms） | 594.1 | 第二——战斗是升级载体 |
| 神秘词（mystery_terms） | 28.4 | 很少——不过度依赖悬念 |

**写作启示**：
- 人物关系是**第一驱动力**，不是世界观设定
- 战斗是**第二驱动力**，是解决关系冲突的手段
- 神秘/悬念不是核心手段，不依赖"挖坑-填坑"模式

### 6.3 世界观构建风格

**progressive layered reveal（渐进分层揭示）**

- 开篇不抛出完整世界观
- 设定随剧情推进逐步分层揭露
- 每卷只增加1-2个新设定层

## 七、情绪引擎（emotional_engine）

### 7.1 核心模式

**pain_delivery_bias: body_and_relationship**

情绪/虐点/爆点主要通过两个维度传递：
1. **身体反应**（疼痛/伤势/身体损伤）
2. **人际关系压力**（被轻视/被背叛/关系中的无力感）

### 7.2 具体表现

| 情绪类型 | 落地方式 | 示例 |
|---------|---------|------|
| 愤怒 | 身体反应：握拳/咬牙/发抖 | 他握紧了拳头，关节发白 |
| 屈辱 | 人际：被当众羞辱/被轻视 | 众人哄笑，他站在那里像根木头 |
| 悲伤 | 身体反应：眼泪/哽咽/说不出话 | 嘴唇动了动，什么声音都没发出来 |
| 复仇决心 | 身体+关系：咬牙+仇恨眼神 | 他盯着那人的背影，眼底像有火在烧 |
| 爽感 | 身体反应：拳头击中/对方倒地 | 那一拳正中面门，鼻梁骨断裂的声音清晰可闻 |

### 7.3 与普通网文的区别

| 普通网文 | 天蚕土豆风格 |
|---------|------------|
| 用心理描写传达情绪 | 用身体反应+关系压力传达 |
| "他心中怒火中烧" | "他一拳砸在墙上，指节裂开，血顺着手背流下来" |
| "她感到无比委屈" | "她的眼泪在眼眶里打转，硬是没掉下来" |
| "他对这个人恨之入骨" | "他看着那人笑，喉咙里像堵了一块石头" |

## 八、负面规则（negative_rules）——踩坑警示

> 以下是风格蒸馏中发现的"天蚕土豆不这样写"的清单：

### 8.1 不要简化成口癖模仿

❌ 错误：刻意模仿"莫欺少年穷"等经典台词，反复使用相同句式
✅ 正确：学习内核（关系驱动+身体反应），不模仿表面句式

### 8.2 不要写百科全书式设定

✅ 正确：渐进分层揭示，每卷只增加1-2个新设定层
❌ 错误：开篇用大段落介绍世界观规则（天蚕土豆从不在章节开头写设定）

### 8.3 不要让对话脱离动作/环境

❌ 错误：长对话块没有动作承接，像剧本
✅ 正确：每3-5句对话必须有一个动作/表情/环境描写穿插

### 8.4 不要写悬浮对话

❌ 错误：
```
"你走吧。"
"我不想走。"
```
✅ 正确：
```
"你走吧。"
他转过身，把门推开了。

"我不想走。"
她站在原地没动，脚尖在地上磨了磨。
```

# AI辅助网文写作

## 背景

2024-2025年，AI写作工具经历了质的飞跃：
- **2024年7月**：AI只能辅助扩写、润色
- **2024年底**：AI能直接写正文
- **2025年初**：AI能写大纲了
- **现在**：有经验的作者使用AI工作流能直出短篇小说，一篇稿费300-500元

AI是**加速器**，不是替代品。真正好的作品依然需要人的判断、情感、独特见解。

---

## 主流AI写作工具

### 通用对话AI
| 工具 | 优势 | 劣势 |
|------|------|------|
| **ChatGPT** | 创意发想、自由发挥、语言流畅 | 中文自然度稍弱、容易"幻觉" |
| **Claude** | 长上下文处理、结构化写作、逻辑连贯 | 中文自然度不如英文 |
| **Gemini** | 整合Google生态、搜索能力强 | 写作深度有限 |
| **DeepSeek** | 中文优化、代码+写作兼顾 | 生态较弱 |

### 专业写作工具
- **Notion AI**：整理资料、扩写笔记、生成待办，弹性有限，不适合直接生成正文
- **Jasper AI**：可设定品牌语气，适合商业文案
- **Copy.ai**：营销文案、社交媒体文案

## AI工作流（网文写作）

### 阶段一：构思与大纲

**使用AI**：
- 用AI做"头脑风暴"——让AI列出10种开局可能、20种情节走向
- 用AI评估大纲的"爽点密度"和"伏笔埋伏"
- 用AI生成"人物设定卡"（性格、动机、弱点）

**提示词示例**：
```
我要写一个[具体流派，如：穿越+修仙]小说，主角性格是[具体描述]，请帮我设计：
1. 三种不同风格的开局方案（各100字）
2. 主角前10章的核心目标
3. 可能的伏笔埋设点（至少3个）
4. 与主角对立的第一个主要势力
```

### 阶段二：细纲与章纲

**使用AI**：
- 将每章细纲拆解成"起承转合"四步
- 让AI为每章生成2-3个具体场景描写
- 让AI检查伏笔是否在预定章节回收

**提示词示例**：
```
第3章细纲：
- 起点：主角在集市被发现有修炼天赋
- 转折：被[势力]看中，邀请入门
- 高潮：主角面临选择：加入or拒绝
- 结尾：主角决定[具体选择]

请将以上细纲扩写成200字的章纲，包含具体的对话和场景描写建议。
```

### 阶段三：正文写作

**AI辅助模式**：
- **扩写模式**：给出关键情节点，AI填充具体描写
- **润色模式**：作者写初稿，AI优化语言
- **补全模式**：作者写70%，AI补全30%（细节、过渡）

**重要原则**：
- AI写初稿，人工审核→修改→定稿
- 不要完全依赖AI写正文（容易同质化）
- AI擅长描写"套路化"场景，核心情节需要人写

### 阶段四：卡文急救

**AI卡文急救提示词**：
```
主角当前处境：[具体描述]
已有资源：[能力/道具/盟友]
已有冲突：[当前面临的问题]
已完成伏笔：[之前埋下的伏笔]

请给出5种可能的"破局方案"，每种方案要包含：剧情走向、可能风险、爽感评估（1-5星）。
```

## AI在网文各环节的可用性评估

| 环节 | AI可用性 | 风险 | 建议 |
|------|---------|------|------|
| **构思/头脑风暴** | ⭐⭐⭐⭐⭐ 极适合 | 低（发散思维） | 大胆用 |
| **大纲制定** | ⭐⭐⭐⭐ 适合 | 中（需人工判断） | AI出方案，人选 |
| **章纲细写** | ⭐⭐⭐⭐ 适合 | 中（套路化） | AI辅助，人工把关 |
| **正文初稿** | ⭐⭐⭐ 可用 | 高（容易同质） | 限用扩写/补全 |
| **升级体系设定** | ⭐⭐⭐⭐ 适合 | 低（架构性工作） | AI可胜任 |
| **配角设定** | ⭐⭐⭐⭐ 适合 | 中（需人设深度） | AI生成，人改 |
| **打斗场面** | ⭐⭐ 可用 | 高（容易套路） | 人工核心框架 |
| **感情线** | ⭐⭐ 可用 | 高（情感不自然） | 谨慎使用 |
| **结尾/高潮** | ⭐⭐ 可用 | 高（同质化） | 人工主导 |

## AI辅助的常见陷阱

### 1. 幻觉问题
AI会编造不存在的"历史事件/术语/人物"，尤其在涉及具体知识时。
**应对**：所有AI生成内容必须人工审核事实性

### 2. 同质化陷阱
所有作者用同一套AI提示词，会产出相同的套路内容。
**应对**：发展个人专属提示词库，保持独特风格

### 3. 过度依赖
"照单全收"会让内容缺乏个人风格，变得平庸。
**应对**：AI是助手，不是主角。人的判断不可替代。

### 4. 深度不够
AI擅长表面化描写，缺乏对人性/情感的深层刻画。
**应对**：核心人物弧光、情感高潮必须人工写

## AI提示词设计原则

1. **具体>模糊**：给出越具体的指令，AI输出质量越高
2. **分步>一步到位**：先让AI做头脑风暴，再一步步细化
3. **角色扮演**：让AI扮演"资深网文编辑"或"同类作家"，产出更符合行业标准
4. **限制条件**：明确告诉AI"不要写什么"比"写什么"更重要
5. **迭代优化**：AI初稿→人工反馈→AI修改，多次迭代

**示例——角色扮演提示词**：
```
你是一位有10年经验的网文作家，擅长[具体流派]，熟悉起点/番茄平台的读者偏好。
请用"编辑视角"评价以下开局：
[粘贴开局文本]

重点评估：
1. 开头钩子是否足够强（1-10分）
2. 主角人设是否有魅力（1-10分）
3. 金手指是否清晰有趣（1-10分）
4. 是否有降低读者留存的风险点
```

## 行业现状（2025年）

- **全职AI写作工作室**：已出现，专门用AI批量生产短篇小说
- **个人作者+AI工作流**：月入数万的案例开始出现（主要在番茄等免费平台）
- **平台态度**：番茄等平台对AI内容暂无明确限制，但质量要求不变
- **稿费天花板**：有经验作者用AI能提高产能，但爆款依然需要人的创意

---

## 总结

AI是**强大的助手**，能让写作更高效：
- 构思阶段：大量借助AI发散思维
- 写作阶段：AI做扩写/润色，人做核心决策
- 修改阶段：AI做重复性检查，人工做深度修改

**不可替代**：对读者的理解、对故事的把控、独特的个人风格、情感共鸣的深度。

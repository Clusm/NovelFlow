# 网文描写词汇速查

> 写作时随时查阅的描写词汇与技巧速查手册，覆盖外貌、行为、场景、心理、打斗、环境六大类。

## 一、人物外貌

### 女性外貌

**眉眼**：蛾眉螓首、柳眉星眼、明眸善睐、眉如远黛、凤眸流转、眉清目秀、眼含秋水、眉蹙春山、眼波潋滟、眸若寒星

**唇鼻**：唇红齿白、鼻若悬胆、樱唇皓齿、唇若点绛、鼻梁挺翘

**面容**：面若凝脂、肌肤胜雪、容光焕发、冰肌玉骨、面若桃花

**发型**：青丝如瀑、云鬓高耸、玉簪绾发、流苏垂肩

**气质**：蕙质兰心、秀外慧中、雍容华贵、亭亭玉立、风华绝代

### 男性外貌

**眉眼**：剑眉星目、朗目疏眉、眉峰凛冽、眸光如炬、双眉入鬓、眸若朗星

**面容**：面如冠玉、仪表堂堂、轮廓分明、气宇轩昂、风度翩翩

**发型服饰**：束发高冠、青衫磊落、锦袍玉带、劲装窄袖、玄衣肃杀

**气质**：温文尔雅、冷峻内敛、沉稳大气、狂放不羁、谦谦君子

### 反派外貌

**眉眼**：鼠目寸光、眼神阴鸷、眼露凶光、眸若蛇蝎、眼含戾气

**面容**：满脸横肉、尖嘴猴腮、面目可憎、面色蜡黄、狰狞可怖

**气质**：阴险狡诈、凶神恶煞、尖酸刻薄、笑里藏刀、蛇蝎心肠

## 二、行为动作（替代直白对话）

核心技巧：**少用「他很生气/她很难过」，用动作+神态代替情绪直白表述。**

| 情绪 | 描写词汇 |
|------|---------|
| 愤怒 | 攥紧拳头、指节泛白、柳眉倒竖、拍案而起、咬牙切齿、青筋暴起、周身冷意蔓延 |
| 开心 | 眉开眼笑、喜上眉梢、嘴角上扬、眼底含笑、脚步轻快、眉眼弯弯 |
| 难过 | 泪眼婆娑、垂首啜泣、肩膀颤抖、指尖冰凉、沉默垂眸、声音哽咽 |
| 犹豫 | 眉头紧锁、搓手踱步、咬唇沉思、欲言又止、眼神闪烁、支支吾吾 |
| 恐惧 | 瞳孔骤缩、后背发凉、手脚发软、呼吸急促、下意识后退 |

## 三、场景环境

### 古风场景

**庭院**：青瓦白墙、曲径通幽、竹影婆娑、荷塘月色、假山叠翠
**听觉**：流水潺潺、风吹竹叶、琴音悠扬、蝉鸣清脆
**嗅觉**：花香袭人、茶香袅袅、檀香氤氲、草木清香

**宫殿**：金碧辉煌、琉璃泛光、盘龙玉柱、宫灯高悬、帘幔低垂

### 都市场景

**街道**：车水马龙、霓虹闪烁、高楼林立、行人匆匆、夜色璀璨
**室内**：暖黄灯光、木质桌椅、咖啡醇香、玻璃窗凝雾

### 玄幻场景

**秘境**：灵气缭绕、奇花发光、云雾缭绕、霞光万丈、古阵符文
**魔界**：暗红天幕、魔气冲天、岩浆翻滚、骸骨遍地、鬼火幽幽

**通用公式**：视觉 + 听觉 + 嗅觉 + 氛围渲染，四维结合。

## 四、心理描写

### 正向情绪

| 情绪 | 词汇 |
|------|------|
| 开心 | 心花怒放、喜不自胜、笑容明媚、呼吸轻快 |
| 激动 | 心潮澎湃、手心冒汗、心跳加速、指尖颤抖 |
| 安心 | 如释重负、眉头舒展、心神安定、周身放松 |
| 期待 | 翘首以盼、眼神发亮、身体前倾、满心希冀 |

### 负向情绪

| 情绪 | 词汇 |
|------|------|
| 愤怒 | 怒火中烧、怒不可遏、浑身紧绷、眼神冰冷 |
| 难过 | 心如刀割、黯然神伤、眼眶泛红、心口发闷 |
| 恐惧 | 胆战心惊、浑身僵硬、手脚冰凉、呼吸发紧 |
| 焦虑 | 坐立难安、心烦意乱、来回踱步、眉头紧蹙 |

### 复杂情绪

纠结、嫉妒、愧疚、释然、隐忍、心酸、无奈

**技巧**：内心想法 + 生理反应 + 细微动作，三重结合，情绪更立体。

## 六、打斗场面

### 动作招式

**攻击**：拳如流星、剑刺如电、刀劈如山、掌风凌厉
**防守**：闪身格挡、腾挪闪避、卸力招架、退守防御
**身法**：身轻如燕、闪转腾挪、步法如风、凌空翻转

### 战斗氛围

**打斗公式**：动作招式 + 环境破坏 + 人物神态 + 气场压迫，节奏快慢结合。

## 八、人物关系

### 亲密关系

亲情（舐犊情深、嘘寒问暖）、爱情（含情脉脉、耳鬓厮磨）、友情（肝胆相照、患难与共）

### 对立关系

敌人（针锋相对、仇深似海）、对手（棋逢对手、惺惺相惜）、陌生人（萍水相逢、疏离淡漠）

## 通用万能技巧

1. **拒绝直白抒情**：不写「他很伤心」，写「他垂下手，指尖泛凉，良久无声」
2. **五感写法**：视觉+听觉+嗅觉+触觉+味觉，任意搭配
3. **细节放大**：抓指尖、眼神、呼吸、眉梢、衣角、发丝等微小细节
4. **动静结合**：静景铺氛围，动景推剧情
5. **成语+短句组合**：高级词汇搭配短句，节奏舒服
6. **氛围感优先**：先铺环境氛围→再写人物动作→最后点情绪

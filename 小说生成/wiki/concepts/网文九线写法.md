# 网文九线写法

## 概述

"九线"是网文大纲写作的核心框架，指同时推进的九条叙事线。通过多线交织，让故事在各个维度同时展开，避免单线叙事的单调感，并将所有情节内容囊括其中。构思时由点到线、由线到面，如同一颗种子长成参天大树。

> **核心理念**：点子（种子）→ 主线（主干）→ 分线（分支）→ 面（整棵树）

---

## 九条线详解

### 一、主角智慧性格线（最重要的线）

主角的性格与智慧直接决定一本书的基调，是九线中权重最高的一条。

**设定要点：**

- 主角可以平凡，但不能窝囊
- 性格决定情节走向：热血主角遇强敌会硬撼；猥琐主角可能跪地求饶再伺机反击
- 能力与性格必须匹配：使暗器的主角必然阴险腹黑，没有人会站在敌人面前光明正大耍暗器
- 主角最终必须功成名就，否则就是虐主，读者不会喜欢

**典型案例：**

| 作品 | 主角性格 | 风格影响 |
|------|---------|---------|
| 《冒牌大英雄》 | 猥琐 | 通篇透着猥琐味，可以耍阴招、使用"猴子偷桃" |
| 《斗破苍穹》 | 热血 | 敢作敢为，绝不存在怕死或偷袭后背 |

**一句话总结**：定好主角性格，才能决定他遇事会怎么反应、故事会往什么方向发展。

---

### 二、配角炮灰线

配角和炮灰的数量远超伙伴，是直接推动情节发展的核心动力。三万字情节中未必出现一个伙伴，但必然出现数个甚至数十个配角炮灰。

**设定格式：**

```
姓名：XXX
性别/年龄：男，30岁
外貌：光头，脸上有刀疤
实力：九级剑圣
性格：残暴，喜欢嗜杀
身份：XX帝国元帅，主角中期遇到的强敌
```

**让配角立起来的两个技巧：**

1. **口头禅和习惯动作**：如《无限恐怖》中赵缀空的"小苹果"、尼奥斯爱吃巧克力
2. **人物故事和八卦**：如写一个残暴角色，可补充"三十年前他曾制造大屠杀，尸横遍野血流成河，婴儿听到他名字都不敢哭"

**核心原则**：不要踩路边跑龙套的小混混当敌人，配角炮灰也要有来历有背景。

---

### 三、技能线（力量体系）

技能线即力量体系的设定，分为主角技能和常人技能两套体系。

**设定框架：**

- **主角技能**：主角独有的能力，如"召唤热武器"
- **常人技能**：世界大多数人的能力，如"斗气魔法"
- 两者必须建立比对关系：手枪破坏力相当于几级魔法？导弹相当于几级禁咒？
- 设定职业等级体系：分为哪些职业？各自几个等级？
- 最好加入技能原理和来历，可以展开大量背景故事

**技能线拉开技巧：**

```
主线：得到功法 → 修炼功法 → 别的势力抢夺 → 功法背后的秘密 → 引来功法主人重视
```

**核心原则**：不要让功法得到后就没下文了，要反复利用、层层展开。

---

### 四、伙伴线

主角身边的伙伴/队友，与配角炮灰设定方式基本一致，但伙伴是正面力量。

**伙伴线的延展性：**
伙伴线是作者最青睐的开荒线，因为延展性最强。一个伙伴可以引出：
- 他背后的势力
- 他与主角的过往渊源
- 他带来的新任务或冲突
- 势力引荐（伙伴可能是帝国军官，引主角进入更高层圈子）

**注意事项：**
- 即便有伙伴帮忙，击杀敌人的功劳也要分给主角一部分，不能让主角打酱油
- 伙伴的出现要有戏剧性，通常在危机关头现身

---

### 五、装备线

主角的神器、法宝、宠物等。

**设定原则：**

1. **得到过程必须有惊险故事**：惊险才好看，来自不易才体现价值
2. **不要直接描写装备有多牛逼**：用传说、大师制造、多少人守护等侧面烘托
3. **避免用烂的桥段**：主角随便上街淘到极品这种事已经过时了

**核心原则**：你花费几千字直接描述装备牛逼，读者未必相信；你用侧面烘托（传说、难度、代价），读者反而能感受到装备的牛逼。

---

### 六、冒险线（副本/奇遇线）

独立的副本、秘境、任务等剧情线，每次冒险都是一次独立的小情节。

**冒险线的节奏安排：**

```
冒险线1（2-3万字）→ 冒险线2（2-3万字）→ 冒险线3（2-3万字）→ 主角升级 → 回归主线
```

**注意事项：**
- 不要连续开同类型冒险（如连续三波小混混），读者会审美疲劳
- 每次冒险要2-3条分线交织，不要单线推进
- 冒险线之间要穿插主角升级，让情节有张有弛

---

### 七、身世线

主角的背景设定，具有很强的延展性：

- **初期**：主角的保护伞（如大家族子弟的身份庇护）
- **中期**：牵扯出势力矛盾（如身世之谜引发派系斗争）
- **后期**：大高潮的支撑点（如身世背后隐藏星际战争真相）

**高级用法**：直接挖一个身世坑作为全书主线来写，如《冒牌大英雄》主角身世牵扯出星际战争最深层次的来历。

---

### 八、势力线（世界观骨架）

整个世界的势力构架设定，支撑一本书的骨架。

**设定格式：**

```
XX门派：
- 宗主：XXX（法神级别）
- 位置：XX帝国境内
- 势力：独霸一方
- 关系：与XX帝国互相勾结，打压XX门派
- 弟子：拥有上万弟子
- 与主角关系：主角最初的敌人
```

**势力线的重要性：**
- 一本书的情节本质是矛盾：国家矛盾、门派矛盾、势力矛盾
- 主角遇到的每个敌人必须有来历，不是街边随便踩的小混混
- 势力之间的博弈可以撑起大量剧情

---

### 九、后宫线（感情线）

主角身边的女性角色/感情线。

**后宫线的特点：**
- 最不好写，新手最不建议写
- 最难把握尺度，写多了容易被封
- 写好了却是大杀器，是拉读者的利器

**相对更容易掌握的开局线索：**
- 技能线：主角得到神奇功法，牵扯势力矛盾
- 装备线：主角获得神器，引出一系列故事
- 冒险线：主角进入秘境奇遇

## 九线展开的由点到面

### 构思四套路

| 套路 | 解释 | 案例 | 适合人群 |
|------|------|------|---------|
| 旧瓶装旧酒 | 跟风框架 | 无限流、随身流 | 新手 |
| 新瓶装旧酒 | 换框架不换故事 | 《召唤万岁》（召唤魔兽→召唤萝莉） | 思路活跃者 |
| 旧瓶装新酒 | 老框架新故事 | 《冒牌大英雄》（机甲古武+猥琐胖子） | 有经验的写手 |
| 新瓶装新酒 | 全新框架全新故事 | 《无限恐怖》《鬼吹灯》 | 天生大神 |

### 由点到线的操作

**步骤1**：从点子出发，拉出一条主线
```
点子：异界召唤热武器
主线：得到魔枪手功法 → 修炼功法 → 别的势力抢夺 → 功法背后的秘密 → 引来功法主人重视
```

**步骤2**：由主线拉出分线（2-3条，不要重复类型）
```
主线：得到功法
├── 分线1：配角炮灰线（从别人手里夺取）
├── 分线2：身世线（父母长辈留给主角）
└── 分线3：冒险线（奇遇、打守护者得宝）
```

**步骤3**：2-3条分线写完（约8-10万字）→ 主角升级 → 回归主线

### 由线到面的节奏

```
开局（1-3万字）：拉出主线 + 2-3条分线
中段（3-10万字）：分线展开、交织、主角成长
高潮（每个小节点）：矛盾激化、战斗、升级
收尾（10万字+）：回归大主线，揭示更深层秘密
```

**核心口诀**：
> 高潮就是矛盾发展到顶点。主角跟人吃饭，不如跟人打架吸引人。

## 九线管理工具

由于九线会产生大量伏笔，建议用表格管理：

| 线名 | 伏笔 | 埋入章节 | 回收章节 | 状态 |
|------|------|---------|---------|------|
| 身世线 | 主角父亲失踪 | 第3章 | 第89章 | 待回收 |
| 技能线 | 功法来历成谜 | 第5章 | 第120章 | 进行中 |
| 势力线 | 门派暗中监视 | 第7章 | - | 伏笔中 |

---

## 核心原则总结

1. **主线不断**：任何时候至少有一条线在推进
2. **线线交织**：不同线路在关键节点交叉，产生化学反应
3. **详略得当**：不同阶段不同线的重要性不同，该详写就详写，该略过就略过
4. **伏笔不乱**：九线并行会产生大量伏笔，必须用表格管理
5. **高潮=矛盾最大化**：不要让主角轻松取胜，戏剧性来自矛盾的激化
6. **由点及面**：从一个创意点逐步发散，最终形成丰满的作品

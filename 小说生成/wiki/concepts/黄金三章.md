# 黄金三章

## 基本概念

**定义**：网文前1%的内容（大约前三章），能筛选80%的读者。读者往往在前三章就决定是否继续追读。

**核心数据**：
- 90%的网文读者会在前3章决定是否弃文
- 留存率（前3章留住的读者）直接决定一本书的生死

## 第一章：钩子法则

### 必须解决的3个问题
1. **这是什么类型的书？** → 第一段就要让读者知道（都市/玄幻/穿越/系统……）
2. **主角是谁？** → 读者要能代入主角
3. **有什么看点？** → 悬念/冲突/金手指要让读者好奇

### 黄金开篇公式
```
[强冲突/悬念/金手指展示] + [主角当前处境] + [立刻产生兴趣的第一个钩子]
```

### 开篇禁忌
- ❌ 大段背景介绍
- ❌ 无关紧要的配角出场
- ❌ 主角没有明确性格
- ❌ 平淡无奇的日常

### 开篇类型推荐

| 类型 | 适用流派 | 开篇特征 |
|------|---------|---------|
| **金手指展示型** | 系统流/修仙流 | 开篇展示金手指的强大 |
| **强冲突型** | 废柴逆袭/都市 | 开篇就是打脸/危机/冲突 |
| **悬念型** | 悬疑/无限流 | 开篇抛出谜团/诡异事件 |
| **反差型** | 轻松搞笑/发疯文学 | 开篇有趣/反讽/出人意料 |

## 第二章：节奏建立

### 第二章的核心任务
1. **建立节奏模式**：让读者知道这本书的"爽点频率"
2. **深化第一天的钩子**：第一天埋下的钩子要有推进
3. **展示第一个小高潮**：第一个情绪释放点

### 节奏公式
```
第1章：新手村/第一天
第2章：冲突升级+小高潮
第3章：第一个阶段胜利/金手指展示+下一个小钩子
```

## 第三章：留存锚点

### 第三章的核心任务
1. **让读者知道自己会喜欢这本书**
2. **建立追读动力**（下一个钩子）
3. **展示本书的核心竞争力**

### 判断"是否留存"的标准
- 读者是否记住了主角的名字和性格
- 读者是否理解了本书的核心设定
- 读者是否对下一个钩子有期待

## 各平台黄金三章差异

### 起点（老白向）
- 可以稍微慢一点，但要有信息量
- 世界观要在前三章铺垫好
- 主角智商要在前三章体现
- **字数**：3000-5000字/章

### 番茄（小白向）
- 节奏更快，爽点更密集
- 前三章要有明确的"打脸"或"金手指展示"
- 情绪要立刻到位
- **字数**：2000-3000字/章

### 飞卢（高强度）
- 前3-10章决定追否
- 每一章都要有至少一个小高潮
- 开局即巅峰，要求更高
- **字数**：2000-3000字/章

## 常见问题

### Q: 没有存稿，前三章写不好怎么办？
**A**: 前三章要反复修改，直到满足：
1. 能用一句话概括前三章的看点
2. 朋友看完前三章想知道后续

### Q: 背景设定太多，开篇放不下怎么办？
**A**: 用**"冰山理论"**——只展示水面上的10%，90%通过情节逐步释放

### Q: 开篇不够"爽"怎么办？
**A**: 重新审视开篇：
1. 是否有明确的金手指？
2. 是否有冲突/压力？
3. 主角是否有明确的目标？

## 快速自检清单

- [ ] 第一段就知道这是什么类型的书

- [ ] 有一个让读者好奇的钩子

- [ ] 没有无关配角干扰
- [ ] 第一章有小高潮或悬念
- [ ] 第二章有冲突升级
- [ ] 第三章能让读者知道"这书我喜欢"

## 黄金三章细化指南

### 第一章：钩子法则（详解）

#### 开头前三行的重要性

读者决定是否继续读，通常只看**前三行**：

```
❌ 错误示范：
"清晨，阳光透过窗帘洒进房间，张三从床上醒来。他看了看时间，发现已经七点了。于是他起床洗漱，准备去上班..."

✓ 正确示范：
"张三睁开眼的瞬间，系统面板在眼前亮起：【检测到宿主，是否绑定？】"
```

#### 黄金第一句的公式

| 类型 | 公式 | 示例 |
|------|------|------|
| **系统流** | [系统出现] + [主角反应] | 眼前弹出一个半透明面板... |
| **冲突型** | [危机/冲突发生] | 房门被一脚踹开 |
| **悬念型** | [异常现象] | 镜子里的自己突然消失了 |
| **人设型** | [主角特质] | 林医生有个职业习惯：从不回头看病人 |

#### 第一章必须完成的5个任务

1. **类型明确**：让读者知道这是都市/玄幻/穿越/系统...
2. **主角代入**：读者要知道主角是谁、什么性格
3. **金手指展示**：让读者知道主角有什么"作弊器"
4. **第一个钩子**：留下一个让读者想继续看的问题
5. **情绪基调**：本书是燃/虐/爽/搞笑？要在第一章就定调

### 第二章：节奏建立（详解）

#### 第二章的核心公式

```
第2章 = 冲突升级 + 第一天钩子推进 + 小高潮 + 新钩子
```

#### 节奏类型对比

| 节奏类型 | 适用平台 | 第一章 | 第二章 |
|----------|----------|--------|--------|
| **番茄风** | 番茄/快节奏 | 建置+冲突 | 冲突升级+小爽 |
| **起点风** | 起点/老白 | 世界观铺垫 | 矛盾初现 |
| **飞卢风** | 飞卢/高强度 | 开局即高潮 | 持续高潮 |

#### 第二章常见错误

- ❌ 还在介绍世界观（节奏拖沓）
- ❌ 没有冲突推进（无聊）
- ❌ 第一天的钩子没有进展（虎头蛇尾）
- ❌ 开新坑太多（信息过载）

### 第三章：留存锚点（详解）

#### 第三章必须回答的问题

读者读完前三章，应该能回答：
- ✅ 这是什么类型的书（都市/玄幻/系统...）
- ✅ 主角是谁，我要不要跟TA
- ✅ 这本书有什么好看的（爽点/看点）
- ✅ 接下来会发生什么（追更动力）

#### 第三章的钩子设计

| 钩子类型 | 示例 | 效果 |
|----------|------|------|
| **选择式** | 主角面临留下还是离开的选择 | 想知道TA怎么选 |
| **危机式** | 更强大的敌人出现 | 想看主角怎么应对 |
| **身份式** | 主角身份即将被揭穿 | 想看打脸时刻 |
| **情感式** | 羁绊角色遭遇危机 | 想看后续情感发展 |

### 各平台详细差异

#### 起点（老白向）
- 字数：3000-5000字/章
- 特点：可以有慢热，但世界观要完整
- 黄金三章要求：
  - 世界观前3章铺好
  - 主角智商要体现
  - 第一个副本/冲突要精彩
  - 有深度设定吸引老白

#### 番茄（小白向）
- 字数：2000-3000字/章
- 特点：节奏快，情绪到位
- 黄金三章要求：
  - 前300字就要抓住读者
  - 每章至少一个小爽点
  - 情绪立刻到位
  - 适合改编短剧

#### 飞卢（高强度）
- 字数：2000-3000字/章
- 特点：开局即巅峰
- 黄金三章要求：
  - 前3-10章决定追否
  - 每章至少一个小高潮
  - 节奏要求极高
  - 同人/脑洞文优势大

#### 晋江（女频）
- 字数：2000-4000字/章
- 特点：感情线要甜/虐到位
- 黄金三章要求：
  - 感情线要有进展（暧昧/冲突）
  - 男主人设要立住
  - 情感钩子要强

## 黄金三章六要素（卡片A核心）

来自网文徐掌柜体系，前三章必须让读者清晰感知**六个要素**，缺一即不成立：

| # | 要素 | 说明 |
|---|------|------|
| 1 | **舞台搭建** | 开篇即给出世界信息（如"大奉京兆府监牢"） |
| 2 | **事件起因/今生延展性** | 原主遗留的驱动事件，非简单背景 |
| 3 | **合理逻辑目标** | 主角结合现状产生的驱动力 |
| 4 | **明确变量** | 穿越者身份 + 金手指 |
| 5 | **解题行进** | 朝目标推进的具体剧情 |
| 6 | **制造落差** | 有起伏才有后续爽点爆发 |

**致命错误**：
- 只堆世界观没有驱动事件
- 主角被动等剧情推，没有"今生延展性"提供的主动方向

### 睁眼穿标准架构

穿越文最经典开局：**主角睁眼即已身处异世界**。

| 要素 | 说明 |
|------|------|
| 直白切入 | 两句话 = 时间/地点/天气 + 主角正在什么情境 |
| 前世今生衔接 | 前世有用写前世，没用直接写今生 |
| 今生延展性（核心） | 原主遗留的驱动事件，为后续行为提供明确方向 |

### 四种开篇节点

| 节点类型 | 特点 | 说明 |
|----------|------|------|
| **危机节点** | 最紧凑 | 主角面临迫在眉睫的危险 |
| **转折节点** | 常见 | 穿越/觉醒/退婚等命运转折时刻 |
| **日常节点** | 节奏慢但自然 | 从日常切入，快速拉入异常 |
| **高潮前置节点** | 吸引力最强 | 先写精彩场景做钩子，再倒叙解释 |

## 实战案例分析

### 成功案例：《我在精神病院学斩神》

**前三章结构**：
- 第1章：世界观建立（神明复苏）+ 主角身份（精神病患者）+ 金手指暗示
- 第2章：第一个副本预告 + 伙伴出现 + 悬念埋设
- 第3章：第一个小高潮（战斗场面）+ 情感羁绊建立

**为什么成功**：
- 开篇建立独特世界观（神明复苏）
- 主角身份有记忆点（精神病患者）
- 群像戏在前三章就铺垫好
- 每章都有钩子

### 成功案例：《十日终焉》

**前三章结构**：
- 第1章：规则世界建立 + 第一个游戏开始
- 第2章：智斗过程 + 角色群像
- 第3章：游戏结束 + 更大的悬念

**为什么成功**：
- 规则明确，读者立刻理解世界
- 每场游戏都是独立悬念
- 智斗精彩，满足推理需求
- 结尾钩子强（十日循环真相）

## 前三章常见问题修复

### 问题1：前三章写了5000字还在铺垫

**症状**：读者已经划走，你还沉浸在世界观里

**修复**：
```
原来的：世界观→主角背景→主角现状→第一个冲突
修改为：冲突→背景（通过行动/对话带出）→第二个冲突
```

### 问题2：主角没有存在感

**症状**：旁白太多，主角像旁观者

**修复**：
```
减少旁白描述
增加主角的：
- 内心活动（心理描写）
- 主动选择（主角在做事）
- 与他人的互动
```

### 问题3：金手指太晚出现

**症状**：前三章没有金手指，读者不知道主角"特殊在哪"

**修复**：
```
金手指最晚在第一章结尾出现
或者：
开篇就是金手指带来的改变（系统绑定/重生回档）
```

### 问题4：配角太多太乱

**症状**：第一章出现10个名字，读者记不住

**修复**：
```
前三章配角控制在3-5人
每个配角要有独特记忆点（口头禅/外貌/性格）
其他配角后续逐步登场
```

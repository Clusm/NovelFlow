# 详细人设模板

> 来源：网文架构师/人设功/L3（v2.1），联网学习整理 | 调用：设计角色时对照填写

## 核心人设公式

```
主角魅力 = 初始身份 × 反差 × 核心执念
```

|| 要素 | 说明 | 示例 |
||------|------|------|
| **初始身份** | 故事起跑线+社会坐标 | 废柴/赘婿/退役王牌 |
| **反差** | 表面vs内心/能力vs性格 | 外表高冷总裁×私下怕黑粘人 |
| **核心执念** | 深层、持久、不动摇的欲念 | 复仇/守护家人/证明自己 |

### 经典执念清单

|| 类型 | 示例 |
||------|------|
| 创造型 | 创造/修复所拥有的"灵魂" |
| 守护型 | 守护特定的人/秩序/信念 |
| 证明型 | 向世界/某人证明自己 |
| 复仇型 | 向仇人讨回公道 |
| 探索型 | 寻找真相/身世/未知 |
| 反抗型 | 反抗命运/体制/强权 |

## 角色三维模型

### 生理层（外在呈现）

```
外貌描写：
├─ 视觉锚点：（1个核心记忆点，如眼尾痣、左手伤疤）
├─ 体型：高/矮/健壮/纤细/特殊
├─ 肤色：白/黄/古铜/苍白/有特征
├─ 眼神：锐利/温柔/空洞/深邃/特别
├─ 步态：挺拔/蹒跚/轻盈/沉重
├─ 服饰：华贵/朴素/特定风格
└─ 标志性装备：（武器/饰品/随身物）
```

### 社会层（身份定位）

```
社会身份：
├─ 明面身份：（公开的职业/地位）
├─ 隐藏身份：（便于打脸的真实身份）
├─ 出身阶层：（上流/中产/底层）
├─ 教育背景：（家学/学院/自学）
├─ 经济状况：（富裕/小康/贫穷）
├─ 社会评价：（名声/口碑/传说）
└─ 组织归属：（宗门/家族/势力）
```

### 心理层（内在驱动力）

```
心理层设计：
├─ 核心欲望：（最深层的渴望）
├─ 表层欲望：（说出口想要什么）
├─ 核心恐惧：（最害怕失去什么）
├─ 心理阴影：（童年创伤/遗憾/愧疚）
├─ 价值观：（人生观/世界观）
├─ 道德底线：（绝对不能触碰的线）
└─ 行为准则：（做什么事的原则）
```

### 欲望清单三问

|| 问题 | 模板答案 |
||------|---------|
| **表面想要什么？** | 角色说出口的目标 |
| **内心真正想要什么？** | 角色可能都不知道的深层渴望 |
| **两个欲望是否冲突？** | 冲突越大越精彩 |

## 角色功能设计原则

### 主角：功能性大于完美性

|| 原则 | 说明 |
||------|------|
| **不完美引出故事线** | 性格缺陷引出更多剧情 |
| **功能性必须有** | 金手指/独特能力是核心看点 |
| **成长空间** | 废柴逆袭比天才碾压更有代入感 |
| **读者代入感** | 让读者能在角色身上找到自己 |

### 配角：记忆点大于完整性

|| 原则 | 说明 |
||------|------|
| **工具人定位** | 给主角制造矛盾或解决矛盾 |
| **一个核心记忆点** | 嗜甜如命的暗卫/结巴的毒医 |
| **一句话能记住** | 不需要完整背景 |
| **独立人格** | 配角也要有自己故事线 |

### 反派：破坏力大于邪恶值

|| 原则 | 说明 |
||------|------|
| **必须有实力** | 再坏但菜也当不了反派 |
| **制造压力** | 悬在主角头顶的利剑 |
| **不纯邪恶** | 有苦衷/被控制/心魔缠身 |
| **立体动机** | 反派视角也有合理性 |

## 人设描写公式

### 天骄型（傲慢天才）

```
【视觉锚点】
华丽装备特写 + 与年龄不符的松弛感

【数据标签】
年龄（必须小）+ 境界（必须高）+ 战绩（必须傲人）

【气息描写】
不收敛的威压 + 环境扭曲

【动作特征】
慢动作（表示游刃有余）+ 俯视视角
```

# AI辅助写作

## 核心概念

AI辅助写作 = 用结构化提示词引导AI参与写作过程，不是让AI直接生成完整作品，而是让AI在特定环节提供支持。

## 章节AI指令模板（[[单章ai指令模板.md]]）

### 时空锚点
- **本章角色**：列出所有出场角色，标注当前情绪状态
- **时间地点**：具体时间 + 具体场景（含细节描写）
- **上章概要**：1-2句话，聚焦上章结尾的悬念或角色状态
- **下章概要**：1句话，指出本章需要铺垫的关键转折
- **本章核心目标**：用"动词+名词"明确，如"实现擂台反杀爽点"

### AI辅助写作的适用场景

1. **卡文时** — 提供情节走向建议
2. **打斗场景** — 补充战斗细节
3. **世界观描写** — 丰富场景细节
4. **润色** — 优化文字节奏

### AI辅助写作的不适用场景

1. **核心情节设计** — 需要作者本人的创意和判断
2. **人物核心对话** — 决定人物性格的关键台词
3. **整体大纲** — 故事走向必须作者把控

## AI提示词设计要点

参考 [[AI提示词.md]] 中的30种流派提示词：

- **核心机制** — 明确流派的核心爽点
- **关键词** — 触发特定叙事模式的关键词
- **禁忌** — 明确什么写法会破坏爽感

## 提示词设计进阶技巧

### 1. 角色扮演法
让AI扮演"资深网文编辑"或"同类作家"：

```
你是一位有10年经验的网文作家，擅长[具体流派]。
请用"编辑视角"评价以下开局：
[粘贴开局文本]

重点评估：
1. 开头钩子是否足够强（1-10分）
2. 主角人设是否有魅力（1-10分）
3. 金手指是否清晰有趣（1-10分）
4. 是否有降低读者留存的风险点
```

### 2. 思维链提示（Chain of Thought）
复杂场景分步生成：
1. 先让AI描述场景氛围
2. 再让AI列出关键动作
3. 最后让AI完成对话描写

### 3. 限制性指令
明确"不要写什么"比"写什么"更重要：
- 禁止出现：[具体内容]
- 禁止让[角色]做出[某种行为]
- 禁止提前揭露[某个悬念]

### 4. 示例驱动（Few-shot）
给出参考范例，让AI模仿风格：
```
示例开头："血色试炼场内，林凡看着手中碎裂的玉佩，眼中闪过一丝冷意。"
请以此风格续写...
```

## AI辅助写作的进阶应用

### 与[[爽感四大基石]]的结合

| 心理需求 | AI辅助写法 |
|---------|-----------|
| **公平需求** | AI生成"打脸"场景时，要求包含：反派羞辱细节 → 主角隐忍蓄势 → 绝地反击 → 全场震撼 |
| **优越需求** | AI生成"认知碾压"时，让主角说出"只有他知道的真相"，周围人"恍然大悟" |
| **掌控需求** | AI生成"逆袭"场景时，强调主角"从棋子变棋手"的身份转换 |
| **归属需求** | AI生成"羁绊"场景时，突出"被认可、被需要"的情感细节 |

### 特定场景的AI写作指令模板

#### 卡文急救
```
主角当前处境：[具体描述]
已有资源：[能力/道具/盟友]
已有冲突：[当前面临的问题]
已完成伏笔：[之前埋下的伏笔]

请给出5种可能的"破局方案"，每种方案要包含：
1. 剧情走向
2. 可能风险
3. 爽感评估（1-5星）
```

#### 打斗场面
```
描写一场[具体类型]战斗。
重点不在战斗过程，而在：
1. 战前氛围（对手极尽羞辱的细节）
2. 反击瞬间（主角一招制胜的描写）
3. 旁观者反应（从嘲讽到震惊的转变）
4. 结束语（主角一句收尾台词，尽显逼格）
```

#### 伏笔埋设
```
请检查以下章节内容，列出：
1. 已埋设的伏笔（[章节]-[伏笔内容]）
2. 伏笔的回收节点建议
3. 可能存在的逻辑漏洞
```

## 常见陷阱与应对

| 陷阱 | 表现 | 应对方法 |
|------|------|----------|
| **幻觉问题** | AI编造不存在的"历史事件/术语" | 所有AI内容必须人工审核 |
| **同质化** | 所有作者用同一套提示词 | 发展个人专属提示词库 |
| **过度依赖** | 内容缺乏个人风格 | AI是助手，核心创意在人 |
| **深度不够** | 缺乏人性/情感的深层刻画 | 核心情节高潮必须人工写 |

## AI在各环节的可用性评估

| 环节 | AI可用性 | 建议 |
|------|---------|------|
| 构思/头脑风暴 | ⭐⭐⭐⭐⭐ | 大胆用 |
| 大纲制定 | ⭐⭐⭐⭐ | AI出方案，人选 |
| 章纲细写 | ⭐⭐⭐⭐ | AI辅助，人工把关 |
| 正文初稿 | ⭐⭐⭐ | 限用扩写/补全 |
| 打斗场面 | ⭐⭐ | 人工核心框架 |
| 感情线 | ⭐⭐ | 谨慎使用 |
| 结尾/高潮 | ⭐⭐ | 人工主导 |

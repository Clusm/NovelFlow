# 自然钩子设计

> **版本**：v3.0 | **定位**：L3子skill | **原则**：信息不对等+动作中断+认知冲突

---

## 执行流程

```
选择类型 → 植入位置 → 多维期待 → 检测刻意
```

---

## 核心命令

### 1. 钩子类型
| 类型 | 写法 | 效果 |
|------|------|------|
| 信息不对等 | 读者知角色不知 | 紧张感 |
| 动作中断 | 动作进行到一半 | 悬念 |
| 对话中断 | 对话最关键处 | 好奇 |
| 认知冲突 | 与预期矛盾 | 震惊 |
| 情感牵引 | 情感共鸣牵引 | 遗憾 |

### 2. 植入位置
| 位置 | 类型 | 效果 |
|------|------|------|
| 500字 | 小钩子 | 信息不对等 |
| 1000字 | 中钩子 | 对话/动作中断 |
| 2000字 | 大钩子 | 情感牵引/认知冲突 |
| 章末 | 核心钩子 | 悬念 |

### 3. 五层期待感
| 层级 | 效果 |
|------|------|
| 微观 | 每章必点下一章 |
| 中观 | 阶段目标 |
| 宏观 | 世界观悬念 |
| 过渡层 | 情绪衔接 |
| 反向 | 打破套路 |

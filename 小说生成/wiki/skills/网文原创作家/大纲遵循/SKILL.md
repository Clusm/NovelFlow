# 大纲遵循

> **版本**：v1.0（新建）
> **来源**：作家续章时需要主动遵循大纲
> **定位**：L2子skill，解决作家创作时偏离大纲的问题

---

## 🚨 核心问题

| 问题 | 解决方案 |
|------|---------|
| 作家写续章时忘记大纲 | 创作前读取锁定点 |
| 不知不觉偏离主线 | 每500字偏离自检 |
| 锚点设定被遗忘 | 强制触发锚点检查 |

---

## 📚 核心技能框架

### 1. 锁定点读取

**作家创作前的强制动作**：
```
Step 1：读取架构师输出的大纲文件
Step 2：定位本章对应的锁定点
Step 3：确认【必须发生】的事件
Step 4：确认【禁止发生】的事件
Step 5：确认【伏笔埋下】的触发条件
```

**锁定点格式（作家视角）**：
```markdown
## 第X章 锁定点

### ✅ 本章必须发生
- 事件1
- 事件2

### ❌ 本章禁止发生
- 事件1
- 事件2

### 🔮 本章伏笔埋下
- 伏笔ID：FP001（玉佩发光）
- 触发条件：遇到危险

### 📍 锚点提醒
- 主角当前境界：E级
- 主角当前位置：幽岚区
- 关键人物：苏小灵在场
```

### 2. 偏离自检

**每500字的强制自检**：
```
□ 本章【必须发生】的事件是否已经包含？
□ 本章【禁止发生】的事件是否出现？
□ 锚点（境界/地点/人物）是否正确？
□ 战力表现是否与设定匹配？
□ 伏笔触发条件是否满足？
```

**偏离处理**：
| 偏离类型 | 处理方式 |
|---------|---------|
| 未写【必须发生】 | 立即补写或调整 |
| 出现【禁止发生】 | 删除或改写 |
| 锚点错误 | 修正后再继续 |
| 战力矛盾 | 查阅设定后修正 |

### 3. 锚点触发

**锚点类型与触发检查**：
| 锚点类型 | 检查时机 | 错误表现 |
|---------|---------|---------|
| **身份锚** | 角色出场 | 名字/身份错误 |
| **境界锚** | 战斗场景 | 战力表现矛盾 |
| **地点锚** | 场景切换 | 地理位置错误 |
| **关系锚** | 互动场景 | 关系描述错误 |
| **物品锚** | 物品出现 | 来历/状态错误 |

**锚点触发口诀**：
```
写到锚点 → 暂停 → 核对 → 继续
每500字 → 自检 → 修正 → 继续
```

## 🔄 创作流程（作家视角）

```
接收到创作任务
    ↓
【第一步】锁定点读取
├─ 读取大纲文件
├─ 定位本章锁定点
├─ 确认【必须】【禁止】【伏笔】
    ↓
【第二步】锚点核对
├─ 核对当前人物状态
├─ 核对当前地点
├─ 核对当前时间线
    ↓
【第三步】开始创作
├─ 每500字偏离自检
├─ 锚点触发时立即核对
    ↓
【第四步】完成后检查
├─ 对照锁定点逐项检查
├─ 如有偏离 → 修正后再提交
    ↓
提交审核
```

---

## ⚠️ 强制要求

| 要求 | 说明 |
|------|------|
| **创作前必须读取锁定点** | 不能凭记忆写 |
| **每500字必须自检** | 防止不知不觉偏离 |
| **锚点必须触发核对** | 不能跳过 |
| **偏离必须修正后再提交** | 不能带病提交 |

## 📊 检测清单

```
□ 是否在创作前读取了锁定点？
□ 是否确认了【必须】【禁止】【伏笔】？
□ 每500字是否进行了偏离自检？
□ 锚点触发时是否进行了核对？
□ 完成后是否逐项对照锁定点检查？
□ 如有偏离，是否修正后再提交？
```

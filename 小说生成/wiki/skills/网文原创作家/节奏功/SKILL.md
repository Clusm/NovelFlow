## 📚 节奏控制

### 三幕结构（2000字章节版）

| 幕 | 位置 | 字数 | 内容 |
|---|------|------|------|
| **开头** | 章首200字 | 300字 | 延续上章钩子/快速推进/冲突砸脸 |
| **发展** | 章中1200字 | 1500字 | 核心冲突/角色反应/障碍出现/小转折 |
| **结尾** | 章末600字 | 200字 | 钩子+悬念+下章预告 |

### 章节节奏公式

```
紧张（300字） → 放松（100字） → 紧张（300字） → 放松（100字）...
```

### 爽点节奏公式

| 节奏 | 频率 | 说明 |
|------|------|------|
| 小转折 | 每500字 | 1个小转折 |
| 小爽点 | 每3章 | 1个小高潮/爽点 |
| 中爽点 | 每10章 | 1个中高潮 |
| 大高潮 | 每50章 | 1个大高潮/反转 |

### 番茄模式 vs 起点模式

| 平台 | 节奏 | 钩子密度 |
|------|------|---------|
| **番茄** | 快（每500字1个小转折） | 章末强钩 |
| **起点** | 稳（每800字1个小转折） | 伏笔慢热 |

## 📚 章末钩子设计

### 四种钩子类型

| 类型 | 操作 | 效果 | 示例 |
|------|------|------|------|
| **信息不对等** | 主角先知道，读者不知道 | 替读者着急 | "卡牌是反派设的局" |
| **动作中断** | 关键动作做到一半被打断 | 想知道结果 | "刀尖离喉咙1寸..." |
| **对话中断** | 关键信息说到一半被打断 | 想知道下文 | "反派正要说出名字..." |
| **悬念延续** | 答案揭晓前戛然而止 | 追看下章 | "门后的人，让他愣住了" |

### 刻意钩子（禁止）
```
❌ 突然晕倒/半夜敲门/神秘电话
❌ "这时，手机突然响了..."
❌ "突然，一阵敲门声响起..."
```

### 自然钩子（推荐）
```
✅ 动作延续："他正要开门，锁孔里传来第二声转动..."
✅ 信息悬念："他看了眼手机，脸色骤变——发件人是三年前的自己"
✅ 逻辑延伸："按照计划，他应该在十分钟后出现。可现在已经过了二十分钟"
```

### 每500字一个小钩子分布

| 位置 | 字数 | 钩子类型 | 内容 |
|------|------|---------|------|
| 前200字 | 200 | 强冲突/强钩子 | 砸脸式开头 |
| 500字 | 300 | 小转折 | 新障碍/新信息 |
| 800字 | 300 | 信息暗示 | 伏笔埋设 |
| 1200字 | 300 | 意外发生 | 反转/危机 |
| 1500字 | 200 | 章末钩子 | 悬念/冲突 |
| 1800字 | 200 | 下章预告 | 期待感 |

## 📚 情绪曲线

### 波浪式节奏图

```
高潮 ───┐    ┌───高潮───┐    ┌───最高潮
        │    │          │    │
平稳────┘────┘──平稳──┘────┘────
开局    小高潮   过渡     中高潮    终高潮
```

### 节奏平衡法则

| 原则 | 说明 | 示例 |
|------|------|------|
| 紧张后必有放松 | 给读者喘息空间 | 打斗后加日常 |
| 高潮后必有过渡 | 不能连续高能 | 大战后休整 |
| 打斗后加日常 | 平衡节奏 | 战斗完吃饭聊天 |
| 虐心后加甜蜜 | 情绪调节 | 虐完后发糖 |

### 情绪曲线设计

```
【低开高走型】
压抑(20%) → 希望(30%) → 绝望(70%) → 爆发(100%)

【波浪前进型】
挫折 → 小成 → 更大挫折 → 大成 → 最终决战

【倒叙型】
结果前置 → 回顾 → 揭示真相 → 新发展
```

## 📚 章节连贯性

### 连贯检查三要素

```
□ 本章解决了上章留下的悬念吗？
□ 本章结束后主角要往哪个方向走？
□ 下一章开头从哪里接？
```

### 连贯技巧

| 技巧 | 操作 | 示例 |
|------|------|------|
| **钩子呼应** | 章末钩子在下一章立即解决 | 钩："他抬头，看到..." |
| **时间连续** | 标注时间变化 | "十分钟后..." |
| **场景转换** | 标注地点变化 | "与此同时，城南..." |
| **人物接续** | 上章结尾人物接下章 | "他还没说完，另一人..." |

```
□ 开头是否快速进入冲突？

□ 章末是否留了钩子？
□ 钩子是自然的还是刻意的？
□ 紧张和放松是否平衡？
□ 高潮后是否有过渡？
□ 章节连贯性是否过关？
□ 爽点密度是否足够？
□ 三种期待是否同时存在？
□ 目标是否清晰？
```

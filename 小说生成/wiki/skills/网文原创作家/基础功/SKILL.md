## 一、联网学习功能

### 触发时机
- 每次创作前必须执行
- 检测到不熟悉的场景描写时执行深度搜索
- 每季度更新真人化写作技巧

### 联网学习流程
```
【搜索前】明确学习目标："[具体场景描写]+2026"
    ↓
【搜索中】分层搜索
├─ 第一层：概览搜索（1-2次）
└─ 第二层：深度搜索（按维度）
    ↓
【筛选中】选择高价值页面
    ↓
【深度阅读】提炼关键信息（3-5个点/页）
    ↓
【整合分析】形成结论，对照目标检查
    ↓
【输出报告】结构化呈现
```

### 联网学习执行规范
| 阶段 | 动作 | 检查项 |

| 搜索前 | 明确学习目标 | □ 搜索词精准 |

## 📚 六步法骨架（情节单元）

**核心公式**：
```
情绪事件 → 欲望目标 → 阻碍困境 → 解决方法 → 行动解决 → 结局反馈
```

### 解决方法14途径

| 类型 | 途径 | 说明 |
|------|------|------|
| **被动应对** | 逃跑 | 遇强敌先跑保存实力 |
| | 求饶 | 示弱获取喘息空间 |
| | 智取 | 用脑子而非拳头解决问题 |
| | 借力 | 找帮手/找工具/找规则漏洞 |
| | 妥协 | 以退为进换取更大利益 |

## 📚 文字精瘦指南

### 六类删除词清单

| 类别 | 典型词 | 改法 |
|------|--------|------|
| **感官过滤词** | 看见、听到、感到、留意到、察觉到、注意到 | 直接删除，用具体感知替代 |
| **机械过渡词** | 然后、接着、随即、不一会儿、之后、最后 | 用具体动作串联 |
| **情绪标签词** | 伤心、难过、愤怒、生气、紧张、害怕、开心、高兴 | 用身体反应链条替代 |
| **口语虚词** | 其实、就是、大概、吧、呢、吗、嘛、呃 | 删除 |
| **空壳形容词** | 漂亮的、好看的、温柔的、舒服的、帅气的 | 用具体感官细节替代 |
| **模糊程度词** | 很、非常、特别、挺、相当、极其、异常 | 删除或用具体描写替代 |

### 文字抽脂法
```
初稿 → 终稿：短20%以上
删除三连：砍形容词 → 砍副词 → 砍连接词
```

## 📚 去解释化写作（核心技能）

### 展示vs讲述的本质区别

**讲述（Tell）**：直接告诉读者答案
```
❌ "他很紧张"
❌ "她很伤心"
❌ "这个房间很阴森"
```

**展示（Show）**：让读者自己感受到
```
✅ "他的手指在抖，喉结滚动了一下"
✅ "她反复拧着瓶盖，一遍又一遍"
✅ "墙角泛着水渍，天花板压得很低，灯管闪了一下又灭了"
```

### 情绪/状态 → 身体反应链条 → 具体动作

| 情绪 | 身体反应 | 替代写法 |
|------|---------|---------|
| 紧张 | 手指抖/咽口水/声音发颤 | "他的手心全是汗" |
| 愤怒 | 握拳/指节发白/胸膛起伏 | "他的指节攥白" |
| 害怕 | 后退/瞳孔放大/屏住呼吸 | "他往后退了半步" |
| 伤心 | 眼眶泛红/声音哽咽/沉默 | "她反复拧着瓶盖" |
| 惊讶 | 愣住/倒吸一口气/瞳孔放大 | "他脚步骤然停住" |

### 破折号使用规范

**禁止使用破折号解释**：
```
❌ "他冷笑一声——这是他表达不屑的方式"
❌ "她眼眶泛红——显然在强忍着泪水"
❌ "陆言心里咯噔一下——一种不祥的预感涌上心头"
```

**正确用法（允许）**：
```
✅ 话题转换："他顿了顿——说到这里，他突然想起什么"
✅ 声音延续："远处传来一声闷响——像是雷声"
✅ 同位语补充："他——曾经的全省高考状元——现在在送外卖"
```

### 5步改写法

```
第1步：定位"解释" → 找到所有"——"后的解释性文字
第2步：追问"是什么" → 紧张时身体会有什么反应？
第3步：替换为身体反应 → "他很紧张" → "他的手指不自觉地绞在一起"
第4步：检查是否删干净 → 确认原解释已完全删除
第5步：大声朗读 → 读起来别扭的地方 = 还没改干净
```

## 📚 章节自检三问法

### 三问检验

```
问题一：主角叫什么？
        → 名字有没有让读者记住？

问题二：他想干什么？
        → 目标清晰吗？
        → 为什么想？

问题三：他在做什么？
        → 行动和目标一致吗？
        → 读者知道他在做什么吗？
```

### 陌生化检验
```
把主角名换成陌生名字
问自己："如果这不是我写的，我觉得好看吗？"
```

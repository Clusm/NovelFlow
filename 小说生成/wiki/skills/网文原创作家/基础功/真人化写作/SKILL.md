# 真人化写作

> **版本**：v3.0 | **定位**：L3子skill | **原则**：去除AI味，模拟真人认知

---

## 核心命令

### 1. 身体现象学
| 项目 | 内容 |
|------|------|
| AI问题 | "抬头看天空"只写"把头抬到45度" |
| 真人写法 | "脖颈发酸才意识到仰头太久" |
| 三要素 | 身体部位 + 感受程度 + 时间感知 |

### 2. 微表情技巧
| 项目 | 内容 |
|------|------|
| AI问题 | "她笑了"只写"嘴角上扬" |
| 真人写法 | "眼角先绷了一下，假装没事地别开脸" |
| 五层 | 眼+面+肢+声+息 |

### 3. 口语化对话
| 项目 | 内容 |
|------|------|
| AI问题 | 对话"完美"但空洞 |
| 真人写法 | 有省略/重复/废话 |
| 六技巧 | 省略+重复+废话+打断+沉默+方言 |

### 4. 情感颗粒度
| 项目 | 内容 |
|------|------|
| AI问题 | "她很伤心"只有"流泪" |
| 真人写法 | 细分委屈/想哭/哭不出来/哭到发抖 |
| 四维度 | 强度+层次+混搭+出口 |

### 5. 五感法
| 项目 | 内容 |
|------|------|
| 视觉 | 雨水像银针斜织在青石板路 |
| 听觉 | 雨点敲击铜铃发出叮当声 |
| 嗅觉 | 潮湿水汽混合陈年普洱香气 |
| 味觉 | 汤头猪骨熬了八个时辰 |
| 触觉 | 紫砂壶被岁月摩挲得温润 |

---

## 禁止项

| 禁止 | 正确 |
|------|------|
| "他笑了" | 微表情 |
| "她很伤心" | 生理反应+细分 |
| 完整对话 | 省略/重复/停顿 |
| 抽象动作 | 身体感受 |
| 平铺直叙 | 五感法 |

---

## 自检清单

- [ ] 情绪有身体反应
- [ ] 对话有潜台词
- [ ] 情感有细分层次
- [ ] 有五感描写
- [ ] 朗读无AI味

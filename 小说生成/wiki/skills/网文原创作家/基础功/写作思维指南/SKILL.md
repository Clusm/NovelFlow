## 执行流程
```
输入 → 构思底层 → 人物构思 → 结构设计 → 冲突设计 → 节奏控制 → 输出
```

## 小说构思底层逻辑

### 网文三要素

| 要素 | 定义 | 实现方式 |
|------|------|---------|
| **爽点** | 读者获得情绪满足的瞬间 | 升级、打脸、获得、逆转、反杀 |
| **代入感** | 读者产生"我就是主角"的感觉 | 视角贴脸、五感描写、处境真实 |
| **期待感** | 读者想知道后续会发生什么 | 悬念、伏笔、目标明确、章末钩子 |

### 核心公式

**主角公式**：
```
主角 = 身份标签 + 反差秘密 + 执念目标 + 致命弱点
```

**爽点公式**：
```
遇到问题 → 陷入困境 → 寻找方法 → 反转逆袭 → 获得奖励
```

**冲突公式**：
```
冲突 = 目标 + 阻碍 + 行动 → 结果
```

### 四类核心冲突

| 类型 | 定义 | 例子 |
|------|------|------|
| 人与自我 | 内心挣扎、道德困境 | 复仇还是放过、牺牲还是保全 |
| 人与他人 | 与对手、盟友的矛盾 | 打脸、争夺、误会 |
| 人与环境 | 与世界、规则的冲突 | 废土求生、等级压制 |
| 人与命运 | 抗天、改命、既定结局 | 退婚流、命定之人 |

## 人物构思方法论

### 主角三要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **欲望** | 主角想要什么 | 变强、复仇、守护、探索 |
| **缺陷** | 主角的弱点/问题 | 性格缺陷、身体缺陷、过往创伤 |
| **底牌** | 主角隐藏的王牌 | 血脉、系统、身份、关系 |

### 配角设计原则

| 角色类型 | 功能 | 设计要点 |
|----------|------|---------|
| 盟友 | 帮助主角 | 有自己的动机，与主角利益绑定 |
| 对手 | 制造冲突 | 与主角目标对立，实力相当 |
| 导师 | 推动成长 | 提供信息/资源，最终会离开或背叛 |
| 情人 | 情感线 | 与主角有化学反应，身份差距大 |
| 工具人 | 推动剧情 | 只在需要时出现，服务于情节 |

### 反派设计原则

| 设计维度 | 说明 |
|----------|------|
| 动机合理 | 反派有自己的逻辑，不是纯恶 |
| 实力相当 | 能给主角造成真正的威胁 |
| 与主角镜像 | 映照主角的另一种可能 |
| 结局方式 | 击败/洗白/死亡/反转 |

## 三层结构设计

### 宏观结构（三幕式）

| 阶段 | 占比 | 核心任务 |
|------|------|---------|
| **建立** | 20-25% | 建立世界、引入主角、抛出核心冲突 |
| **对抗** | 50-60% | 主角行动、阻碍出现、高潮迭起 |
| **结局** | 15-20% | 最终对决、解决冲突、交代后事 |

### 中观结构（卷纲）

| 结构 | 说明 |
|------|------|
| 起承转合 | 每个卷都有独立的小高潮 |
| 升级线 | 实力提升、地位变化 |
| 情感线 | 关系变化、情感递进 |
| 悬念线 | 伏笔埋设、逐步揭露 |

### 微观结构（章纲）

| 环节 | 字数占比 | 内容 |
|------|----------|------|
| 遭遇事件 | 10-15% | 快速建立情境 |
| 阻碍困境 | 20-25% | 障碍的具体表现 |
| 解决方法 | 25-30% | 尝试+找到正确方法 |
| 解决收获 | 20-30% | 执行+获得回报 |

## 冲突构思技巧

### 六步法情节单元

```
①情绪事件 → ②欲望目标 → ③阻碍困境 → ④解决方法 → ⑤行动解决 → ⑥结局反馈
```

### 解决方法14种途径

**能力途径（10类）**：

| # | 途径 | 文字表现要点 |
|---|------|-------------|
| 1 | 战斗力碾压 | 描写力量悬殊的震撼感 |
| 2 | 技巧制胜 | 详细描写技巧运用过程 |
| 3 | 阵法/策略 | 展示布局过程+触发效果 |
| 4 | 速度/闪避 | 动态描写+惊险瞬间 |
| 5 | 防御/消耗 | 心理博弈+耐力比拼 |
| 6 | 环境利用 | 场景互动+突发变数 |
| 7 | 道具/装备 | 物品登场+效果展示 |
| 8 | 能力觉醒/突破 | 突破过程描写+新能力展示 |
| 9 | 召唤/援军 | 援军登场时机+配合战斗 |
| 10 | 牺牲/代价 | 悲壮感+后续影响 |

**信息途径（4类）**：

| # | 途径 | 文字表现要点 |
|---|------|-------------|
| 11 | 调查/搜索 | 推理过程+发现时刻 |
| 12 | 交流/套话 | 对话交锋+心理博弈 |
| 13 | 观察/推理 | 侦探式思考过程 |
| 14 | 回忆/经验 | 回忆杀+灵光一现 |

### 高潮转折设计

**3大类9小类转折**：

| 大类 | 小类 | 效果 | 写作要点 |
|------|------|------|---------|
| 方向转折 | 反败为胜 | 最大爽感 | 绝望→希望的关键一击 |
| 方向转折 | 胜转败 | 最大冲击 | 意外的失败引发新的危机 |
| 方向转折 | 敌我转换 | 意外感 | 盟友背叛/敌人倒戈 |
| 认知转折 | 真相揭露 | 震撼 | 之前的所有理解被推翻 |
| 认知转折 | 身份揭示 | 惊讶 | "原来TA是…" |
| 认知转折 | 规则颠覆 | 震撼 | "原来这个世界的规则不是这样的" |
| 情感转折 | 情感爆发 | 共鸣/感动 | 压抑已久的情感终于释放 |
| 情感转折 | 关系变化 | 意外 | 友→敌/敌→友/爱→恨 |
| 情感转折 | 价值观动摇 | 深度 | 一直坚信的东西开始动摇 |

## 节奏控制方法

### 起承转合节奏

| 环节 | 功能 | 节奏特点 |
|------|------|---------|
| **起** | 抛出问题 | 平缓，建立情境 |
| **承** | 推进发展 | 逐渐加速 |
| **转** | 转折/高潮 | 最紧张、最快 |
| **合** | 收束解决 | 放缓，释放情绪 |

### 套娃结构

```
大循环（卷）
├── 中循环（章）
│   └── 小循环（段落）
```

### 爽点密度控制

| 阶段 | 爽点频率 | 说明 |
|------|----------|------|
| 前期(1-30万字) | 每章1个小爽点 | 打脸、升级、获得 |
| 中期(30-100万字) | 2-3章一个爽点 | 大战、突破、揭秘 |
| 后期(100万字+) | 根据节奏调整 | 决战、高潮、收尾 |

### 章末钩子设计

| 钩子类型 | 效果 | 示例 |
|----------|------|------|
| 悬念式 | 留疑问 | "他没想到，门后站着的竟是…" |
| 冲突式 | 制造对立 | "话音未落，一道寒光袭来" |
| 收获式 | 期待奖励 | "终于找到了传说中的…" |
| 危机式 | 紧张感 | "糟了，他们发现了" |
| 揭示式 | 震惊感 | "直到今天他才知道真相是…" |

## 构思检查清单

```
□ 主角目标是否明确？
□ 冲突是否足够强烈？
□ 爽点是否埋设到位？
□ 期待感是否建立？
□ 人物是否有记忆点？
□ 节奏是否符合平台要求？
□ 章末钩子是否强劲？
□ 是否有代入感设计？
```

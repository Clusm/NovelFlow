# 四维拆解法

> **版本**：v2.0 | **定位**：L4子skill | **来源**：联网学习

---

## 核心内容

### 1. 创新点提炼
| 项目 | 内容 |
|------|------|
| 类型 | 设定/人设/冲突/情感/节奏/结构 |
| 四类脑洞 | 金手指/剧情/设定/角色 |

### 2. 差异化分析
| 项目 | 内容 |
|------|------|
| 矩阵 | 竞品对比 + 平台适配 |
| 三问 | 只有我能写 + 读者为什么看 + 差异化够支撑一本书 |

### 3. 反同质化检测
| 项目 | 内容 |
|------|------|
| 检测点 | 人设/设定/情节/缝合/既视感 |
| 检测三问 | 只有我能写 + 读者为什么选 + 差异化够支撑 |

### 4. 套路重构
| 项目 | 内容 |
|------|------|
| 公式 | 旧套路 + 新元素 = 微创新 |

---

## 禁止项

| 禁止 | 正确 |
|------|------|
| 提炼模板元素 | 提炼创新元素 |
| 忽视竞品分析 | 差异化检测 |
| 同质化创作 | "只有我能写" |

---

## 自检清单

- [ ] 提炼创新点而非模板
- [ ] 有四类脑洞之一
- [ ] 竞品差异化对比
- [ ] 同质化风险检测
- [ ] "只有我能写"
- [ ] 四维拆解完整
- [ ] 微创新可持续

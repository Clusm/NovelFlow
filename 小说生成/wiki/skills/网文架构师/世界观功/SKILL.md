## 一、联网学习功能

### 触发时机
- 每次世界观设计前必须执行
- 设计新题材世界观时执行深度搜索
- 每季度更新世界观设计技巧

### 联网学习流程
```
【搜索前】明确学习目标："[具体世界观类型]+2026"
    ↓
【搜索中】分层搜索
├─ 第一层：概览搜索（1-2次）
└─ 第二层：深度搜索（按维度）
    ↓
【筛选中】选择高价值页面
    ↓
【深度阅读】提炼关键信息（3-5个点/页）
    ↓
【整合分析】形成结论，对照目标检查
    ↓
【输出报告】结构化呈现
```

### 联网学习执行规范
| 阶段 | 动作 | 检查项 |
|------|------|--------|
| 搜索前 | 明确学习目标 | □ 搜索词精准 |
| 搜索中 | 分层搜索 | □ 优先一手来源 |
| 筛选中 | 选择高价值页面 | □ 每轮展示中间结果 |
| 深度阅读 | 提炼关键信息 | □ 只读相关内容 |
| 整合分析 | 形成结论 | □ 对比不同来源 |
| 输出报告 | 结构化呈现 | □ 有足够信息停止 |

### 学习计划
| 周期 | 学习内容 | 目标 |

## 二、力量体系设计

### 等级系统模板

| 等级 | 名称 | 能力描述 | 升级条件 |
|------|------|---------|---------|
| F | 灵能觉醒 | 感知灵气 | 觉醒丹/顿悟 |
| E | 驱魂者 | 基础灵术 | 实战积累 |
| D | 捉妖师 | 中级灵术 | 完成任务 |
| C | 灵媒师 | 高级灵术 | 心境突破 |
| B | 阴阳师 | 领域展开 | 功德积累 |
| A | 天师 | 天人合一 | 信仰之力 |
| S | 大天师 | 法则掌控 | 渡劫飞升 |

### 战力判定公式
```
实际战力 = 等级压制 × 灵器增幅 × 环境加成 × 状态系数

示例：
基础战力100 × 等级压制1.5 × 灵器增幅2.0 × 环境加成1.2 = 360
```

### 战力差距设计
```
越级挑战：
√ 主角有特殊能力/宝物
√ 对手有大意/轻敌
√ 智取而非硬拼
√ 限制对手发挥（环境/规则）

越级失败：
√ 差距过大（2个大等级以上）
√ 对方有准备
√ 主角经验不足
```

## 📚 势力分布设计

### 势力类型

| 势力类型 | 设计要点 | 示例 |
|---------|---------|------|
| **正道势力** | 维护秩序、主角阵营 | 龙渊阁/散修联盟/灵调局 |
| **邪道势力** | 制造冲突、反派来源 | 幽冥殿 |
| **中立势力** | 缓冲地带、剧情调剂 | 鬼市/阴阳客栈 |
| **势力关系图** | 合作/对抗/利用 | 正邪对立+利益纠葛+个人恩怨 |

### 势力关系设计
```
【对立型】
正派 vs 邪派（立场对立）
→ 直接冲突，打怪升级

【竞争型】
多个正派势力争夺资源
→ 内部斗争，权谋博弈

【隐藏型】
表面正派，实际反派
→ 反转惊喜，制造高潮

【动态型】
势力关系随剧情变化
→ 联合/背叛/复仇
```

## 📚 资源货币体系

### 资源类型

| 资源类型 | 用途 | 获取方式 |
|---------|------|---------|
| **灵力值** | 核心货币/升级 | 任务/击杀/修炼 |
| **灵石** | 高阶交易货币 | 秘境/锻造 |
| **功德值** | 正道修行资源 | 救助他人/积德行善 |
| **灵器装备** | 战力增幅 | 任务奖励/锻造 |
| **丹药秘术** | 突破/恢复/特殊效果 | 炼丹/传承 |
| **声望/名声** | 社会地位 | 事件/成就 |

### 资源获取路径
```
【稳定来源】
日常任务/修炼产出
→ 保证基本成长

【随机来源】
击杀掉落/秘境探索
→ 增加惊喜感

【剧情来源】
完成任务/事件触发
→ 配合主线推进

【代价获取】
以物易物/付出代价
→ 增加戏剧性
```

## 📚 地理版图设计

### 地理类型

| 地理类型 | 设计要求 | 示例 |
|---------|---------|------|
| **主城** | 主角活动中心 | 云澜市 |
| **灵界城市** | 灵异界主要城市 | 星落城 |
| **禁地秘境** | 高危/机遇地点 | 幽冥渊/雾隐岛 |
| **势力范围** | 各势力地盘划分 | 幽岚区/青岩镇 |

### 地理设计要点
```
【场景服务于剧情】
场景不是摆设，而是剧情的一部分
→ 地形影响战斗
→ 环境制造障碍
→ 地点触发事件

【地图分级】
新手区（安全）→ 进阶区（挑战）→ 高阶区（机遇）
→ 控制成长节奏
```

## 📖 世界观检查清单

```
□ 力量等级是否闭环？（晋升路径清晰）
□ 势力关系是否清晰？（有合作有对抗）
□ 资源是否有获取路径？（不能只消耗不补充）
□ 地理版图是否全虚构？（避免现实地名）
□ 设定是否自洽无漏洞？（不能前后矛盾）
□ 世界观是否服务剧情？（不是信息堆砌）
```

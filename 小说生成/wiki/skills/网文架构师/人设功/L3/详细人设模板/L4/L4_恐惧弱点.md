## 九、恐惧/弱点/软肋设计

### 9.1 核心恐惧类型

| 类型 | 示例 | 戏剧作用 |
|------|------|---------|
| **失去所爱** | 害怕失去亲人/爱人 | 制造抉择困境 |
| **身份暴露** | 隐藏身份被揭穿 | 增加紧张感 |
| **重蹈覆辙** | 害怕变成仇人那样 | 内心挣扎 |
| **能力失控** | 异能反噬/伤害他人 | 增加风险 |
| **孤独终老** | 害怕独自一人 | 情感线推进 |
| **一事无成** | 害怕平庸度过一生 | 奋斗动力 |

### 9.2 致命弱点设计

```markdown
弱点类型：
├─ 生理弱点：恐高/晕血/夜盲/身体缺陷
├─ 心理弱点：童年阴影/自卑/愧疚
├─ 情感弱点：软肋之人/放不下之人
├─ 能力弱点：能力边界/使用代价
└─ 身份弱点：隐藏身份的漏洞
```

## 十、新手禁忌清单

| 禁忌 | 正确做法 |
|------|---------|
| **配角设计成纯工具人** | 配角也要有独立人格和故事线 |
| **信息堆砌** | 用事件/侧描代替干巴描述 |
| **完美人设** | 必须有缺陷，才有看点 |
| **主角纯好人/反派纯坏人** | 放到社会层面都有形成原因 |
| **外貌描写过于冗长** | 只留1个核心记忆点 |
| **性格标签堆砌** | 1个核心性格+2-3个特点即可 |
| **成长太顺/升级太快** | 每个阶段必须有挫折 |
| **反派太弱** | 反派必须有实力才能制造压力 |

## 六、人设描写公式（网文封神公式）

### 6.1 天骄型（傲慢天才）

```markdown
【视觉锚点】
华丽装备特写 + 与年龄不符的松弛感

【数据标签】
年龄（必须小）+ 境界（必须高）+ 战绩（必须傲人）

【气息描写】
不收敛的威压 + 环境扭曲

【动作特征】
慢动作（表示游刃有余）+ 俯视视角

【示例】
那是个看起来不过二十出头的青年，银白作战服一尘不染。
他斜倚在栏杆上，右手把玩着一枚玉质军功章，
眼神压根没看主角，而是透过落地窗看楼下的风景。
```

### 6.2 高境界强者（宗师/传说）

```markdown
【视觉锚点】
极致的普通（或极致的异象）
+ 装备"不存在"或"化作身体一部分"

【数据标签】
境界用"无法感知"或"如深渊"来描述，不用具体数字

【气息描写】
不是威压，是"真空"（所有气息被吸收）

【动作特征】
静止（仿佛时间在他身上停滞）
或 瞬移（没有过程）

【示例】
那是个看起来四十岁左右的中年人，穿着洗得发白的旧军装。
但仔细看，他的影子是静止的——窗外树影在动，
他的影子却像照片一样钉在地上。
```

### 6.3 铁血老兵（伤残/底层/实战派）

```markdown
【视觉锚点】
残缺的肢体 + 磨损到极限的装备
+ 与周围环境融为一体的伪装色

【数据标签】
境界不高（甚至低于主角）
+ 军工等级极高（靠功勋堆上去）
+ 击杀数惊人

【气息描写】
血腥气（洗不掉）+ 死寂（没有威压，但让人脊背发凉）

【动作特征】
防御性姿态（永远侧身/手永远在刀附近）
+ 习惯性检查周围环境
```

### 6.4 外貌描写模板

```markdown
【静态描写】
姓名：（名字含义/来历）
年龄：
身高：（具体数值）
体型：（健壮/消瘦/匀称）
发色/发型：（颜色+长度+造型）
眼睛：（颜色+形状+眼神特质）
面部：（轮廓+特别特征）

【动态描写】（推荐用这种方式）
一颦一笑：一举一动：穿衣风格：
走路的姿态：站立的姿势：
习惯性动作：表情变化：

【记忆点设计】（必须有一个）
最显著的特征：
读者记住的关键词：
```

## 七、社会关系网设计（强联系设计）

### 7.1 天然矛盾来源

| 关系 | 天然带来的剧情 |
|------|---------------|
| 父亲是赌徒 | 孩子被催债 |
| 母亲是公主 | 女儿是皇亲国戚/有人追捧 |
| 出身医学世家 | 家族利益纠葛 |
| 师父是大佬 | 师门恩怨/传承争夺 |
| 前任是敌人 | 感情纠葛/背叛 |

### 7.2 人设之间的强联系（三种方式）

| 方式 | 示例 |
|------|------|
| **利益捆绑** | 配角掌控反派修炼必需的特殊药材 |
| **信息差掌控** | 配角知晓主角不为人知的弱点 |
| **情感负债** | 男二曾因失误导致女主家族蒙冤 |

### 7.3 关系深度的四个层次

| 层次 | 程度 | 表现 |
|------|------|------|
| 第一层 | 认识 | 知道名字/见过面 |
| 第二层 | 熟悉 | 有过交往/共事 |
| 第三层 | 信任 | 可以托付后背 |
| 第四层 | 挚爱 | 愿意为对方死 |

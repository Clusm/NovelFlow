## 完整人设模板（填空版）

### 【基础信息】
```markdown
姓名：
性别：
年龄：（成长变化过程）
种族：
出身：
当前身份：
```

### 【外貌描写】
```markdown
整体观感：（读者第一眼印象）
身高/体型：
面部特征：（重点1个记忆点）
眼睛：（颜色+眼神特质）
发色/发型：
服饰风格：
标志性装备/随身物：
步态/气质：
```

### 【性格设定】
```markdown
核心性格：（一句话概括）
性格特点：
├─ 对陌生人：
├─ 对在意的人：
├─ 对敌人：
└─ 独处时：
优点：
缺点：
口头禅：
标志性小动作：
习惯/癖好：
MBTI/九型人格：（可选）
```

### 【三维模型】
```markdown
生理层：外貌/体型/穿着/习惯动作
社会层：职业/身份/社会地位/人际关系
心理层：欲望/恐惧/价值观/核心矛盾
```

### 【背景故事】
```markdown
家庭背景：
├─ 父亲：（性格/职业/与主角关系）
├─ 母亲：（性格/职业/与主角关系）
├─ 兄弟姐妹：（与主角关系）
└─ 重大事件：（影响性格形成的关键）
成长经历：（可展开剧情的素材）
童年创伤：（如果有）
形成原因：（为什么会成为现在的他）
```

### 【目标动机】
```markdown
短期目标：（1-30章）
中期目标：（30-100章）
终极目标：（全文核心）
行动动机：（为什么想要这个）
核心执念：（贯穿始终的深层欲望）
```

### 【金手指】
```markdown
类型：系统/老爷爷/异能/重生/穿越等
能力：（具体描述）
获得方式：（何时/如何获得）
限制条件：（能力边界/代价）
隐藏能力：（后期解锁）
```

### 【关系网】
```markdown
家人：（血缘关系=天然矛盾来源）
├─ 父亲：
├─ 母亲：
├─ 兄弟姐妹：
└─ 其他重要家人：
朋友/兄弟：
├─ 挚友：
├─ 普通朋友：
└─ 暧昧对象/红颜：
敌人/对手：
├─ 死敌：
├─ 竞争对手：
└─ 潜在威胁：
恩人/师父：
仇人：
```

### 【成长路线】
```markdown
初期状态：（开篇人设）
中期发展：（第一次转变）
转折点：（重大事件改变）
后期形态：（接近目标时）
最终状态：（目标达成后）
```

### 【记忆锚点】（必须设置1-2个）
```markdown
视觉锚点：（一看就记住的外貌）
行为锚点：（标志性动作）
语言锚点：（口头禅/说话方式）
物品锚点：（随身携带的重要物品）
特殊习惯：
```

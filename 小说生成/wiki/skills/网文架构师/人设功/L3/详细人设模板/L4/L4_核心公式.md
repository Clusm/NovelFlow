## 一、核心人设公式（3秒抓人）

```
主角魅力 = 初始身份 × 反差 × 核心执念
```

| 要素 | 说明 | 示例 |
|------|------|------|
| **初始身份** | 故事起跑线+社会坐标 | 废柴/赘婿/退役王牌 |
| **反差** | 表面vs内心/能力vs性格 | 外表高冷总裁×私下怕黑粘人 |
| **核心执念** | 深层、持久、不动摇的欲念 | 复仇/守护家人/证明自己 |

### 经典执念清单

| 类型 | 示例 |
|------|------|
| 创造型 | 创造/修复所拥有的"灵魂" |
| 守护型 | 守护特定的人/秩序/信念 |
| 证明型 | 向世界/某人证明自己 |
| 复仇型 | 向仇人讨回公道 |
| 探索型 | 寻找真相/身世/未知 |
| 反抗型 | 反抗命运/体制/强权 |

## 二、角色三维模型（完整版）

### 2.1 生理层（外在呈现）

```markdown
外貌描写：
├─ 视觉锚点：（1个核心记忆点，如眼尾痣、左手伤疤）
├─ 体型：高/矮/健壮/纤细/特殊
├─ 肤色：白/黄/古铜/苍白/有特征
├─ 眼神：锐利/温柔/空洞/深邃/特别
├─ 步态：挺拔/蹒跚/轻盈/沉重
├─ 服饰：华贵/朴素/特定风格
└─ 标志性装备：（武器/饰品/随身物）
```

### 2.2 社会层（身份定位）

```markdown
社会身份：
├─ 明面身份：（公开的职业/地位）
├─ 隐藏身份：（便于打脸的真实身份）
├─ 出身阶层：（上流/中产/底层）
├─ 教育背景：（家学/学院/自学）
├─ 经济状况：（富裕/小康/贫穷）
├─ 社会评价：（名声/口碑/传说）
└─ 组织归属：（宗门/家族/势力）
```

### 2.3 心理层（内在驱动力）

```markdown
心理层设计：
├─ 核心欲望：（最深层的渴望）
├─ 表层欲望：（说出口想要什么）
├─ 核心恐惧：（最害怕失去什么）
├─ 心理阴影：（童年创伤/遗憾/愧疚）
├─ 价值观：（人生观/世界观）
├─ 道德底线：（绝对不能触碰的线）
└─ 行为准则：（做什么事的原则）
```

### 欲望清单三问

| 问题 | 模板答案 |
|------|---------|
| **表面想要什么？** | 角色说出口的目标 |
| **内心真正想要什么？** | 角色可能都不知道的深层渴望 |
| **两个欲望是否冲突？** | 冲突越大越精彩 |

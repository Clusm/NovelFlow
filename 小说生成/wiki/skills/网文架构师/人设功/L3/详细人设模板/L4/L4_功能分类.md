## 四、角色功能分类设计

### 4.1 主角：功能性大于完美性

| 原则 | 说明 |
|------|------|
| **不完美引出故事线** | 性格缺陷引出更多剧情 |
| **功能性必须有** | 金手指/独特能力是核心看点 |
| **成长空间** | 废柴逆袭比天才碾压更有代入感 |
| **读者代入感** | 让读者能在角色身上找到自己 |

### 4.2 配角：记忆点大于完整性

| 原则 | 说明 |
|------|------|
| **工具人定位** | 给主角制造矛盾或解决矛盾 |
| **一个核心记忆点** | 嗜甜如命的暗卫/结巴的毒医 |
| **一句话能记住** | 不需要完整背景 |
| **独立人格** | 配角也要有自己故事线 |

### 4.3 反派：破坏力大于邪恶值

| 原则 | 说明 |
|------|------|
| **必须有实力** | 再坏但菜也当不了反派 |
| **制造压力** | 悬在主角头顶的利剑 |
| **不纯邪恶** | 有苦衷/被控制/心魔缠身 |
| **立体动机** | 反派视角也有合理性 |

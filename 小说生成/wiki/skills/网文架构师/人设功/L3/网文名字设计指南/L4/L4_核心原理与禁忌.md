# 人设功·L4·核心原理与禁忌

> **版本**：v1.0 | **父级**：网文名字设计指南/L4

---

## 一、核心原理：名如其人

```
名字 = 气质 = 读者的第一印象 = 角色形象的视觉化
```

| 核心原则 | 说明 |
|---------|------|
| **好读好记** | 别用生僻字，读者记不住就弃文 |
| **符合人设** | 名字=气质，温柔/强势/清冷/市井，一听就要有感觉 |
| **避免撞名** | 别满篇都是"顾、陆、沈、苏"，容易串戏 |
| **时代统一** | 古代/现代/玄幻，风格不能乱 |

---

## 二、四大禁忌（绝对避免）

### 禁忌1：生僻字
| 错误示例 | 问题 | 后果 |
|---------|------|------|
| 孙恊、张姷 | 读者不会读 | 弃文 |
| 勗攜鹽 | 生僻到离谱 | 考验读者耐心 |
| 翀、龑、彧 | 非常见字 | 阅读障碍 |

### 禁忌2：谐音梗
| 错误示例 | 谐音含义 | 问题 |
|---------|---------|------|
| 杜子腾 | 肚子疼 | 搞笑出戏 |
| 范统 | 饭桶 | 侮辱角色 |
| 史珍香 | 失真香 | 尴尬 |
| 秦寿生 | 禽兽生 | 严重问题 |

### 禁忌3：历史名人
| 绝对禁止 | 原因 |
|---------|------|
| 毛泽东、邓小平 | 政治风险 |
| 马云、马化腾 | 现代名人 |
| 秦始皇、刘邦 | 除非特定题材 |

### 禁忌4：过度中二
| 错误示例 | 适用场景 | 说明 |
|---------|---------|------|
| 天煞孤星 | 主角还行 | 配角用太尴尬 |
| 孤独求败 | 可以用 | 但要配合人设 |
| 夜殇冥 | 太过中二 | 非必要不用 |

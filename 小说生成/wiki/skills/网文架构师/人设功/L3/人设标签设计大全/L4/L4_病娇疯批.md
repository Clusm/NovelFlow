## 五、病娇/疯批人设设计

### 底层逻辑

| 特点 | 说明 |
|------|------|
| **起点高** | 智商高/颜值高/手段狠 |
| **占有欲强** | 对在意的人控制欲强 |
| **极度隐忍** | 表面平静/内心波涛汹涌 |
| **态度反差** | 对外人冷漠/对在意的人温柔 |

### 三步塑造法

```
Step 1：设计成长环境 → 塑造占有欲/控制欲/社交孤立
Step 2：安排情感依托 → 能弥补内心缺失的人
Step 3：设计黑化触发 → 外界威胁/爱慕对象逃离
```

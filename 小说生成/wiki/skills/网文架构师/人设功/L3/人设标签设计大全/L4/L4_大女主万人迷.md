## 六、大女主/人设设计

### 设计原则

| 原则 | 说明 |
|------|------|
| **野心家** | 善用手段向上攀爬 |
| **独立自强** | 不依附任何人 |
| **清醒理智** | 知道自己要什么 |
| **武力爆表** | 实力强大不靠男人 |
| **反骨少女** | 不服管有自己想法 |

### 人设类型

| 类型 | 特征 | 经典桥段 |
|------|------|---------|
| **钓系美人** | 欲擒故纵 | 主动但让对方先表白 |
| **钝感少女** | 感情迟钝 | 反应慢半拍 |
| **娇弱小白花** | 柔弱外表/坚韧内心 | 楚楚可怜但有骨气 |

## 七、双强/万人迷人设

### 双强设计

| 要素 | 说明 |
|------|------|
| **能力匹配** | 双方实力对等 |
| **人格对等** | 相互尊重 |
| **相互滋养** | 共同成长 |
| **独立目标** | 各自有追求 |

### 万人迷设计

| 要素 | 说明 |
|------|------|
| **某方面极突出** | 颜值/能力/性格 |
| **不同类型都吸引** | 忠犬/狼狗/病娇 |
| **独特记忆点** | 一句话记住 |

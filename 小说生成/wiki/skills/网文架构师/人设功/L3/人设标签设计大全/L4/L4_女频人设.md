## 三、番茄女频爆款人设（2025-2026）

### 三大关键词

| 关键词 | 说明 | 示例 |
|--------|------|------|
| **成长虐 > 狗血虐** | 贴近现实/无恶人却虐心 | 原生家庭/命运抗争 |
| **爽点要炸也要真** | 欲扬先抑+超预期满足 | 打脸但合理 |
| **女主要"不完美但真"** | 有缺点/有软肋/真实感 | 不困于情/不依附 |

### 女频爆款人设类型

| 类型 | 特征 | 经典标签 |
|------|------|---------|
| **疯批小师叔** | 毒舌欠揍+重情重义 | 五行缺德 |
| **编制大佬** | 杀伐果断+秩序重建 | 不搞恋爱 |
| **坚韧成长型** | 从惊恐到独立 | 无突然黑化 |
| **毒舌女主** | 嘴毒心善 | 带掌门铁锅炖大鹅 |
| **清醒大女主** | 不恋爱脑/专注搞事业 | 自我成长 |
| **反骨少女** | 不服管/有自己想法 | 越让怎样越不 |

### 番茄女频核心反差

```
表面：毒舌/沙雕/不完美
反差：关键时刻重情重义
执念：守护+成长+自我价值
```

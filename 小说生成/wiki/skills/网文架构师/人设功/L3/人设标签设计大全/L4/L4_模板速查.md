## 十、爆款人设模板

### 模板一：【美强惨】男女通杀

```
标签：强大多智+颜值逆天
缺口：某个悲惨过去/内心创伤
组合：强者+惨痛经历 = 让人心疼
```

### 模板二：【疯批/黑莲花】当下最火

```
标签：睚眦必报+不圣母+神经质
反差：表面正常/内心疯狂
逻辑：偏执+病态 = 疯批属性
```

### 模板三：【反差萌】沙雕搞笑

```
标签：脑回路清奇+不按套路
反差：平时不正经/关键时刻帅
逻辑：消解沉重+轻松感
```

### 模板四：【废柴逆袭】经典套路

```
标签：出身低+天赋差+被打压
金手指：系统/穿越/重生
成长：废柴→强者→打脸
```

## 十二、设计检查清单

```
□ 核心公式是否完整（身份×反差×执念）？
□ 反差是否有三层（身份/行为/秘密）？
□ 执念是否有形成原因（童年创伤/重大事件）？
□ 性格标签是否有反差设计？
□ 身份标签是否与性格匹配？
□ 能力标签是否有明确限制？
□ 缺陷标签是否有形成原因？
□ 恋爱属性是否有独特设计？
□ 名场面/记忆点是否有设计？
□ 成长弧光是否完整（起点→中点→终点）？
□ 读者能否一句话记住？
□ 是否符合目标平台风格？
□ 标签组合是否新颖独特？
□ 是否有新手禁忌需要避免？
```

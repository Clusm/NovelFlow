## 七、历史/军事人设设计

### 历史文核心特征

```
核心：穿越/改革/权谋
题材：历史/军事/架空
爽点：现代知识/制度改革/战争谋略
```

### 历史标配人设

| 要素 | 内容 |
|------|------|
| **初始身份** | 赘婿/商人/小官 |
| **金手指** | 现代知识/历史预知 |
| **执念** | 改革/守护/登顶 |
| **核心矛盾** | 旧势力vs新思想 |

### 经典案例：《赘婿》宁毅

```
身份：现代商业精英→武朝赘婿
初始：无心争霸/只想安稳
转变：被乱世逼成枭雄
执念：守护+改变
反差：对敌人狠/对妻子温柔
```

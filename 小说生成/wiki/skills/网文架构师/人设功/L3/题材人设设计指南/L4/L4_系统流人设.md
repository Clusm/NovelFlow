## 四、系统流人设设计

### 系统流核心特征

```
核心：绑定系统+完成任务+获得奖励
题材：都市/玄幻/游戏/无限流
爽点：即时反馈/趣味设定/装逼
```

### 系统流标配人设

| 要素 | 内容 |
|------|------|
| **初始身份** | 普通人/废柴/被看不起 |
| **金手指** | 系统（签到/任务/扮演） |
| **反差** | 内心吐槽/表面装逼 |
| **执念** | 变强/完成任务/回家 |

### 系统类型与人设搭配

| 系统类型 | 人设特点 |
|---------|---------|
| **签到系统** | 淡定/低调/闷声发大财 |
| **任务系统** | 积极/被迫营业/吐槽 |
| **扮演系统** | 双重人格/演技/反差 |
| **选择系统** | 选择困难/后果反差 |

### 经典案例：《战神别融合度》

```
主角：陈宇
身份：大帝战神（穿越者）
反差：表面威严/内心吐槽
执念：维持人设+保护家人
笑点：被迫装逼+融合度机制
```

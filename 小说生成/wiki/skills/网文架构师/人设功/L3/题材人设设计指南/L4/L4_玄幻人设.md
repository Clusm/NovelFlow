## 三、玄幻/修仙文人设设计

### 玄幻标配人设模型

```
身份：废柴杂役/外门弟子/被退婚
金手指：系统/老爷爷/特殊体质
目标：修炼变强/复仇/守护
爽点：越级打怪/打脸天才/逆袭
```

### 废柴逆袭型

| 要素 | 内容 |
|------|------|
| **初始身份** | 杂役/废物/被欺压 |
| **反差** | 实际有隐藏天赋 |
| **执念** | 证明自己/复仇 |
| **成长弧光** | 废柴→强者→守护者 |

### 天才碾压型

| 要素 | 内容 |
|------|------|
| **初始身份** | 天才/少主/有背景 |
| **反差** | 表面狂傲/内心孤独 |
| **执念** | 登顶/守护/探索 |
| **成长弧光** | 天才→更高峰→责任 |

### 穿越异界型

| 要素 | 内容 |
|------|------|
| **初始身份** | 穿越者/带现代知识 |
| **反差** | 思想现代/行为古代 |
| **执念** | 回家/适应/掌控 |
| **成长弧光** | 不适应→融入→超越 |

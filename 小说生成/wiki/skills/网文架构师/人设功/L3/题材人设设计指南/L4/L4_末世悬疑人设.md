## 五、末世/无限流人设设计

### 末世文核心特征

```
核心：生存压力/资源争夺/异能觉醒
题材：末世/废土/丧尸/末日
爽点：异能升级/基地崛起/人类希望
```

### 末世标配人设

| 要素 | 内容 |
|------|------|
| **初始身份** | 普通人/退役兵/科研员 |
| **金手指** | 异能/空间/重生记忆 |
| **执念** | 活下去/守护/重建 |
| **核心矛盾** | 人性vs生存 |

### 反套路设计

| 类型 | 传统 | 反套路 |
|------|------|-------|
| **圣母** | 无条件救人 | 有底线/有代价 |
| **强者** | 无敌孤独 | 社恐/不适应 |
| **普通人** | 纯受害者 | 有手段/有成长 |

## 六、悬疑/推理人设设计

### 悬疑文核心特征

```
核心：谜题/推理/反转
题材：推理/惊悚/探险
爽点：解开谜题/揭露真相/智斗
```

### 悬疑标配人设

| 要素 | 内容 |
|------|------|
| **初始身份** | 侦探/警察/普通人 |
| **金手指** | 观察力/推理能力/特殊能力 |
| **执念** | 解谜/揭露真相/复仇 |
| **核心矛盾** | 迷雾vs真相 |

### 悬疑主角类型

| 类型 | 特点 |
|------|------|
| **冷静理智型** | 分析推理/不动感情 |
| **感性共鸣型** | 洞察人心/共情能力强 |
| **双重人格型** | 白天夜晚不同人格 |

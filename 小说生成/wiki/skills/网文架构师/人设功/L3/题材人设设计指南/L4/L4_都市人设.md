## 二、都市文人设设计

### 都市三大类型

| 类型 | 核心特点 | 人设定位 |
|------|---------|---------|
| **新媒体文** | 节奏快/情绪强/章节卡点 | 战神/赘婿/首富 |
| **无线文** | 拉仇恨/装逼打脸 | 神医/奶爸/鉴宝 |
| **脑洞文** | 设定新颖/趣味性 | 系统/神豪/直播 |

### 1. 战神文主角

**题材特点**：王者归来/多重身份/守护

| 身份层次 | 内容 |
|---------|------|
| **金手指身份** | 战神/首富/战神 |
| **明面身份** | 废物/赘婿/普通人 |

**标配人设**：
```
性格：冷静/杀伐果断/深情
反差：对外人狠/对家人温柔
执念：守护/复仇
标配：老婆孩子热炕头
```

### 2. 赘婿文主角

**题材特点**：小人物/家庭矛盾/翻身

**底层逻辑**：
```
女婿=小人物身份（普遍性）
金手指=隐藏大佬身份
矛盾=家庭地位低+社会地位低
爽点=打脸+翻身
```

**标配人设**：
```
明面：废物/被看不起/窝囊
真实：隐藏身份/实力强大
反差：平时窝囊/关键时刻霸气
执念：守护妻子/证明自己
```

### 3. 神医文主角

**题材特点**：医术/救人/扮猪吃虎

**标配人设**：
```
身份：废物/被退婚/被看不起
技能：绝世医术/疑难杂症
反差：表面废物/实际神医
执念：救人/复仇/守护
```

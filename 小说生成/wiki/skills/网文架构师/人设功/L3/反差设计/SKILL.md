## 📚 反差设计公式

### 核心公式
```
身份标签 + 反差秘密 + 执念目标 + 致命弱点
```

### 四要素详解

| 要素 | 定义 | 示例（陆言） |
|------|------|-------------|
| **身份标签** | 最直接的外在特征 | 外卖小哥 |
| **反差秘密** | 内心深处与外表相反 | 怂包内心有尊严 |
| **执念目标** | 穷其一生追求 | 证明自己不是废物 |
| **致命弱点** | 被戳中就崩溃 | 恐高症/苏小灵 |

### 五种反差类型

| 类型 | 模板 | 示例 |
|------|------|------|
| **表面vs内心** | 嘴贱但善良 | 陆言嘴贱但心善 |
| **能力vs性格** | 强但怂 | 战斗天才但社交废物 |
| **身份vs行为** | 高身份做低身份事 | 总裁送外卖/神仙捡垃圾 |
| **过去vs现在** | 过去狠现在怂/反之 | 曾大佬现怕事/曾废物现无敌 |
| **公开vs秘密** | 公开身份/秘密目标 | 正道人士/邪教卧底 |

## 📚 成长弧光设计

### 三阶段模型

| 阶段 | 状态 | 典型事件 | 内心变化 |
|------|------|---------|---------|
| **起点** | 初始缺陷 | 故事开始 | "我是废物" |
| **中点** | 经历磨砺 | 第一次成长/失败 | "我想改变" |
| **终点** | 完成转变 | 高潮对决 | "我就是XX" |

### 成长可视化方法

| 方法 | 示例 |
|------|------|
| **修为实质化** | 修仙文：灵气雾状→金丹期凝为实体 |
| **物品升级** | 从二手电动车→定制超跑钥匙串 |
| **服装变化** | 从帆布鞋体恤衫→西装革履 |

### 黑化/觉醒触发点（设置3个关键事件）

| 触发类型 | 示例 |
|---------|------|
| **亲友背叛** | 女主被亲生父母送人换取利益 |
| **力量反噬** | 修炼时陷入绝境 |
| **信仰崩塌** | 发现一直敬仰的大善人是幕后黑手 |
| **隐藏开关** | 童年记忆封印/血脉诅咒触发 |

## 📚 记忆点强化设计

### 标志性行为设计

| 类型 | 示例 |
|------|------|
| **语言特征** | 被围困时说"轮到某某一千里不留行了" |
| **动作标签** | 转笔时必失误/紧张就啃衣角 |
| **时间标签** | 每逢月圆夜功力尽失 |

### 反差设定（反差萌）

| 类型 | 示例 |
|------|------|
| **视觉反差** | 娇小可爱女生用比她还高的大铁锤 |
| **行为反差** | 高冷人设有奇怪笑点/战力天花板的生活白痴 |
| **语言反差** | 硬汉人设口头禅是"你讨厌死了啦" |

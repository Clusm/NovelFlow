# 卷纲设计指南

## 元数据
> 版本：v2.0 | 来源：联网学习-白金大神/网文俱乐部/长篇架构 | 定位：L3子skill

## 执行流程
```
总纲 → 卷数划分 → 卷纲撰写 → 阶段划分 → 与下卷衔接 → 完成
```

## 白金大神卷纲法（百万字长篇标配）

### 长篇架构三幕式（白金大神标准）

| 幕 | 篇幅 | 占比 | 核心任务 |
|---|------|------|---------|
| **第一幕：起步期** | 50万字 | 25% | 吸引读者、建立世界观、第一次成长 |
| **第二幕：发展期** | 120万字 | 60% | 推动剧情、深化矛盾、多次高潮 |
| **第三幕：冲刺期** | 30万字 | 15% | 解决矛盾、收束伏笔、终极对决 |

### 分卷分篇结构（斗破苍穹/凡人修仙传模板）

**200万字小说分卷示例**：
```
第一卷：凡人篇（50万字）
├─ 开局建立（10万字）
├─ 崛起之路（20万字）
└─ 凡人巅峰（20万字）

第二卷：修士篇（60万字）
├─ 初入修真界（20万字）
├─ 宗门争锋（20万字）
└─ 修士巅峰（20万字）

第三卷：仙人篇（60万字）
├─ 飞升仙界（20万字）
├─ 仙界风云（20万字）
└─ 仙人巅峰（20万字）

第四卷：终极篇（30万字）
├─ 最终对决（20万字）
└─ 完美结局（10万字）
```

### 卷纲三要素模板

| 要素 | 说明 | 示例 |
|------|------|------|
| **本卷目标** | 这一卷主要推进什么 | "主角加入宗门，开始修炼之路" |
| **关键事件** | 最重要的2-3个事件 | "宗门大比夺冠/发现身世秘密/与女主相遇" |
| **卷末状态** | 卷末角色关系推到何种状态 | "实力提升、真相揭露、情感升温" |

## 卷纲撰写标准

### 每卷必须写清楚的3件事

1. 这一卷主要推进什么
2. 这一卷最重要的2-3个事件是什么
3. 卷末要把角色关系推到什么状态

### 字数要求

| 类型 | 每卷字数 | 章节数 |
|------|----------|--------|
| 第一卷 | 30-50万字 | 吸引读者，不宜太长 |
| 中间卷 | 50-80万字 | 主要剧情推进 |
| 最后卷 | 20-40万字 | 冲刺结局，不宜拖沓 |

---

## 多主线并行结构（白金大神标配）

### 主线类型与占比

| 主线 | 占比 | 内容 |
|------|------|------|
| **实力提升线** | 60% | 炼气→筑基→金丹→元婴→化神 |
| **情感发展线** | 20% | 初遇→相识→相知→相爱 |
| **势力建立线** | 15% | 散修→加入宗门→建立势力 |
| **复仇/目标线** | 5% | 仇人出现→调查→逐个击杀 |

### 主线交织法则

```
第一卷：主线1为主，主线2、4埋伏笔
第二卷：主线1、2并行，主线3开始
第三卷：多线并进，逐步收束
第四卷：所有主线汇聚，终极对决
```

## 波浪式节奏（白金大神控制法）

### 节奏曲线

```
高潮 ───┐    ┌───高潮───┐    ┌───最高潮
        │    │          │    │
平稳────┘────┘──平稳──┘────┘────平稳
  开局    小高潮    过渡    中高潮    过渡    终高潮
```

### 节奏调控方法

| 加快节奏 | 放慢节奏 |
|----------|----------|
| 缩短单章字数 | 增加单章字数 |
| 增加对话比例 | 增加景物描写 |
| 增加动作描写 | 增加内心独白 |
| 制造紧张感 | 日常轻松剧情 |
| 快速切换场景 | 详细展开细节 |

### 张弛之道

```
紧张后必有放松
高潮后必有过渡
打斗后加日常
虐心后加甜蜜
一张一弛，文武之道
```

## 卷间衔接设计

### 衔接三要素

| 要素 | 说明 | 实现方式 |
|------|------|---------|
| **收尾** | 本卷收束方式 | 解决核心冲突+留下次要悬念 |
| **钩子** | 下卷开启伏笔 | 身份揭示/新势力登场/新目标抛出 |
| **过渡** | 自然过渡 | 主角整理收获、展望未来 |

### 大地图转换

| 阶段 | 写法 |
|------|------|
| 地图A收尾 | 埋下下一个地图的伏笔 |
| 过渡章节 | 主角整理收获、展望未来 |
| 地图B开场 | 新环境、新挑战、新人物 |

---

## 伏笔三层次（白金大神管理法）

| 类型 | 埋设周期 | 回收节点 | 数量控制 |
|------|---------|---------|---------|
| **短期** | 10-30万字内 | 每卷3-5个 | 及时回收 |
| **中期** | 50-100万字 | 每卷1-2个 | 逐步揭露 |
| **长期** | 贯穿全文 | 大结局 | 全文3-5个 |

## 卷检查清单

```
□ 本卷目标是否清晰？
□ 是否有2-3个关键事件？
□ 卷末角色状态是否有变化？
□ 是否推进了主线？
□ 与上卷是否有逻辑衔接？
□ 是否为下卷埋下伏笔？
□ 字数是否在30-80万字？
□ 起承转合是否完整？
□ 是否有情感线推进？
□ 是否有升级感？
□ 爽点密度是否合适？
□ 多主线是否交织合理？
□ 伏笔是否有回收计划？
```

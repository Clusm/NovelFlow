## 白金大神章纲法（天蚕土豆/唐家三少/我吃西红柿）

### 三级大纲体系

| 级别 | 内容 | 字数 | 说明 |
|------|------|------|------|
| **一级大纲** | 一句话故事核心 | 50字内 | 永远不能跑偏 |
| **二级大纲** | 分卷规划 | 10-20字/卷 | 核心冲突+阶段目标+爽点节点 |
| **三级大纲** | 单章章纲 | 200-300字/章 | 本章任务+冲突推进+爽点+钩子 |

### 斗破苍穹章纲模板（可套用）

**第1章 陨落的天才**：
- 纳兰嫣然上门退婚（500字）
- 当众羞辱（800字）
- 主角忍辱（500字）
- 发誓三年后复仇（200字）

**第2章 斗气大陆**：
- 交代世界秩序（500字）

- 总计2000字

## 章纲核心要素

### 三要素模板

| 要素 | 说明 | 示例 |
|------|------|------|
| **开场** | 本章开场场景/情境 | "主角在雨夜的废弃仓库醒来" |
| **关键动作** | 本章需完成的2-3个关键事件 | "发现系统激活、与追兵战斗、获得线索" |
| **结尾钩子** | 章末留下的追读悬念 | "门后站着一个他从未想过会再见的人" |

### 章纲撰写标准

**字数要求**：2000-3000字/章（番茄/起点标准）

**内容要求**：
- 本章冲突是什么
- 情节走向哪里
- 章末落点/钩子

## 起承转合四步法

| 步骤 | 功能 | 内容 | 字数占比 |
|------|------|------|---------|
| **起** | 承接上章/开场 | 自然衔接上一章结尾，或新场景开启 | 10-15% |
| **承** | 情节推进 | 主角行动、障碍出现、心理变化 | 50-60% |
| **转** | 情节转折 | 意外发生、冲突升级、高潮来临 | 15-20% |
| **合** | 章节收束 | 解决部分问题、留下新悬念 | 10-15% |

## 黄金三章公式（白金大神标配）

### 万能公式：生死危机 + 身份碾压 + 金手指 + 未来爽饼

**第一章（生死危机）**：
```
开头：主角陷入绝境（退婚/被欺/系统激活/危机降临）
目标：让读者产生共情和不甘
技巧：藏"倔强细节"——被嘲讽时指甲掐进掌心、饿肚子也坚持练功
```

**第二章（金手指）**：
```
承接：立刻给甜，金手指即时上线
目标：让读者知道"他要逆袭"
技巧：激活必须和"惨"挂钩——血祭/濒死/绝境
```

**第三章（画爽饼）**：
```
推进：用金手指小试牛刀
目标：让读者预判"接下来要打脸了"
技巧：留大钩子——"三年之约"、"秘境阴谋"
```

## 五种章末钩子

| 钩子类型 | 效果 | 示例 |
|----------|------|------|
| **悬念式** | 留疑问 | "他没想到，门后站着的竟是…" |
| **冲突式** | 制造对立 | "话音未落，一道寒光袭来" |
| **收获式** | 期待奖励 | "终于找到了传说中的…" |
| **危机式** | 紧张感 | "糟了，他们发现了" |
| **揭示式** | 震惊感 | "直到今天他才知道真相是…" |

## 爽点密度控制（白金大神标准）

| 阶段 | 章节范围 | 爽点频率 | 类型 |
|------|----------|----------|------|
| 前期 | 1-30万字 | 每章1个小爽点 | 打脸、升级、获得 |
| 中期 | 30-100万字 | 2-3章一个爽点 | 大战、突破、揭秘 |
| 后期 | 100万字+ | 根据节奏调整 | 决战、高潮、收尾 |

### 常见爽点类型

| 类型 | 说明 | 经典场景 |
|------|------|---------|
| **打脸爽** | 证明自己，打脸嘲讽者 | 废物少爷击败天才 |
| **升级爽** | 实力提升，突破瓶颈 | 突破斗者、筑基、金丹 |
| **获得爽** | 得到宝物、功法、美女 | 获得传承、捡到神器 |
| **击败爽** | 战胜强敌，赢得尊重 | 跨阶击败BOSS |
| **装逼爽** | 展现实力，震惊众人 | 当众展示隐藏实力 |

## 数据增长点规划（番茄小说）

| 阶段 | 章节范围 | 目标 | 操作 |
|------|----------|------|------|
| 留存 | 1-3章 | 留住读者，追读率>35% | 每章结尾留悬念 |
| 完读 | 4-10章 | 提升完读 | 每500字小转折 |
| 互动 | 11-20章 | 引发评论 | 安排争议点 |
| 付费 | 21-30章 | 准备付费 | 安排大高潮 |

## 章纲检查清单

```
□ 开场是否承接上章？
□ 是否有2-3个关键动作？
□ 起承转合是否完整？
□ 章末钩子是否强劲？
□ 是否符合卷纲要求？
□ 字数是否在2000-3000字？
□ 是否有爽点埋设？
□ 是否为下章做好铺垫？
□ 是否完成本章核心任务？
□ 金手指是否有展示？
□ 是否推进了主线？
```

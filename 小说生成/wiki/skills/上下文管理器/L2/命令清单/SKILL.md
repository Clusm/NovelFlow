# 核心命令清单

> **版本**：v2.0 | **定位**：5个核心命令 | **来源**：联网学习

---

## 命令1：项目初始化

| 项目 | 内容 |
|------|------|
| 触发 | 新项目启动 |
| 动作 | 创建文件夹 + 存储设定 + 初始化伏笔表 + 初始化人物表 |
| 输出 | 项目记忆库 |

---

## 命令2：上下文恢复

| 项目 | 内容 |
|------|------|
| 触发 | 原创作家开始新章节 |
| 动作 | 读取档案 + 加载伏笔 + 加载人物 + 检查上章 |
| 输出 | 章节上下文文件 |

---

## 命令3：一致性检查

| 项目 | 内容 |
|------|------|
| 触发 | 原创作家完成章节 |
| 动作 | 对比设定 + 检测矛盾 |
| 输出 | 冲突预警（如有） |

---

## 命令4：伏笔更新

| 项目 | 内容 |
|------|------|
| 触发 | 原创作家完成章节 |
| 动作 | 扫描伏笔 + 更新表 + 检查超时 |
| 输出 | 伏笔状态报告 |

---

## 命令5：大纲偏离检查

| 项目 | 内容 |
|------|------|
| 触发 | 原创作家完成章节 |
| 动作 | 对比章纲 + 计算偏离度 |
| 输出 | 偏离预警（>30%） |

---

## 命令6：向量检索【新增】

| 项目 | 内容 |
|------|------|
| 触发 | 原创作家开始章节/查询设定 |
| 动作 | 加载向量库 + 语义检索 + 返回结果 |
| 输入 | 查询文本、topK数量 |
| 输出 | 检索结果（带相似度分数） |
| 集成 | novel-vector-search/vector.js |

---

## 命令调用时机

```
架构师完成 → 命令1
拆书完成 → 更新记忆库
创意完成 → 更新记忆库
作家开始 → 命令2 + 命令6（向量检索）
作家完成 → 命令3+4+5
审核 → 数据支持
发布 → 日志归档
```

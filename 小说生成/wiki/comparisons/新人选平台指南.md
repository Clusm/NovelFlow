# 新人选平台指南

## 为什么要对比

新人作者面临的第一道关卡不是"写什么"，而是"投哪里"。平台选错了，轻则数据惨淡、积极性受挫，重则浪费半年时间从头再来。本指南帮助新人作者根据自身情况，选择最适合自己的起步平台。

## 起点中文网 — 精品路线

### 适合人群

- 有一定写作基础，理解网文基本套路
- 目标是IP改编，愿意长线经营
- 能承受前期低收益，有完本经验
- 擅长复杂设定、长线伏笔、世界观架构

### 优劣势

| 优势 | 劣势 |
|------|------|
| IP天花板最高（《庆余年》《诡秘之主》） | 签约门槛高，竞争激烈 |
| 老白读者付费意愿强 | 订阅分成比例低（作者到手约40-50%） |
| 编辑体系成熟，内投有反馈 | 新人曝光机会少，容易被埋 |
| 品牌溢价：起点作者身份业内认可 | 日更压力不小（3000字/天起） |

### 新人常见误区

1. **直发裸奔**：满6000字直接发表，没有内投，编辑看不到
2. **内投没准备**：内投需要10万字大纲+2万字正文，没准备好就乱投会被拉黑
3. **追读不够就上架**：起点追读数据决定推荐量，新书期断更=死刑
4. **写自己想写的**：起点老白口味刁，不是你认为好的就能火

### 新人上手建议

```
新人起步路径（推荐）：
1. 先在番茄/飞卢写一本完本（积累经验、验证套路）
2. 有完本经验后，再内投起点（有筹码）
3. 如果执意起点：先写30万字存稿，再内投编辑
```

## 番茄小说 — 新人起步最优解

### 适合人群

- 完全零基础的新人作者
- 想快速验证自己的书有没有市场
- 能接受日更4000字以上的高强度
- 擅长快节奏、短情节、强反转

### 优劣势

| 优势 | 劣势 |
|------|------|
| 签约门槛低，算法分发公平 | 广告分成收入不稳定 |
| 今日头条系导流，流量大 | 算法决定生死，内容质量之外有运气成分 |
| 新人也有机会被推荐 | 无明确编辑对接，靠自己摸索 |
| IP改编势头猛（《十日终焉》《斩神》） | 盗版问题影响收益 |

### 算法核心指标

番茄小说的推荐逻辑围绕两个核心指标：

1. **完读率** = 读者读完了前几章
   - 前3章完读率决定初始推荐量
   - 前10章完读率决定是否进入更大流量池
   - 目标：前3章完读率 > 40%

2. **追读率** = 读者是否持续追更
   - 每日追更读者比例决定续推荐
   - 断更是追读率的最大杀手

### 新人上手建议

```
新人起步路径（推荐）：
1. 前3章必须有一个强钩子（危机/金手指/身份揭秘）
2. 第8章、第15章是两个算法关键节点，务必有高潮
3. 日更4000字是基本盘，断更超过3天算法降权
4. 完本后立刻开新书，不要等（养笔名效率低）
```

## 飞卢小说 — 快速变现路线

### 适合人群

- 手速快（日更一万字不费劲）
- 想快速验证市场，不追求IP
- 纯小白爽文写手（退婚、打脸、装逼）
- 需要稿费正向反馈激励

### 优劣势

| 优势 | 劣势 |
|------|------|
| VIP票实时结算，爆款周结算 | IP价值几乎为零 |
| 编辑响应快（QQ群） | 日更要求极高（5000-10000字/天） |
| 小白读者集中，套路成熟好上手 | 不适合长线创作，天花板明显 |
| 防盗能力强 | 订阅价格高，读者消费压力大 |

### 飞卢生存法则

1. **黄金五章**：前5章必须出第一个爽点，否则直接扑
2. **日更不能断**：断更1天收入跌30%，断更3天基本废
3. **防盗是生命线**：飞卢防盗做得好，但你的书火了还是有盗版
4. **套路要新**：飞卢老白看腻了套路，要有自己的创新点

### 新人上手建议

```
新人起步路径（推荐）：
1. 先研究飞卢天榜（前50名），拆解他们的黄金五章
2. 选择一个热门题材（都市/玄幻/同人）跟风
3. 前5章必须出爽点，不要铺垫
4. 有收益后立刻开第二本，不要恋战（飞卢不适合养笔名）
```

## 新人常见问题

### Q1：同时在多个平台发布可以吗？

**不建议**。大多数平台有独家协议，签约后作品只能在本站发布。签约前可以发不同的平台试水，但一旦签了独家，就要遵守协议。

### Q2：番茄和飞卢哪个更容易出成绩？

**番茄更适合新人**。番茄的算法分发给了新人曝光机会，飞卢虽然变现快但需要极高的日更强度，新人容易崩溃。

### Q3：起点签约被拒了怎么办？

1. 检查是不是内投格式不对（大纲+正文缺一不可）
2. 换个题材再投（起点编辑口味不同）
3. 先去番茄/飞卢写一本完本，有了经验再投

### Q4：日更断了几天还能恢复吗？

- **番茄**：断更3天内影响不大，超过3天算法降权明显
- **起点**：新书期断更=推荐减少，上架后断更=读者流失
- **飞卢**：断更1天收入跌30%，断更3天基本废

### Q5：新人第一本书应该写多少字？

| 类型 | 建议字数 | 说明 |
|------|---------|------|
| 番茄风 | 30-50万字 | 快节奏，完本快，数据不好及时切 |
| 传统升级流 | 100万字+ | 长线精品，低于100万字不出成绩 |
| 飞卢体 | 30-50万字 | 短平快，完本就开下一本 |
| 女频 | 30-50万字 | 晋江标准完本字数 |

## 总结

新人选平台的核心逻辑：

1. **零经验**：先去番茄，积累经验+验证市场
2. **有经验**：根据目标选择——IP向=起点，变现向=飞卢
3. **女频**：直接晋江
4. **最重要的一点**：选定平台后，不要频繁换平台，坚持写完一本书再说

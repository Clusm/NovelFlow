# 信息差碾压模组

## 基本信息

- **类别**：认知碾压类
- **核心机制**：所有人困于局内，唯有主角知晓全局真相、规则漏洞或未来剧情
- **满足需求**：优越需求（众人皆醉我独醒）
- **情绪阶段**：压抑（众人绝望）→ 爆发（主角点破）→ 震撼（全场回过神）

## 核心机制详解

信息差碾压的本质是**认知维度差**。主角站在更高的认知层面，看穿了局中人都看不见的东西。这种碾压不是实力的碾压，而是**信息与判断力的碾压**。

### 信息差的类型

| 类型 | 主角知道的内容 | 经典场景 |
|-----|--------------|---------|
| **重生信息差** | 已知的未来剧情走向 | 知道秘境中哪个选择通向宝藏 |
| **穿书信息差** | 原著角色的真实身份/命运 | 识别隐藏的幕后黑手 |
| **先知信息差** | 规则漏洞/隐藏机制 | 发现丹药配方的致命缺陷 |
| **专业信息差** | 超越当前时代的知识 | 识别古董真伪、医术盲区 |
| **人心信息差** | 没人知道的秘密/动机 | 揭穿看似好人的阴谋 |

### 爽感产生机制

1. **困局构建**：先描写众人陷入绝境的绝望感，积蓄情绪
2. **沉默观察**：主角沉默不语，旁人以为是放弃
3. **轻描淡写**：主角说一句"常识性"的话，或指出最明显的破绽
4. **恍然大悟**：旁观者从困惑到震惊到恐惧
5. **优越锁定**：主角的平静与周围人的震撼形成鲜明对比

## 经典案例

### 网文经典

- 《全职高手》：叶修对游戏的理解远超同龄人，能看到别人看不到的战术机会
- 《诡秘之主》：主角利用穿越者的历史知识预判社会变革
- 《我欲封天》：孟浩以丹道宗师的身份碾压同龄人
- 《庆余年》：范闲利用现代法律知识和文学作品震惊古人

### 细分流派

| 流派 | 信息差来源 | 代表作品 |
|-----|----------|---------|

## AI 写作要点

### 核心写作公式

```
绝望场景 → 主角沉默 → 一句话点破 → 全场震撼 → 淡淡收尾
```

### 详细操作指南

**第一步：构建绝望困局（200-500字）**

- 描写困境的"无解感"，让读者也觉得没有办法
- 多个权威人物都表示放弃
- 用具体数字和细节增加真实感（如：十二位长老联合推演均失败）
- 配角的绝望反应要有层次：先愤怒 → 再绝望 → 最后麻木

**第二步：沉默观察期（100-200字）**

- 主角的沉默被误解为"吓傻了"或"放弃了"
- 反派的得意嘲讽
- 旁观者的同情或幸灾乐祸
- **关键**：此时主角的表情应该是平静甚至带笑意的

**第三步：点破时刻（300-500字）**

- 主角开口，用一句看似"显而易见"的话指出破绽
- 句式特点：简短、平静、不带情绪
- 示例台词：
  - "这个阵法的核心不在兑位，在乾位。"
  - "他说的三七分成，是毛利，不是净利。"
  - "你们都忽略了一个前提：氧气足够。"
  - "答案很简单，题目印错了。"

**第四步：震惊与回味（200-300字）**

- 旁观者的反应：愣住 → 质疑 → 验证 → 恐惧
- 使用"不可能！"、"你怎么知道的！"等台词
- 多个配角的连续震惊反应
- 权威人物（如长老、专家）推翻自己的判断
- **高级技巧**：让反对主角的人亲自验证后无法反驳

**第五步：淡然收尾（50-100字）**

- 主角的反应：微微一笑，转身离去，或者继续做自己的事
- 不解释原理（或者轻描淡写一句）
- 留白，让震撼持续发酵

### 高级写作技巧

**1. 信息差的层次感**

不要一次性揭露所有信息，让主角的信息差有层次：
- 第一层：主角知道，其他人不知道
- 第二层：主角知道，还知道其他人不知道
- 第三层：主角知道，其他人知道自己不知道

**2. 反派的认知局限**

让反派的失败显得"合理"，因为他们的认知确实有盲区：
- 反派是基于正确的信息做判断
- 但主角的信息更全面
- 这种失败比"反派突然变傻"更有爽感

**3. 团体信息差的叠加**

可以让主角不只是碾压单个反派，而是碾压整个权威群体：
- 全场权威都束手无策
- 主角一句话终结
- 这种"群体碾压"比单点碾压爽感更强

**4. 信息差的来源暗示**

不要只写"主角知道答案"，要暗示为什么主角知道：
- 重生的记忆
- 穿越的知识
- 特殊的观察力
- 隐藏的师承

## 常见剧情模式

### 模式一：秘境/遗迹探索

```
【困境】众人进入秘境，按照攻略摸索，三天后仍未找到出口，死亡逼近
【主角】一直在默默观察，最后说："这个秘境的出口不在深处，在入口。"
【结果】原来入口处的"普通石壁"是传送阵
```

### 模式二：商业/职场博弈

```
【困境】谈判陷入僵局，我方被逼到绝境，要签不平等条约
【主角】翻阅合同后说："注意第十二条的最后一句。"
【结果】对方律师当场脸色大变，己方反败为胜
```

### 模式三：医术/毒药破解

```
【困境】患者生命垂危，所有名医会诊均束手无策
【主角】诊脉后说："这不是寒症，是计量中毒。"
【结果】调整药方后患者苏醒
```

### 模式四：阴谋揭露

```
【困境】主角被指控犯罪，所有证据指向主角，无人相信其清白
【主角】在法庭上说："请允许我传唤真正的证人。"——然后揭穿伪证
【结果】反派崩溃，局势逆转
```

## 变体与组合

### 组合模组

| 组合对象 | 结合效果 |
|---------|---------|
| [[退婚打脸模组]] | 信息差揭露+打脸，先用信息差证明自己，再碾压对手 |
| [[格局碾压模组]] | 从信息差升级到格局差，信息 → 判断 → 布局 |
| [[马甲曝光模组]] | 信息差+身份曝光，揭示主角知道的比所有人多 |
| [[权力反转模组]] | 信息差揭露阴谋后，翻转权力结构 |

### 变体分类

| 变体 | 特点 |
|-----|------|
| **渐进式** | 主角多次展示信息差，每次碾压一小点，逐步积累 |
| **爆发式** | 全场憋屈后，主角一句话全场秒杀 |
| **连续式** | 主角连续点破多个"不可能"，每破一次全场震惊一次 |
| **反转式** | 先让读者以为主角是错的，最后证明主角从一开始就是对的 |

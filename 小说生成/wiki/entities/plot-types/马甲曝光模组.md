# 马甲曝光模组

## 基本信息

- **类别**：身份反转类
- **核心机制**：平时低调或被人轻视的身份，被揭露为令人战栗的大佬
- **满足需求**：优越感需求（身份碾压）、掌控需求（幕后大佬）、公平需求（真相大白）
- **情绪阶段**：轻视（不为人知）→ 铺垫（蛛丝马迹）→ 揭露（身份曝光）→ 震惊（全场死寂）→ 崩溃（既往态度反转）

## 核心机制详解

马甲曝光的本质是**"身份落差带来的认知颠覆"**。平时越被轻视，曝光后反差越大。读者在阅读时会代入主角，享受"你们看不起我，结果我才是你们惹不起的人"的极致爽感。

### 马甲曝光的类型

| 类型 | 马甲身份 | 曝光效果 | 代表作品 |
|-----|---------|---------|---------|
| **战神归家** | 战场王者/殿主/战神 | 家人被欺辱，曝光后敌人跪地求饶 | 《都市全能奶爸》类战神文 |
| **龙王赘婿** | 龙殿之主/隐世家族 | 被妻子家轻视，曝光后老丈人跪舔 | 《龙王令》类赘婿文 |
| **隐藏掌门** | 超级宗门传人/古武世家 | 被宗门轻视，曝光后宗主跪拜 | 《师父别低调》类掌门文 |
| **首富隐藏** | 世界首富/财阀之主 | 被当成穷人，曝光后打脸拜金者 | 《史上最强炼气期》类神医文 |
| **学神低调** | 顶级学者/教授/大牛 | 被当成学渣，曝光后同学震惊 | 《全职艺术家》类文艺文 |
| **继承者隐匿** | 顶级家族继承人 | 被赶出家门，曝光后家族跪迎 | 《李显》类继承人文 |

### 爽感产生机制

1. **轻视铺垫**：充分描写他人对主角马甲身份的轻视与不屑
2. **蛛丝马迹**：埋下伏笔，暗示主角真实身份的不凡
3. **渐进揭露**：通过证据层层揭露身份
4. **震惊定格**：曝光瞬间全场死寂，凝固在震惊中
5. **态度崩塌**：反派经历完整的"不屑→怀疑→震惊→恐惧→崩溃"链条

## AI 写作要点

### 核心写作公式

```
轻视铺垫（憋屈）→ 蛛丝马迹（伏笔）→ 触发事件（曝光契机）→ 渐进揭露（仪式感）→ 全场震惊（定格）→ 态度崩塌（爽感高潮）
```

### 详细操作指南

**第一步：轻视铺垫（300-500字）**

- 充分描写反派对主角"真实身份"的轻视
- 描写具体的轻视事件，越具体越爽：
  - "就凭你？一个废物也配？"
  - "你知道什么是真正的权势吗？给你一辈子也接触不到"
  - "我一句话就能让你从这座城市消失"
- 旁观者的冷漠或幸灾乐祸
- 主角的隐忍，但要让读者知道主角在演戏
- **关键**：让读者积累怒气，等待爆发

**第二步：蛛丝马迹（200-300字）**

- 埋下伏笔，暗示主角身份的不凡
- 常用伏笔：
  - 一个不起眼的信物（玉佩、戒指、令牌）
  - 某些人见到主角后的异常反应
  - 主角不经意间展现的见识/能力
  - 手机来电显示"殿主，您该回来了"
- 让读者隐隐感觉主角不简单，但反派还在作死
- **关键**：伏笔要自然，不能太刻意

**第三步：触发事件（200-300字）**

- 某个事件成为身份曝光的导火索
- 常用触发：
  - 反派招惹了不该招惹的人，牵连到主角
  - 主角的"靠山"登场找人
  - 关键信物被暴露
  - 场合特殊，必须亮明身份
- 反派还浑然不知危险将近
- **关键**：触发事件要合理，让曝光显得顺理成章

**第四步：渐进揭露（300-500字）**

- 通过一连串证据层层揭露身份，这是最关键的仪式感
- 揭露的层次递进：
  - 第一层：有人对主角表现出异常恭敬
  - 第二层：某个大人物出现，称呼主角特殊称谓
  - 第三层：信物被认出，全场哗然
  - 第四层：最终确认，所有人震惊
- 每个层次的反应：
  - 反派："你们搞错了吧？这个人就是个废物"
  - 旁人开始怀疑人生
  - 最终确认，全场死寂

**第五步：全场震惊（200-300字）**

- 曝光瞬间的定格描写
- 全场死寂，落针可闻
- 反派的嘴还张着，凝固在"不可能"的表情
- 常用描写：
  - "空气仿佛凝固了"
  - "所有人都屏住了呼吸"
  - "他（反派）的笑容僵在脸上，像是被抽走了魂魄"
- **关键**：这个瞬间要慢，要详细，让读者充分体验爽感

**第六步：态度崩塌（300-500字）**

- 反派的完整崩溃链条：
  1. **不屑**："哈，就你？"
  2. **怀疑**："等等，你们是不是认错人了？"
  3. **震惊**："这...这不可能！"
  4. **恐惧**："我...我刚才说了什么..."
  5. **崩溃**："求...求您饶命..."
- 跪地求饶是最经典的崩溃方式
- 旁观者的态度转变：先前嘲笑主角的人开始颤抖
- 主角说一句收尾台词：
  - "现在，你可以再说一遍刚才的话"
  - "跪下"
  - "后悔吗？"
  - "有些人，你这辈子都惹不起"
- **关键**：崩溃要有层次，不能一步到位

### 高级写作技巧

**1. 马甲的多重叠加**

不要只写一层身份，多层身份叠加更爽：
- 第一层：普通身份（被轻视）
- 第二层：隐藏的大身份（震惊）
- 第三层：更恐怖的幕后身份（彻底崩溃）
- 例子：女婿/徒弟/学生 → 殿主 → 幕后之人的靠山

**2. 曝光的仪式感营造**

仪式感来自于"正式"：
- 正式场合的曝光（宴会/大会/典礼）
- 权威人物的见证
- 完整流程的揭露
- 不可辩驳的证据

**3. 反派的前后对比**

用对比强化爽感：
- 曝光前：高高在上，肆意践踏主角
- 曝光后：跪地求饶，瑟瑟发抖
- 同一张嘴，前后说的话形成讽刺

**4. 配角的反应描写**

除了反派崩溃，还要写配角反应：
- 先前帮助主角的人：欣慰、自豪
- 中立的旁观者：震惊、后悔
- 先前嘲笑的人：恐惧、颤抖

## 常见剧情模式

### 模式一：战神归家

```
【铺垫】主角隐姓埋名当上门女婿，被妻子一家人嫌弃
【日常】家人被欺辱，主角暗中出手解决小麻烦
【触发】老婆公司上市，大人物云集，前来祝贺
【揭露】主办方："恭迎殿主回归！"众人惊呆
【崩溃】前羞辱主角的富二代跪地求饶
【收尾】妻子一家震惊，发现自己嫁的竟是战神
```

### 模式二：龙王赘婿

```
【铺垫】主角被当成废物赘婿，连妻子都不正眼看他
【日常】被各种欺负，但主角从不还手（其实是不屑）
【触发】龙王殿的人来找龙王令
【揭露】龙王令出现，所有人认出那是龙王信物
【崩溃】老丈人当场下跪："您...您是龙王？！"
【收尾】妻子家族开始讨好主角，但主角只对妻子温柔
```

### 模式三：隐藏掌门

```
【铺垫】主角是宗门最底层弟子，被师兄师姐欺负
【日常】干最脏最累的活，被当成杂役
【触发】宗门大比，主角被迫上场
【揭露】一出手震惊全场，太上长老认出他是师叔祖
【崩溃】曾经欺负主角的师兄师姐跪地求饶
【收尾】主角接任掌门，但行事依旧低调
```

### 模式四：神医归来

```
【铺垫】主角被当成废物上门，被丈母娘嫌弃
【日常】偶尔露一手医术，但被当成运气
【触发】丈母娘重病，丈母娘哭着求主角出手
【揭露】主角开始治疗，使用的针法震惊所有老中医
【崩溃】医学泰斗跪地拜师："师公，请收我为徒！"
【收尾】全家震惊，发现主角竟是传说中的神医
```

### 变体分类

| 变体 | 特点 |
|-----|------|
| **单层曝光** | 一重身份，曝光即爽 |
| **多层叠加** | 多重身份，连续震惊，爽感翻倍 |
| **反转再反转** | 先以为曝光了，结果还有更深身份 |
| **被迫曝光** | 被敌人逼到不得不曝光 |
| **主动曝光** | 主角主动选择曝光，享受碾压 |

### 进阶方向

- **身份升维**：从一个小身份到更高维度的身份，逐层震惊
- **势力曝光**：不仅个人身份曝光，还展示背后势力
- **终极身份**：最后揭示主角其实是幕后BOSS的BOSS

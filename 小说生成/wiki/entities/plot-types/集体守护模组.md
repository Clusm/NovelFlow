# 集体守护模组

## 基本信息

- **类别**：情感共鸣类
- **核心机制**："犯我宗门/家园/同胞者，虽远必诛"，为保护身后之人而战
- **满足需求**：归属需求（被需要）、价值实现（成为依靠）、公平需求（惩恶扬善）
- **情绪阶段**：悲壮（同胞受辱）→ 愤怒（家园被毁）→ 誓师（无路可退）→ 爆发（守护者降临）→ 哽咽（劫后余生）

## 核心机制详解

集体守护的本质是**"身后即是底线"的决死意志**。主角不是为了自己复仇，而是为了身后那些无法战斗的人而战。这种守护欲带来的爽感，不在于杀伐果断，而在于"我若死战，身后之人便无人守护"的悲壮感与责任感。

### 集体守护的类型

| 类型 | 守护对象 | 爽感来源 | 代表作品 |
|-----|---------|---------|---------|
| **宗门守护** | 同门师兄弟、长辈 | 从被逐之人到宗门脊梁 | 《完美世界》石昊守护石村 |
| **家园守护** | 家乡、父老乡亲 | 寸步不退、以一敌百 | 《仙逆》王林守护腾家镇 |
| **国运守护** | 国家、天下苍生 | 虽远必诛、人若犯我 | 《大国重工》类国运文 |
| **战友守护** | 袍泽、兄弟 | 替他们活下去 | 《雪中悍刀行》北凉死战 |
| **文明守护** | 人类族群、文明火种 | 文明不容玷污 | 《全球高武》人类崛起 |

### 爽感产生机制

1. **悲壮铺垫**：描写守护对象的无助与绝望，让读者产生保护欲
2. **绝境呈现**：敌人强大到令人窒息，主角看似毫无胜算
3. **誓师决意**："身后即是XXX，我已无路可退"的誓言
4. **极限爆发**：在绝境中爆发，守护的意志转化为战斗的力量
5. **劫后余生**：胜利后描写劫后余生的哽咽与彼此扶持

## 经典案例

### 网文经典

- 《完美世界》：石昊从荒域崛起，每次战斗都是为了守护石村亲人，最终独断万古
- 《仙逆》：王林从凡人修士到逆天强者，守护的始终是当初那个小镇和爱人
- 《雪中悍刀行》：徐凤年守护北凉三十万铁骑，"剑九黄背匣武帝城，老剑仙血洗春秋"
- 《庆余年》：范闲守护庆国，监察院门口"文学常识"石碑是文人的脊梁
- 《全职法师》：莫凡守护城市，每一场战斗都是为了身后的普通人
- 《大国重工》：重工厂守护国家尊严，"大国重工，扬我国威"

## AI 写作要点

### 核心写作公式

```
同胞受辱（悲壮铺垫）→ 家园被毁（绝境呈现）→ 誓师决战（决死意志）→ 极限爆发（守护者降临）→ 劫后余生（情感共鸣）
```

### 详细操作指南

**第一步：同胞受辱（300-500字）**

- 描写守护对象的无助与弱小的日常
- 敌人登场时的强大与傲慢，肆意欺凌守护对象
- 守护对象眼中的恐惧、绝望、无力
- 旁观者的冷漠或敢怒不敢言
- **关键**：让读者产生"想要保护他们"的强烈欲望

**第二步：家园被毁（300-400字）**

- 触发事件：敌人发起进攻，家园陷入危机
- 描写守护对象仓皇逃窜或拼死抵抗
- 主角看到这一幕的愤怒与自责（为什么不早点变强）
- 绝境呈现：敌人太强，我方几乎没有胜算
- 常用桥段：
  - "三年了，终于等到这一天——但来得太晚了"
  - "如果我能再强一点，就不用看着他们受苦"
  - "他们的眼神，和当年的我一模一样"

**第三步：誓师决战（200-300字）**

- 主角登场，宣言"今天，谁也别想动他们一根汗毛"
- 用具体台词或行动表达守护的决心
- 最好有"无路可退"的设定强化背水一战感
- 守护对象的反应：从绝望到看到希望
- 常用桥段：
  - "他们叫我废物？我今天就让你们看看，废物是怎么护住你们的"
  - "想动他们，从我的尸体上踏过去"
  - "这一战，不是为了赢，是为了告诉他们——有人愿意为他们而死"

**第四步：极限爆发（400-600字）**

- 战斗中穿插回忆杀，强化守护的动机
- 在生死关头突破极限，爆发强大力量
- 战斗要有层次感，不要一击必杀，而是层层递进
- 展示主角的战斗智慧与决死意志
- 常用桥段：
  - 燃烧生命/精血/潜力来换取力量
  - 在即将落败时因为某个契机（守护对象的呼喊、曾经誓言的回响）而突破
  - 以伤换伤、以命换命的打法

**第五步：劫后余生（200-300字）**

- 战斗结束，主角倒在血泊中，守护对象围上来
- 描写劫后余生的哽咽、拥抱、彼此扶持
- 不要写太多胜利后的荣耀，重点在情感
- 常用结尾：
  - "我们...赢了？"
  - "他们没事...太好了..."
  - 昏迷前的微笑

### 高级写作技巧

**1. 守护对象的反差塑造**

守护对象的反差越大，守护的爽感越强：
- 平时越弱小、越不起眼
- 战时越需要保护
- 胜利后他们的反应越感人

**2. "无路可退"的绝境营造**

背水一战的爽感来自于"身后即是底线"：
- 身后是家人/爱人/宗门，没有退路
- 每退一步，就有人要死
- 主角宁可战死也不后退

**3. 悲壮氛围的营造**

不要只写战斗，要写氛围：
- 战前的寂静、紧张的等待
- 战斗中的火光、惨叫、血腥
- 战斗后的废墟、寂静、重生

**4. 集体群像的描写**

守护不只是主角一个人的事：
- 描写配角也在战斗、也在守护
- 每个守护者都有自己的"身后之人"
- 汇聚成集体的力量

## 常见剧情模式

### 模式一：宗门存亡之战

```
【受辱】外敌入侵，宗门弟子被肆意屠杀，长老被侮辱
【求援】向各大势力求援，无人响应，"他们不敢惹这个人"
【绝境】敌人兵临城下，宗门上下绝望
【誓师】主角归来："今天，我替他们还债"
【爆发】以一敌万，宗门先辈的意志加持
【胜利】宗门保存，主角成为宗门精神图腾
```

### 模式二：家园沦陷收复

```
【日常】描写家园的平静生活，温馨的邻里关系
【沦陷】外敌来袭，家园被占，父老被奴役
【流亡】主角带着残余的族人流亡
【崛起】在流亡中变强，找到收复家园的方法
【反攻】带着队伍打回来，"我带他们回家"
【守护】夺回家园后，誓死守护
```

### 模式三：国运对决

```
【压制】国家被列强欺凌，人民在水深火热中
【不公】国际场合被羞辱，"落后就要挨打"
【觉醒】主角代表国家出战，每一战都是为了国家尊严
【死战】在绝对劣势下死战不退
【翻盘】绝境中突破，为国家赢得尊严
【影响】带动全民族的精神觉醒
```

### 模式四：末世守护

```
【末日】世界末日降临，怪物横行
【废墟】人类只能蜷缩在最后的堡垒中
【绝望】资源耗尽，绝望蔓延
【希望】主角带着希望出现，"我来带你们活下去"
【战斗】为了守护每一个人而死战
【传承】即使主角倒下，他的意志会传承下去
```

## 变体与组合

### 组合模组

| 组合对象 | 结合效果 |
|---------|---------|
| [[马甲曝光模组]] | 守护+身份曝光，平时被轻视的人原来是守护者 |
| [[权力反转模组]] | 守护后获得权力，成为守护者的合法代表 |
| [[信息差碾压模组]] | 用信息差提前布局，保护身后之人 |
| [[退婚打脸模组]] | 守护重要之人时，被曾经看不起的人目睹 |
| [[成长逆袭]] | 在守护中成长，越守护越强 |

### 变体分类

| 变体 | 特点 |
|-----|------|
| **悲壮型** | 主角最终倒下，但守护住了想守护的人（悲剧美） |
| **逆转型** | 前期被压制，后期绝地翻盘，守护成功 |
| **循环型** | 守护→失去→更强→再次守护，守护对象越来越多 |
| **觉醒型** | 从个人守护到集体守护，从一个人到一支军队 |

### 进阶方向

- **维度提升**：从守护一个家庭到守护一个宗门，再到守护一个国家，最后守护一个文明
- **精神传承**：主角的守护精神传承下去，影响更多人成为守护者
- **体系建立**：不仅自己守护，还建立守护的体系，让更多人能守护自己想守护的人

# 地位认可模组

## 基本信息

- **类别**：身份反转类
- **核心机制**：被曾经高不可攀的集体、权威或个人，正式且隆重地接纳、推崇
- **满足需求**：掌控需求（从被否定到被肯定）+ 归属需求（被接纳的感觉）
- **情绪阶段**：压抑（不被认可）→ 转折（收到认可信号）→ 爆发（正式认可仪式）→ 满足（主角淡然以对）

## 核心机制详解

地位认可模组本质上是对**"被看见"需求的满足**。主角曾经被否定、被轻视、被排斥在外，现在被曾经仰望的存在正式承认。

### 认可的类型

| 类型 | 认可来源 | 经典场景 |
|-----|---------|---------|
| **学府认可** | 至高学府破格录取 | 扫地僧开门、宗师收徒 |
| **帝王认可** | 皇帝亲自册封 | 金口玉言、圣旨降临 |
| **宗门认可** | 宗门正式接纳 | 拜师典礼、传承仪式 |
| **行业认可** | 领域权威的承认 | 泰斗鞠躬、同行拜服 |
| **榜单认可** | 权威榜单的认证 | 天榜题名、战神认可 |
| **团体认可** | 精英团体的接纳 | 加入核心圈子、成为核心成员 |
| **对手认可** | 宿敌/劲敌的承认 | 最大的敌人说"你赢了" |

### 爽感产生机制

1. **压抑铺垫**：主角曾被这个集体/权威否定或轻视
2. **信号出现**：收到认可的前兆（特使来访、圣旨将至等）
3. **仪式构建**：详细描写认可仪式的庄重感和参与者的反应
4. **旁观震撼**：旁观者的羡慕、震惊、难以置信
5. **淡然回应**：主角保持平静，逼格更高

## AI 写作要点

### 核心写作公式

```
曾经被否定的背景 → 认可信号出现 → 仪式细节展开 → 旁观者反应 → 主角淡然回应
```

### 详细操作指南

**第一步：背景铺垫（200-400字）**

- 交代主角曾经被否定的历史
- 被否定的理由要具体且不公（如：出身低、资质差、被陷害）
- 曾被轻视的具体事件要有画面感
- **情绪**：委屈、愤怒、憋屈 → 等待翻盘

**第二步：认可信号（200-300字）**

- 突然的消息打破平静
- 消息来源要有权威性（皇帝特使、宗门长老、正式公文）
- 传递者的态度要有戏剧性转变（从轻视到恭敬）
- 其他人的反应：震惊、不相信、打探消息

**第三步：仪式细节（400-600字）——核心爆点**

这是最关键的环节，要调动一切感官描写：

| 仪式要素 | 描写重点 |
|---------|---------|
| **地点** | 庄重场所（太和殿、大殿、传承殿），注意光线、空间感 |
| **物品** | 圣旨/令牌/信物的细节描写（材质、工艺、刻字） |
| **人物态度** | 特使/权威的恭敬态度（跪拜、躬身、低头不敢直视） |
| **台词内容** | 正式认可的宣言内容（引用原文更有冲击力） |
| **背景反应** | 全场鸦雀无声或倒吸凉气 |
| **BGM暗示** | 用"落针可闻"、"庄严肃穆"等词营造氛围 |

**示例场景描写：**

> "圣旨展开，金色绢帛在阳光下熠熠生辉。
> '奉天承运，皇帝诏曰：特擢林凡为镇国大将军，赐金印一枚、虎符一對——'
> 特使念到此处，声音微微发颤。
> 满朝文武的目光齐刷刷落在那个站在角落里的年轻人身上。
> 三个月前，他们还嘲笑他是'废物'。
> 而此刻，连丞相都躬身站在他身后半步。"

**第四步：旁观者反应（200-300字）**

- 用多个配角的反应构成"反应链"
- 曾经嘲笑主角的人：震惊 → 尴尬 → 恐惧 → 讨好
- 曾经看好主角的人：欣慰 → 自豪 → 与有荣焉
- 旁观群众的反应：羡慕、议论、震撼

**反应链示例：**

```
主角曾经的上司：
  "不可能！他凭什么！"（质疑）
  → "这一定是搞错了！"（拒绝相信）
  → "镇国大将军...是我当年逼走的那个？"（恐惧）
  → 低着头挤到主角面前谄媚："大人还记得卑职吗？"（崩溃）

旁观者：
  "那个废物...不，大人！"
  "天哪，他是怎么做到的！"
  "我刚才是不是说错话了..."
```

**第五步：主角淡然回应（100-200字）**

- 主角的反应越淡，效果越好
- 可以：
  - 平静接旨："臣，接旨。"
  - 微微一笑："意料之中。"
  - 不卑不亢："多谢陛下。"
- **不要**：激动落泪、长篇大论、得意炫耀
- 主角的平静与周围的轰动形成鲜明对比

### 高级写作技巧

**1. 认可的反差对比**

| 场景 | 过去 | 现在 |
|-----|-----|-----|
| 态度 | 被拒之门外 | 特使登门 |
| 称呼 | "废物" | "大人" |
| 站位 | 站在角落 | 站在中央 |
| 座位 | 没有座位 | 主座 |
| 目光 | 无人正视 | 所有人注视 |

**2. 仪式感的层次叠加**

不要只写一个认可，要让认可一个接一个：
- 第一重：地方权威认可
- 第二重：更高层级的认可
- 第三重：最高层级的认可
- 每多一重，爽感叠加

**3. 认可的其他人见证**

让重要配角亲眼见证主角被认可：
- 前未婚妻看着主角被皇室求婚
- 曾经的上司看着主角升职
- 宿敌看着主角被更高权威认可

**4. 仪式后的余波**

认可仪式结束后，写一段后续反应：
- 之前疏远的人纷纷示好
- 各方势力重新评估主角价值
- 主角身边的人感到自豪

**5. 反派当场崩溃**

让反派在认可仪式现场崩溃：
- 跪地求饶
- 失魂落魄
- 试图逃跑
- 丑态毕露

## 常见剧情模式

### 模式一：破格录取

```
【背景】主角因出身问题被顶级学府拒绝
【转折】主角做出惊人成果（解决学术难题、创造功法）
【认可】学府院长亲自登门，携圣旨破格录取
【亮点】曾经拒绝主角的考官站在门口迎接
```

### 模式二：帝王封赏

```
【背景】主角被诬陷后平反，或立下大功
【认可】皇帝在金殿举行正式册封仪式
【亮点】满朝文武亲眼见证，配角当场崩溃
【收尾】主角淡然接过金印，毫不动容
```

### 模式三：宗师收徒

```
【背景】主角曾被某位宗师轻视或拒绝
【转折】主角展现出惊世骇俗的天赋或实力
【认可】宗师当众宣布收主角为亲传弟子
【亮点】宗师向主角行礼（更高位者向低位者低头）
```

### 模式四：团体接纳

```
【背景】主角申请加入精英团体被拒多次
【转折】主角凭实力证明自己
【认可】团长亲自授予核心令牌
【亮点】全体成员起立鼓掌，曾经的拒绝者低头
```

## 变体与组合

### 组合模组

| 组合对象 | 结合效果 |
|---------|---------|
| [[马甲曝光模组]] | 先曝光身份（被认可的基础），再正式认可 |
| [[排行榜逆袭模组]] | 先通过榜单证明实力，再获得官方认可 |
| [[权力反转模组]] | 认可带来权力反转，权力又巩固地位 |
| [[退婚打脸模组]] | 退婚前否定主角，退婚后主角被更高存在认可 |

### 变体分类

| 变体 | 特点 |
|-----|------|
| **单次认可** | 一次性获得最高认可，一锤定音 |
| **逐级晋升** | 从最低级一步步爬到最高级 |
| **回归认可** | 离开后回归，被重新认可并赋予更高地位 |
| **追加认可** | 获得认可后，因新功劳追加更高级别认可 |
| **死后认可** | 主角牺牲后被追认，活着的人代为领受 |
| **被动认可** | 主角不想接受但被迫接受（被迫营业） |

# 格局碾压模组

## 基本信息

- **类别**：认知碾压类
- **核心机制**：反派还在计较一城一地，主角已在布局大陆/星球/诸天
- **满足需求**：优越需求（认知上的全面碾压）、掌控需求（布局者的全局观）
- **情绪阶段**：期待（反派展示野心）→ 困惑（主角反应平淡）→ 震惊（看清主角格局）→ 恐惧（意识到差距）→ 臣服（仰望高山）

## 核心机制详解

格局碾压的本质是**认知维度差**。主角站在更高的认知层面，看穿了所有人都看不见的可能性。反派的雄心壮志，在主角的蓝图中，只是微不足道的一个注脚。

### 格局碾压的类型

| 类型 | 反派的格局 | 主角的格局 | 差距表现 |
|-----|-----------|-----------|---------|
| **空间格局** | 占领一座城 | 布局整个大陆 | 地图展开，城镇只是棋子 |
| **时间格局** | 赢得当前战争 | 谋划百年后的局面 | 历史潮流，尽在推演 |
| **层次格局** | 争夺资源 | 掌控规则制定 | 降维打击，不在同一层次 |
| **维度格局** | 世俗权力巅峰 | 超脱世俗之上 | 仙凡之别，维度差异 |
| **系统格局** | 解决单个问题 | 构建系统生态 | 以平台对单点，全面压制 |

### 爽感产生机制

1. **反派的野心展示**：反派得意洋洋地展示其终极目标
2. **主角的平淡反应**：众人期待的震惊没有出现，主角只是微微一笑
3. **地图/蓝图的展开**：主角展开更大的地图、更多的可能性
4. **对比的震撼**：反派目标的渺小，主角布局的宏大，形成强烈对比
5. **反派的绝望**：反派意识到自己不过是主角大棋中的一个小卒

## AI 写作要点

### 核心写作公式

```
反派展示野心（得意）→ 主角淡然反应（不屑）→ 展开更大格局（碾压）→ 对比产生绝望（崩溃）
```

### 详细操作指南

**第一步：反派野心展示（200-300字）**

- 反派志得意满，展示其终极目标
- 常用场景：反派的登基大典、联盟大会、密室谋反
- 反派的台词充满自信："等我夺取XX，天下就是我的了"
- 旁观者的震惊和附和，强化反派的强大感
- **关键**：先让反派的野心显得很宏大，这样对比才够爽

**第二步：主角的异常反应（100-200字）**

- 主角的反应与氛围格格不入
- 不是愤怒，不是恐惧，而是淡淡的微笑
- 旁观者的困惑："他怎么不害怕？"
- 反派的疑惑："你为什么不怕？"
- **关键**：主角的淡然要显得"不正常"，增加悬念

**第三步：格局展开（400-600字）**

- 这是全文的核心，要详细写
- 主角展开地图/沙盘/星图
- 用视觉符号直观展示格局差距
- 常用道具：
  - 地图（天下图、江山图）
  - 沙盘（军事推演）
  - 星图（宇宙格局）
  - 棋盘（博弈推演）
  - 气泡图（层级展示）

**第四步：对比与解说（300-400字）**

- 主角轻描淡写地指出反派目标的"位置"
- 经典台词：
  - "你的目标不错，可惜连这张图的百分之一都不到。"
  - "你的棋下得很好，但你看不懂棋盘。"
  - "你以为这是终点，其实只是起点。"
  - "我布局百年，你争的不过是我百年前的一个棋子。"
- 反派的反应：困惑 → 震惊 → 细思极恐 → 绝望

**第五步：尘埃落定（100-200字）**

- 主角转身离去，留下一句意味深长的话
- 反派瘫倒在地，喃喃自语
- 旁观者的恐惧和敬畏
- **关键**：不要写主角太多话，留白更有力量

### 高级写作技巧

**1. 格局的层次感**

不要一步到位，让格局有递进层次：
- 第一层：对手以为的终点，主角的起点
- 第二层：对手看到的全局，主角的局部
- 第三层：对手看不到的更高维度

**2. 数字的冲击力**

用具体数字强化格局差距：
- "你占据的三城，在我眼中不过是一个数字。"
- "你苦心经营的十年规划，只是我蓝图中三个月的事。"
- "你的万两黄金，换不来我布局的一成。"

**3. 沉默的力量**

主角不要话太多，有时候沉默本身就是碾压：
- 主角只是指了指地图的某处
- 所有人顺着手指看去，才恍然大悟
- 这种"后知后觉"的震撼更强

**4. 反派的认知局限**

让反派的失败显得"合理"，因为他们的认知确实有盲区：
- 反派是基于正确的信息做判断
- 但主角的信息更全面
- 这种失败比"反派突然变傻"更有爽感

**5. 视觉化展示**

尽可能用视觉符号展示格局：
- 沙盘推演，让所有人亲眼看到对比
- 星图展示，宇宙级别的格局差距
- 地图展开，江山如画尽收眼底
- 棋局展示，众生皆为棋子

## 常见剧情模式

### 模式一：王朝争霸

```
【反派展示】反王得意洋洋："等我登基，天下就是我的了！"
【主角淡然】主角在角落微微一笑，不发一言
【格局展开】谋士展开天下图："主公请看，反王的版图......在这里。"
【对比震撼】主角指着地图另一处："我的版图，在这里。整个天下，不过是我棋盘的一角。"
【反派崩溃】反王看清对比后，双腿发软，万念俱灰
```

### 模式二：星际征途

```
【反派展示】星际帝国皇帝："等我统一银河系，人类就迎来永恒和平！"
【主角淡然】主角看着星图，只是说："银河系？"
【格局展开】主角展开本宇宙群图："这才是我的棋盘。一千二百个星系，八百个文明。"
【对比震撼】皇帝看着自己苦心经营的银河系，在星图上只是一个点
【反派崩溃】"这不可能......人类已知宇宙之外，还有这么多？"
```

### 模式三：宗门较量

```
【反派展示】天才弟子："等我成为宗主，就清理门户，让宗门唯我独尊！"
【主角淡然】主角只是摆弄手中的茶杯
【格局展开】主角展开位面图："你的宗门，在这个世界。在我的位面里，有三千六百个这样的宗门。"
【对比震撼】天才弟子的宗门自傲，在主角眼中不过是沧海一粟
【反派崩溃】"......三千六百个？"
```

### 模式四：幕后黑手

```
【反派展示】反派认为自己是棋手，操控一切
【主角揭露】主角指出："你以为自己在下棋，其实你也是棋子。"
【格局展示】主角展开更高层次的棋盘，反派和自己的棋盘都在其中
【震撼对比】反派苦心经营的棋局，不过是主角棋盘的一个角落
【反派崩溃】"我的一切......都是你的布局？"
```

## 变体与组合

### 组合模组

| 组合对象 | 结合效果 |
|---------|---------|
| [[信息差碾压模组]] | 信息差揭示+格局碾压，从"我知道你不知道"升级到"你的全局不过是我的局部" |
| [[权力反转模组]] | 格局碾压后的权力反转，掌控格局后重新定义规则 |
| [[文化降维模组]] | 格局碾压+文化碾压，双重降维打击 |
| [[马甲曝光模组]] | 格局碾压+身份曝光，揭示主角的格局远超当前所有势力 |

### 变体分类

| 变体 | 特点 |
|-----|------|
| **单向式** | 只有主角碾压反派，主角格局远大 |
| **双向式** | 主角碾压反派，但也有更大的存在在碾压主角（诸天流） |
| **渐进式** | 格局逐级提升，每次提升都是一次碾压 |
| **反转式** | 先让读者以为主角格局小，最后揭示主角格局远超想象 |
| **嵌套式** | 格局中有格局，主角的棋盘也在别人的棋盘中 |

### 进阶方向

- **维度提升**：从王朝→大陆→位面→宇宙→多元宇宙，逐级碾压
- **时间拓展**：从当前→十年→百年→万劫→永恒
- **层次递进**：从棋子→棋手→规则制定者→宇宙意志

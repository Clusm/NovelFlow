# 投稿运营指南

## 基本信息

- **类型**：运营知识汇总
- **适用阶段**：新人起步 ~ 职业发展
- **核心价值**：帮助作者从"写完"到"赚到"，少走弯路

---

## 一、平台选择决策树

选择平台是新人第一步，直接决定稿费天花板与成长路径。按自身情况对号入座：

| 你的情况 | 推荐平台 | 理由 |
|---------|---------|------|
| 新人无完本经验 | **番茄小说** | 算法分发，签约门槛低，努力有回报 |
| 有完本经验，想冲IP | **起点中文网** | IP孵化能力最强，订阅制读者黏性高 |
| 手速快（5000+字/天），想快速变现 | **飞卢小说** | 防盗强，结算快，周结算 |
| 女频向（言情/耽美） | **晋江文学城** | 女频第一，版权改编活跃 |
| 写短篇/中篇（20-50万字） | **飞卢体** | 脑洞+快节奏，编辑易过 |
| 想写新媒体/知乎风 | **知乎盐选** | 单篇500-5000元，适合短篇 |
| 有创意设定，想长期养笔名 | **起点内投** | 直接对接编辑，有修改反馈 |

## 各大平台详细分析

### 起点中文网

**定位**：中国网文老大哥，男频第一平台

**稿费模式**：
- 付费订阅（VIP章节），作者分成40-50%
- 打赏（礼物分成50%）
- 月票排名奖励
- 各类榜单奖金

**日更压力**：3000-5000字/天

**签约难度**：高。需要内投或直发6000字审稿，编辑筛选严格。

**IP价值**：极强。《庆余年》《全职高手》《赘婿》《琅琊榜》等现象级IP均出自起点。

**新人适合度**：★★☆☆☆（竞争激烈，但一旦出头回报丰厚）

**适合题材**：玄幻、仙侠、都市、科幻、游戏（老白文）

### 番茄小说

**定位**：字节跳动旗下，免费阅读+算法推荐

**稿费模式**：
- 广告分成（透明可查）
- 全勤奖（日更4000字600元/月）
- 飞书计划（单书月收入破10万可申请）

**日更压力**：4000字以上

**签约难度**：低（满2万字申请，7个工作日出结果）

**推荐机制**：
```
第8章：第一次算法节点（决定初始推荐量）
第15章：第二次算法节点（决定是否扩大推荐）
完读率>30%：进入更大流量池
```

**"巅峰榜"**：每月TOP30，上榜作品获得额外推荐资源，是爆款制造机。《十日终焉》《我在精神病院学斩神》均出自此榜。

**IP价值**：强（动漫改编居多）

**新人适合度**：★★★★★（门槛低，算法公平，努力可见）

**适合题材**：都市爽文、穿越、系统、悬疑（番茄对脑洞文友好）

### 飞卢小说

**定位**：脑洞文、主播风、飞卢风发源地

**稿费模式**：
- VIP票实时结算（60-70%归作者）
- 周结算
- 打赏

**日更压力**：5000-10000字/天（卷王之王）

**签约难度**：中

**特点**：
- 防盗强
- 节奏极快，开局即高潮
- 读者喜欢"一天一个神技"的金手指
- 同人/综武非常火

**IP价值**：几乎无

**新人适合度**：★★★☆☆（好赚钱但伤笔名）

**适合题材**：洪荒流、同人文、游戏系统、综武

### 晋江文学城

**定位**：女频第一，版权改编活跃

**稿费模式**：
- VIP订阅（晋江币）
- 晋江保底+分成
- 各类榜单奖励

**日更压力**：3000字以上

**签约难度**：中等

**IP价值**：强（影视剧改编频繁）

**适合题材**：言情、耽美、百合、修仙侠

**特别注意**：晋江有严格的审核制度，政治、BL等敏感内容受限

### 知乎盐选

**定位**：短篇/中篇，单篇变现

**稿费模式**：单篇500-5000元保底，根据阅读量有额外分成

**适合题材**：短篇故事、脑洞文、知乎风叙事

**特点**：重故事性，轻套路，适合有创意的短篇作者

## 番茄小说详细运营攻略

番茄是当下最适合新人起步的平台，以下是详细运营指南：

### 签约流程

1. 满2万字在作家后台申请
2. 7个工作日出结果
3. 签约后需连续3天每日更新4000字以上才能进入推荐节点

### 推荐节点

**验证期**（8万字前后）：
- 系统圈定一批读者测试作品
- 根据这批读者的数据决定是否扩大推荐

**首秀**（10-30万字）：
- 决定作品命运的关键期
- 数据好则进入更大流量池
- 数据差则可能被遗忘

**书测**（20-50万字）：
- 多书名实验，对比不同书名的数据效果
- 申请完结后不可撤销

### 提升完读率的技巧

**前三章定生死**：
- 开局要有钩子，让读者想看下去
- 期待感要建立——读者为什么要追更？
- 主角要讨喜或有魅力

**节奏把控**：
- 每章结尾留悬念
- 小高潮每3-5章一个
- 大高潮每15-20章一个

**互动数据**：
- 评论区积极回复读者
- 段评、章评多与读者互动
- 催更票是重要指标

## 起点中文网投稿攻略

### 内投方式

1. 准备3000-5000字正文+大纲
2. 私信编辑或发邮箱
3. 优点：直接对接编辑，有修改反馈
4. 缺点：被拒可能性高

### 直发方式

1. 直接在网站发表
2. 6000字后可申请签约
3. 优点：数据说话
4. 缺点：编辑主动捞人概率低

### 起点生存法则

**订阅制核心指标**：
- 收藏、推荐票、月票
- 订阅率（付费读者比例）
- 追读率（持续追更比例）

**提升订阅技巧**：
- VIP章节要留钩子
- 不能为了字数水剧情
- 重要情节要放在订阅章节

## 飞卢小说生存指南

### 飞卢特点

- 节奏极快："开局就打架"
- 金手指要强："一天一个神技"
- 同人多：综武、洪荒、漫威同人是主流
- 防偷特别强

### 飞卢日更技巧

- 建议存稿30章再发
- 日更10000字是基本线
- 有爆发才有推荐

### 飞卢不适合谁

- 写惯了慢热文的作者
- 不适应飞卢特有"主播风"的作者
- 想养笔名的作者

## 新人常见问题

### Q：签约被拒怎么办？

**答**：
1. 分析被拒原因——是开头不够好还是题材问题
2. 可以切书重开（番茄切书影响小）
3. 换平台尝试
4. 找编辑或有经验的作者看稿提意见

### Q：写到一半没灵感了怎么办？

**答**：
1. 提前写大纲，留足缓冲
2. 看同类型爆款找灵感
3. 与读者互动，从评论中找灵感
4. 实在不行可以申请断更

### Q：要不要全职写作？

**答**：
1. 先兼职写，确认能稳定月入5000+再考虑全职
2. 看清平台规律，有些平台收入不稳定
3. 有稳定工作再写作，心态更稳

### Q：如何提升写作水平？

**答**：
1. 多看爆款，分析为什么火
2. 练习写大纲，锻炼结构能力
3. 建立读者反馈机制
4. 参加写作社群，互相学习

---

## 避坑指南

1. **不要一书多投**：一个平台签约就要独家
2. **不要断更**：断更严重影响推荐和读者信任
3. **不要抄袭**：抄袭会被封号，永久黑名单
4. **不要违规**：涉黄涉政涉暴内容不要碰
5. **不要裸更**：一定要有存稿再发
6. **不要只看钱**：前100万字是练手，不要太在意收入

## 进阶：IP改编入门

当作品有了一定成绩后，可以考虑IP改编方向：

**IP改编评估维度**：
- 世界观是否独特完整
- 人物是否有辨识度
- 剧情是否有记忆点
- 是否有影视化可行性

**版权开发路径**：
- 平台代理：起点、晋江等平台有版权代理
- 自主授权：直接与影视公司、游戏公司谈
- 有声/动漫：门槛相对较低，适合新人

**IP改编注意事项**：
- 签合同时要看清楚授权期限和分成比例
- 不要过早卖版权，初写者容易被坑
- 建议找专业律师看合同

# 卡牌收集流

## 定义

以收集/培养/组合"卡牌"为核心机制的游戏化写作流派。主角通过获取卡牌、培养卡牌、组合卡牌形成战斗力，卡牌的稀有度/组合效果/培养路线构成核心策略深度。核心爽感：**收集的满足感**——集齐一套强力卡组的快感不亚于任何其他爽点。

## 核心机制

**卡牌获取**：通过抽卡/任务奖励/战斗掉落/特殊事件获得卡牌。稀有度影响强度——但不是稀有=最强，普通卡牌组合起来可能比SSR更强。

**卡牌培养**：卡牌需要升级/升星/觉醒/突破，培养材料稀缺性决定节奏。培养要有方向感。

**组合效果**：特定卡牌组合产生额外效果，鼓励收集特定组合。这叫"羁绊"或"套装效果"。

**对战系统**：卡牌对战是小说的核心战斗方式，不是传统的打怪升级。

**稀有度体系**：SSR/SR/R/N等稀有度分级，保底机制是标配。

## 卡牌类型

| 类型 | 代表 | 特点 |
|------|------|------|
| 角色卡 | 英雄/战士/法师 | 有技能有属性，可升级 |
| 技能卡 | 火球术/治疗术 | 直接产生效果 |
| 装备卡 | 武器/防具/饰品 | 增强角色能力 |
| 事件卡 | buff/debuff | 改变战局 |
| 召唤卡 | 召唤兽/守护灵 | 独立单位参战 |
| 陷阱卡 | 反伤/反弹 | 防守反击 |
| 魔法卡 | 元素攻击/结界 | 特定条件下触发 |
| 融合卡 | 合体技能 | 多卡组合 |

## 抽卡机制设计

**概率公示**：SSR概率多少要有明确设定，让读者知道稀有度是真实的。

**保底机制**：抽够多少次必定出SSR，防止读者运气太差失去信心。

**井喷机制**：累计抽卡达到某个数额外送一张，提升长期目标感。

**单抽vs十连**：单抽有时出货，十连有保底，给玩家选择空间。

**天井系统**：集齐特定碎片兑换限定卡牌。

**概率递增**：每次不出货，下一次概率微微提升。

**限定卡池**：特定时间限定卡牌，制造稀缺感。

**《卡牌收集系统》**：穿越异世界获得卡牌系统，用卡牌战斗。

**《我的卡牌游戏》**：都市背景，卡牌对战竞技。

**《卡牌之无限轮回》**：卡牌+无限流融合。

**《最强卡师》**（廿又）：制卡师流，自己制作卡牌。

**《卡牌制造商》**：创造属于自己的卡牌。

**《亿点点制卡天赋》**：现代都市背景的制卡故事。

## 组合策略

**套装效果**：收集齐某套卡牌触发额外效果，是收集的动力之一。

**阵营克制**：不同阵营卡牌相互克制，不是无脑堆最强。

**技能联动**：特定卡牌配合产生组合技能，1+1>2。

**属性平衡**：不同属性有相克关系，策略性丰富。

**养成路线**：同一张卡牌有多种养成方向。

**卡牌融合**：两张卡牌可以融合成更高级卡牌。

## 流派变体

**集换式卡牌型**：卡牌可以自由交易，玩家之间可以买卖。

**自创卡牌型**：主角可以自己制作卡牌，有创造性。

**文明发展型**：用卡牌建设和发展领地/文明。

**生存对抗型**：在末日/绝境中用卡牌求生。

**竞技对战型**：卡牌对战成为竞技体育。

**卡牌精灵型**：卡牌有生命，可以培养感情。

## 核心写作公式

**开局引导**（1-10章）：主角获得第一个卡牌/抽卡系统，快速建立"收集"的概念。

**核心收集**（10-50章）：通过各种方式获取关键卡牌，建立初始卡组。每一次抽卡都是期待。

**组合探索**（50-100章）：尝试不同卡牌组合，发现强力组合的瞬间是重要的爽点。

**终极收集**（100章+）：集齐终极卡组/解锁全部羁绊，实现收集完美感。

**竞技巅峰**（100-200章）：用卡牌对战登顶竞技场。

**卡牌奥秘**（200章+）：发现卡牌背后的秘密。

## 写作技巧

**卡牌要有灵魂**：每张卡牌要有独特的外观描述/背景故事/性格特征，不只是数值差异。

**收集要有意义**：卡牌收集不能只是数值积累，要与剧情紧密结合。

**对战要精彩**：卡牌对战的描写要有战术感，不是简单的数值对比。

**稀有度要合理**：太容易获得失去稀缺感，太难获得失去期待感。

**组合要有策略**：让读者思考什么样的组合最强，而不是无脑堆SSR。

**升级要有惊喜**：每次培养/升级都要有期待感。

## 避坑

1. **卡牌没有个性**：每个卡牌要有独特的存在感，不只是数值差异
2. **收集太容易**：稀有卡牌要有稀缺感才有价值，让人珍惜
3. **组合效果太明显**：最优解太单一失去探索乐趣，给多种可行组合
4. **抽卡太随机**：纯看脸会让读者烦躁，给保底机制
5. **战斗只看稀有度**：普通卡也能通过组合超越SSR卡，设计更有策略深度
6. **卡牌无背景**：每张卡牌要有自己的故事，不是只有属性数值
7. **数值膨胀**：卡牌等级/属性到后期变成天文数字，建立合理的数值天花板
8. **战斗描写单调**：对战要有战术变化，不是简单的属性对比

## 卡牌属性设计参考

| 属性 | 说明 |
|------|------|
| 稀有度 | R/SR/SSR/UR/限定 |
| 阵营 | 阵营间有克制关系 |
| 属性 | 火/水/风/雷/暗/光/无 |
| 等级 | 1-100，升级需要材料 |
| 星级 | 1-5星，升星需要同名卡 |
| 技能数 | 1-3个主动/被动技能 |
| 成长潜力 | 影响升级属性成长 |
| 羁绊 | 与特定卡牌组合触发 |
| 专属装备 | 进一步提升单卡强度 |

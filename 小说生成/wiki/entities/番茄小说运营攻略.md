# 番茄小说运营攻略

## 平台概况

番茄小说属字节跳动（今日头条系），采用**免费阅读+广告分成**模式。2024年月活用户数亿，是目前流量最大的网文平台。2023年11月推出"巅峰榜"（TOP30），成为行业标杆，让《十日终焉》《我在精神病院学斩神》等爆款跨平台破圈。

## 核心特色：算法推荐

番茄与起点完全不同——**算法决定生死，编辑对接极弱**：
- 系统根据完读率、追读率、互动数据自动分配推荐量
- 作者几乎不需要主动联系编辑，作品数据好就推，不好就沉
- "巅峰榜"：每月TOP30，上榜作品额外获得大量推荐资源

## 签约与投稿

### 签约方式
1. 在番茄作家后台（番茄写作助手APP/网页版）提交
2. 正文满2万字即可申请签约
3. 审核周期短（通常3-5个工作日）
4. 签约门槛远低于起点

### 投稿要点
- **开篇即爆**：番茄用户耐心极低，前3章必须抛出强钩子
- **节奏要快**：每章结尾留悬念，不能平铺直叙
- **短情节+快反转**：首都师范大学许苗苗教授指出，番茄小说的特点是"节奏快，长故事、短情节、快反转，像微短剧"
- **完读率是核心**：前10章完读率决定第一波推荐量

## 算法关键节点

| 章节 | 重要性 | 决定因素 |
|------|--------|---------|
| **第1-3章** | 极高 | 留存率，决定是否给初始推荐 |
| **第8章** | 高 | 第一次算法评级 |
| **第15章** | 高 | 第二次推荐量分配 |
| **追读率** | 贯穿全程 | 决定推荐是否继续 |

## 收益模式

### 广告分成（月结）
- 作品内嵌广告，按展现量（CPM）分成
- 完读率越高 → 广告展现越多 → 收益越高
- 中下层作者月均几百到几千元，爆款可达数十万/月

### 巅峰榜奖金
- 登上巅峰榜TOP30有额外奖金
- 上榜作品IP改编加速（番茄官方推动）

### 短剧/动漫改编
- 2024年：番茄原创短剧改编超500部，有声超2600部
- 动漫改编：《斩神之凡尘神域》2024年播放量破7亿

## 2024年爆款类型（番茄"巅峰榜"规律）

番茄巅峰榜揭示了当前最流行的新类型：

### 1. 规则怪谈 + 无限流
代表作品：《十日终焉》《诡舍》《诸神愚戏》
- 将"规则怪谈"从独立类型变为可混合的"元素"
- 核心特征：高设定严谨性 + 无限想象 + 复杂人性
- 《十日终焉》北大教授邵燕君高度评价："技术均衡，完成度高"

### 2. 都市高武 + 克苏鲁 + 群像
代表作品：《我在精神病院学斩神》
- 神明复苏 + 高武降临 + 克苏鲁 + 热血成长 + 群像叙事
- "老白+小白"融合的"番茄风"形成

### 3. 国潮 + 传统文化
代表作品：《我不是戏神》
- 传统文化元素（戏曲、醒狮、祭神）现代化转化
- "国潮"写作新风向

### 4. 修仙稳健流 + 日常化
代表作品：《长生烬》
- "长生""系统""苟"的元素组合
- "体验最好的修仙文之一"

### 5. 创新女频（新女性主体）
代表作品：《坤仪》《绮靡》《攀高枝》
- 女主不再恋爱脑，目标明确、坚韧不拔
- "新女性主体"创作风潮

## "番茄风"核心特征

1. **快节奏**：每章有反转，留钩子
2. **短情节**：强冲突，快速推进
3. **情感激发**：短小"萌要素"串联成长线
4. **多元素融合**：不再单一类型，数据库化混搭
5. **破圈意识**：追求圈外影响力，不只是老白读者

## 运营建议

1. **前三章定生死**：开篇平淡直接算法降权
2. **追更率是关键**：让读者追更比吸引新读者更重要
3. **题材要新**：番茄上老套路极难爆，参考巅峰榜选材
4. **日更建议4000字以上**：算法认为活跃度影响推荐权重
5. **参考巅峰榜选材**：不是抄套路，是理解当前市场风向

# 配角类型-老爷爷

## 基本信息

- **适用流派**：[[系统任务流]]、升级流
- **核心特征**：隐世高手/前辈，主动或被动将毕生所学传给主角
- **戏份权重**：中期重要配角（功成后逐渐退场）

## 关键词

> 传承、考验、最后托付

## 子类型划分

### 1. 师徒传承型
- **特征**：正式收徒，系统传授功法技能
- **案例**：药老→萧炎、独孤求败→杨过
- **写法要点**：师徒互动增加情感深度

### 2. 遗物托付型
- **特征**：留下功法秘笈，主角自行修炼
- **案例**：独孤求败剑冢、张无忌得九阳真经
- **写法要点**：增加主角探索和悟性描写

### 3. 灵魂指引型
- **特征**：以灵魂/意识状态陪伴主角成长
- **案例**：药老、血溪宗前任
- **写法要点**：增加对话和内心戏

### 4. 临终托付型
- **特征**：生命最后时刻将一切传给主角
- **案例**：射雕/神雕中的多位前辈
- **写法要点**：渲染悲壮气氛，强化使命感

### 5. 交易型/合作型
- **特征**：非无偿传承，要求对等回报
- **案例**：部分系统流中的老爷爷角色
- **写法要点**：明确契约关系，增加博弈感

## 人物弧光设计

### 经典弧光模式

**弧光A：传承与消逝**
```
初始：主角偶得机缘，遇老爷爷
发展：老爷爷指导修炼，倾囊相授
高潮：老爷爷耗尽最后力量/生命
结局：主角承担遗志，继续前行
```
> 药老结局模式：灵魂力量耗尽后转世重生

**弧光B：考验与认可**
```
初始：老爷爷不轻传，试探主角心性
发展：通过一系列考验
高潮：正式认可，传授核心
结局：老爷爷安心退场/消散
```

**弧光C：重逢与回归**
```
初始：老爷爷假死/沉睡
发展：主角成长过程中逐步唤醒
高潮：老爷爷恢复部分力量
结局：老爷爷复活或超脱
```

### 弧光变化关键节点

| 节点 | 典型事件 | 情感转折 |
|------|----------|----------|
| 初次相遇 | 主角获得传承媒介 | 惊喜、疑惑 |
| 考验阶段 | 老爷爷设局考验心性 | 紧张、成长 |
| 倾囊相授 | 传授核心功法/秘宝 | 感恩、珍惜 |
| 最后托付 | 生命/灵魂即将耗尽 | 悲壮、责任 |
| 继承遗志 | 主角继续未竟之路 | 坚定、成熟 |

## 情节功能

1. **实力过渡**：让主角实力合理跨越瓶颈
2. **世界观拓展**：通过老爷爷口述展现更广阔世界
3. **动机强化**：继承遗志增加主角行动合理性
4. **爽感加成**：获得前辈毕生心血的天降馅饼

## 写作技巧

### 老爷爷的身份设计

| 身份层次 | 示例 | 作用 |
|----------|------|------|
| 远古大能 | 独孤求败、帝落时代强者 | 提升传承分量 |
| 隐世高手 | 风尊者、散修高人 | 制造意外惊喜 |
| 宗门前辈 | 药老、前代宗主 | 连接更大势力 |
| 特殊存在 | 神兽化形、器灵 | 增加新奇感 |

### 传承设计原则

1. **稀缺性**：传承必须独一无二，非烂大街功法
2. **适配性**：与主角天赋/性格高度契合
3. **成长性**：功法有进阶空间，可持续提升
4. **代价性**：传承往往附带责任或隐患

### 退场处理

| 退场方式 | 适用类型 | 情感效果 |
|----------|----------|----------|
| 灵魂消散 | 灵魂体老爷爷 | 悲壮、感动 |
| 假死沉睡 | 暂时性退场 | 伏笔、悬念 |
| 飞升离去 | 功成身退 | 超然、向往 |
| 牺牲保护 | 为主角挡劫 | 震撼、悲痛 |
| 传承完成隐退 | 自然退场 | 和谐、圆满 |

### 角色亮点设计
- 独特的口头禅或说话方式
- 高深莫测的行为习惯
- 不为人知的过去秘辛
- 与主角的独特情感羁绊

## 注意事项

### 避免老爷爷类型的问题
- **过度依赖**：主角成长不应全靠老爷爷
- **战力崩坏**：老爷爷实力要合理设定
- **情感突兀**：师徒感情需要事件积累
- **退场仓促**：重要角色退场需足够铺垫

### 创新方向
- 老爷爷本身有弱点需主角弥补
- 老爷爷与主角双向成长
- 老爷爷有复杂过去，非纯粹好人
- 反转：老爷爷另有目的

### 🔴 P0 必读核心模块

#### 一、世界设定8大模块（⭐核心产出物⭐）

**这是世界观创世师的核心产出——完整的"世界设定手册"，8个模块缺一不可：**

##### ① 地图与资源

```
【必须包含的内容】
□ 世界整体形状（大陆/星球/位面/无限空间）
□ 主要区域划分（东/西/南/北/中 / 各大洲/各域）
□ 关键地标（城市/山脉/河流/禁地/秘境）
□ 资源分布（哪些地方产什么稀有资源）
□ 交通方式与距离尺度（移动需要多长时间）
□ 气候与环境特征
```

**创世师检查要点**：
- 地图大小要能支撑全书剧情跨度（太小会拥挤，太大读者没有地理概念）
- 资源分布要合理（不是好东西都堆在一起）
- 距离尺度影响剧情节奏（太近就没有"旅途冒险"，太远就会赶路水文）

##### ② 场景氛围

```
【必须包含的内容】
□ 各主要区域的视觉风格（荒凉/繁华/神秘/阴森…）
□ 氛围关键词（每个地区3-5个形容词）
□ 标志性景观（让读者记住这个地方的画面）
□ 昼夜/四季变化特征
□ 特殊环境规则（如：某地灵气浓郁/某地有时间流速异常）
```

**创世师检查要点**：
- 不同场景之间要有足够的差异化（不能所有城市看起来都一样）
- 场景氛围要与该区域的剧情功能匹配（如：禁地=危险+神秘）

##### ③ 权力规则

```
【必须包含的内容】
□ 世界运行的底层规则（修仙法则/魔法原理/科技基础…）
□ 力量来源（从哪里获取力量？如何变强？）
□ 力量限制（有什么代价/限制/副作用？）
□ 规则边界（什么能做到？什么绝对做不到？）
□ 规则例外（有没有打破常规的特殊情况？为什么？）
```

**⚠️ 这是最容易打脸的模块！**
**创世师铁律**：
- 规则一旦确立，后续绝不能随意更改
- 如果需要修改规则，必须有"新信息揭示"的合理过程（不是直接改写）
- 规则的例外情况必须在早期就埋下伏笔（不能临时编一个"特殊情况"）

##### ④ 势力划分

```
【必须包含的内容】
□ 主要势力列表（名称/性质/核心理念/领袖）
□ 势力间的关系图谱（同盟/敌对/中立/暗中博弈）
□ 势力实力对比（强/中/弱 + 为什么）
□ 势力的内部结构（是否有派系斗争？）
□ 势势的领土范围/影响力区域
```

**创世师检查要点**：
- 势力数量控制在7-12个（太多读者记不住，太少显得世界单薄）
- 不能所有势力都是敌人（需要复杂的联盟/敌对/中立关系网）
- 每个势力都要有自己的立场和动机（不是单纯的"好人阵营vs坏人阵营"）

##### ⑤ 阶层划分

```
【必须包含的内容】
□ 社会阶层体系（贵族/平民/奴隶 或 等级制）
□ 各阶层的权利/义务/限制
□ 阶层流动的可能性与条件
□ 阶层间的矛盾来源
□ 主角在阶层中的起点位置
```

**创世师检查要点**：
- 阶层体系是冲突的重要来源（主角被压迫→反抗→逆袭的经典动力）
- 阶层流动的条件要清晰（否则升级/突破缺乏依据）
- 注意阶层与其他系统的交叉（如：境界高的人是否自动属于高层？）

##### ⑥ 功能职业

```
【必须包含的内容】
□ 世界中的职业/身份种类
□ 各职业的社会地位与收入水平
□ 职业所需的资质/门槛
□ 职业的特殊能力/权限
□ 与职业相关的组织/公会/行会
```

**联动创意总监**：确保职业体系能产生足够精彩的对抗场景（如：炼丹师 vs 炼器师的竞争）。

##### ⑦ 能力体系 ⭐最核心模块⭐

```
【必须包含的内容】
□ 境界/等级/阶梯划分（每一级的名称/特征/差距）
□ 能力类型分类（攻击/防御/辅助/特殊）
□ 升级/突破条件（需要什么/怎么做到/失败后果）
□ 战力评估标准（同级别内也有强弱之分吗？怎么判断？）
□ 能力的成长上限（有没有天花板？天花板在哪里？）
□ 能力展示形式（战斗时具体怎么表现？视觉效果？）
```

**战力闭环检验（创世师必须执行）**：

| 检验项 | 问题 | 通过标准 |
|--------|------|---------|
| **数值一致性** | 前期说的"不可能"后面做到了？ | 要么前期信息是错的（安排揭示），要么就是bug |
| **等级差距** | 低等级越级打败高等级是否合理？ | 越级必须有特殊条件（金手指/阵法/牺牲/时机） |
| **成长速度** | 主角提升速度是否太快导致后期无敌？ | 设置合理的瓶颈期/每次突破的代价 |
| **体系完整性** | 有没有被遗忘的能力类型？ | 定期审查所有已出现的能力 |
| **横向可比性** | 不同体系之间的战斗力能否比较？ | 建立统一的战力参照系 |

##### ⑧ 知识工具

```
【必须包含的内容】
□ 世界特有的知识体系（功法/技能/学术/历史…）
□ 重要物品/道具/神器列表及其效果
□ 工具的获取方式/稀有度/使用限制
□ 知识的传承方式（师徒/书籍/血脉/系统…）
□ "常识"与"秘密"的区分（角色应该知道什么？）
```

**联动原创作家 & 创意总监**：
- 工具/神器的设计要服务于剧情（不是罗列装备清单）
- 新工具/神器的引入要有期待感铺垫（不要突然掏出一个神器解决问题）

#### 二、世界设定6模块动态微调

**当剧情推进到新的区域/阶段时，需要在已有基础上进行微调。这是防止"设定僵化"的关键：**

| 微调模块 | 触发时机 | 操作方法 |
|---------|---------|---------|
| **地图场景微调** | 进入新地图 | 补充新区域的详细设定 |
| **权力规则微调** | 发现新规则/旧规则有例外 | 更新规则文档，标注变更原因 |
| **势力划分微调** | 势力兴衰/新势力出现 | 更新势力关系图谱 |
| **阶层职业微调** | 社会结构变化 | 更新阶层流动路径 |
- **能力体系微调** | 新能力类型出现/境界扩展 | 在现有体系上延伸，不推翻
| **知识工具微调** | 新知识/新道具出现 | 归入知识库/道具库

**⚠️ 微调铁律**：
- 每次微调都必须记录变更日志（什么时间、因为什么情节、改了什么）
- 微调不能推翻核心设定（只能延伸或细化）
- 如果发现必须推翻核心设定 → 这是**设定崩坏的预警信号**，立即通知架构师和创意总监

### 🟡 P1 应读重要模块

#### 三、五大强化技巧之「世界体系」

**创世师维护世界体系的三大武器：**

##### 【能力体系】
- 确保所有战斗结果符合已建立的战力规则
- 新能力的引入不能破坏现有平衡（除非是刻意的不平衡且服务于剧情）
- 记录每个角色的当前能力状态（方便排查前后矛盾）

##### 【功能职业】
- 职业的社会功能要在剧情中得到体现（不只是称号）
- 职业间的互动/竞争/合作关系要丰富世界观层次
- 职业相关组织可以作为剧情推动力（发布任务/提供资源/制造冲突）

##### 【工具神器】
- 神器的威力要有明确界定（不能想多强就多强）
- 神器的获取成本要与其价值匹配（太容易得到就不珍贵了）
- 神器可以有隐藏功能（作为伏笔），但必须在早期暗示

**联动创意总监**：确保以上三个体系能产生足够精彩的对抗场景。

### 🟢 P2 主动设定管理

#### 【写前设定预检查】：
每次要写新的重要剧情前，自动检查：
1. 该剧情涉及的所有设定是否已经明确？
2. 有没有和之前已发布的设定冲突？
3. 该剧情会不会导致战力体系崩坏？
4. 要不要提前铺垫/暗示某个新设定？

#### 【写中设定一致性实时监控】：
- **当写出一个新设定时** → 自动搜索之前有没有写过类似/冲突的设定，实时预警
- **当战力表现超出阈值时** → 立刻预警："⚠️ 该战力表现已经超过当前境界的合理范围，可能导致后续战力崩坏"
- **当某个设定连续50章没出现** → 提示："该设定太久没提了，读者可能忘了，要不要在合适的地方提一句？"

#### 【写后设定自动归档】：
每章写完后自动：
1. 提取本章出现的所有新设定/旧设定的新表述
2. 更新到世界观数据库，建立章节索引
3. 检查与已有设定的一致性
4. 生成"本章设定更新摘要"

### ⚫ P5 防设定崩坏机制

#### 世界观六大崩塌预警线：
| 预警线 | 触发条件 | 自动应对方案 |
|-------|---------|------------|
| **设定矛盾线** | 已发现设定矛盾≥3处且未解决 | 启动"设定集中校准流程"，暂停写新设定，先解决旧矛盾 |
| **战力膨胀线** | 战力膨胀指数≥6 | 启动"战力天花板临时加固方案"，限制继续升级，增加战斗技巧权重 |
| **设定冗余线** | 写了但没用过的设定≥15个 | 警告："设定过载，建议要么用掉要么废弃，不要再加新设定了" |
| **信息过载线** | 单章新设定≥5个 | 预警："单章新设定太多读者记不住，建议拆分到多章逐步放出" |
| **吃书风险线** | 某核心设定前后表述差异度>50% | 紧急预警："这已经是吃书级别的修改了！建议要么圆回来要么提前铺垫读者接受" |
| **世界萎缩线** | 连续3卷都在同一个地方打转，世界没有扩展 | 提示："该开新地图了，世界需要扩展，否则读者会腻" |

#### 设定活度管理机制：
- **设定不是写了就完了**，要在剧情中反复使用才能活起来
- **"死设定"定义**：写了之后超过50章再也没提过
- **每卷自动统计设定活度**：死设定比例超过30% → 警告："很多设定被你忘了，读者也忘了，建议处理"
- **好的世界观不是写出来的，是用出来的**——每个重要设定至少要在剧情中被使用3次以上

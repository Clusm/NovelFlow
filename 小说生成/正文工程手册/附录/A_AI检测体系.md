## 一、24项AI检测词清单

### 阈值说明

- `=0`：该词禁止出现，一旦出现必须修复
- `≤2`：该词在整章中出现次数不得超过2次

### 检测词完整列表（共24项）

#### A类：=0（必须为零）

| 词汇 | 禁出现原因 |
|------|-----------|
| 而是 | 判断性连接词，AI最爱用，破坏叙事自然感 |
| 因而 | 因果连接词，正式书面语，网络小说中不自然 |

#### B类：≤2（顺序/因果词）

| 词汇 | 阈值 | 典型AI用法 |
|------|------|-----------|
| 然后 | ≤2 | 场景描述过度依赖顺序连接 |
| 因此 | ≤2 | 解释性因果，破坏悬念 |
| 所以 | ≤2 | 因果说明过于直白 |
| 于是 | ≤2 | 顺序词，人为制造因果感 |

#### C类：≤2（转折词）

| 词汇 | 阈值 |
|------|------|
| 然而 | ≤2 |
| 但是 | ≤2 |
| 不过 | ≤2 |

| 词汇 | 阈值 |

| 宛如 | ≤2 |

#### E类：≤2（顺序词）

| 词汇 | 阈值 |

## 二、检测原理

### AI写作的三大特征

**1. 过度连接词**
```
AI: 他走进房间，然后坐下来，然后拿起书，然后开始读。
自然: 他走进房间，一屁股坐下，拿起书翻了起来。
```

**2. 过度揣测词**
```
AI: 他似乎很高兴，仿佛中了大奖，似乎有什么秘密。
自然: 他咧着嘴笑，眼睛都眯成了一条缝。
```

**3. 过度判断词**
```
AI: 这是他人生的转折点，是为了证明自己。
自然: 他攥紧了拳头，指关节发白。
```

### 阈值原理

- `=0` 项：一旦出现，几乎可以肯定是AI生成或严重AI化的文字
- `≤2` 项：自然写作中可能出现2次以内，超过2次则AI化严重

## 三、去AI化核心方法

### 方法一：删除连接词

```
❌ 他站起身，然后走到门口，然后推开门，然后走出去。
✅ 他站起身，走到门口，推开门，走了出去。
```

### 方法二：用动作替代因果

```
❌ 他很高兴，所以跳了起来。
✅ 他高兴得原地蹦了起来。
```

### 方法三：感官替代揣测

```
❌ 他似乎很紧张，仿佛心里压着什么。
✅ 他手心直冒汗，攥着衣角不说话。
```

### 方法四：对话替代判断

```
❌ 他很愤怒，认为这是对他的侮辱。
✅ "你算什么东西！"他吼道。
```

### 方法五：扩写稀释

当某个AI词超标但无法直接删除时，用新内容稀释：
```
场景中增加具体动作、环境描写、人物互动
→ AI词在全文中的占比自然下降
```

## 四、快速检测脚本

```python
text = open('ChXX_标题.md').read()
total = len(text.replace('\n', ''))

print(f"总字符: {total}")

zero_words = ['而是', '因而']
for w in zero_words:
    if w in text:
        print(f"❌ {w}: {text.count(w)} (必须=0)")

threshold_words = ['仿佛','微微','然后','最后','然而','但是','因此','所以',
                   '不过','似乎','如同','宛如','首先','其次','再次',
                   '因为','于是','次','不但','不仅','仿若','恰似']
for w in threshold_words:
    c = text.count(w)
    if c:
        status = "❌" if c > 2 else "⚠️" if c == 2 else "⚠️"
        print(f"{status} {w}: {c}")

# 综合判定
ai_score = sum(min(text.count(w), 3) for w in threshold_words)
print(f"\nAI化指数: {ai_score} (>10为严重AI化)")
```

> ⚠️ 脚本中 `threshold_words` 共21词，`zero_words` 共2词，合计检测词23个（=0和≤2分开统计）。完整24词见第一节分类表格。

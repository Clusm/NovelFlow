## 一、九线框架速查

| 序号 | 线名 | 定义 | 核心场景 |
|------|------|------|---------|
| 1 | 成长线 | 实力/境界提升 | 修炼、顿悟、考核 |
| 2 | 情感线 | 与各角色的感情发展 | 相识→暧昧→确认→冲突→升华 |
| 3 | 主线 | 核心目标/任务 | 复仇、揭秘、登顶 |
| 4 | 冲突线 | 与反派的对抗 | 阴谋揭露、正面冲突、最终决战 |
| 5 | 副线 | 辅助剧情 | 支线任务、意外事件 |
| 6 | 势力线 | 派系/阵营关系 | 结盟、背叛、清洗 |
| 7 | 资源线 | 宝物/功法/金钱获取 | 探索、交易、抢夺 |
| 8 | 认知线 | 信息差/认知升级 | 揭秘真相、识破阴谋 |
| 9 | 羁绊线 | 配角成长/配角剧情 | 跟班的崛起、对手洗白 |

## 二、单章九线配置

### 普通章（无重大剧情推进）

| 必选线 | 可选线 | 字数分配 |
|--------|--------|---------|
| 成长线（30%）+ 情感线（30%） | 副线（20%）+ 认知线（20%） | 2600-3000字 |

### 爽章（打脸/逆袭/成就）

| 必选线 | 可选线 | 字数分配 |
|--------|--------|---------|
| 冲突线（50%）+ 成长线（30%） | 资源线（20%） | 3000-4500字 |

### 感情章（单独的感情互动）

| 必选线 | 可选线 | 字数分配 |
|--------|--------|---------|
| 情感线（60%）+ 羁绊线（20%） | 认知线（20%） | 2600-3500字 |

## 三、九线交叉写法

### 交叉原理

两条线同时推进，比单独一条线更有张力。

| 交叉组合 | 效果 |
|----------|------|
| 成长线 + 冲突线 | 边打边升级，打完升一级 |
| 情感线 + 冲突线 | 英雄救美，美人心动 |
| 主线 + 认知线 | 边揭秘边行动，信息就是力量 |
| 资源线 + 冲突线 | 抢夺宝物，场面刺激 |
| 情感线 + 羁绊线 | 配角助攻，助攻变情敌 |

### 示例

```
主线：复仇
情感线：与女主角的感情
冲突线：与反派的对抗

本章结构：
- 主线（复仇）推进：系统提示仇人下落
- 情感线（女主角）：托人送信来关心
- 冲突线（反派）：反派设下新陷阱
```

## 四、九线节奏配置

| 阶段 | 主要推进线 | 辅助线 |
|------|-----------|--------|
| 前期（Ch1-30） | 成长线 + 情感线 | 主线引入 |
| 中期（Ch31-70） | 冲突线 + 势力线 | 成长线 + 情感线 |
| 后期（Ch71-100） | 主线 + 冲突线 | 资源线 + 认知线 |

---

## 五、九线速查清单

```
□ 本章有成长推进（实力/境界/认知）？
□ 本章有情感互动（爱情/亲情/友情）？
□ 本章有冲突升级（与反派的矛盾加深）？
□ 本章有信息揭露（认知提升）？
□ 本章有资源获取（物品/财富/人脉）？
□ 本章埋了哪些伏笔（近伏/中伏/远伏）？
□ 下章计划回收哪个伏笔？
```

## 六、九线与爽点对应

| 爽点类型 | 主要配合线 |
|----------|-----------|
| 打脸 | 冲突线 + 成长线 |
| 逆袭 | 成长线 + 资源线 |
| 装逼 | 成长线 + 认知线（信息差） |
| 收集 | 资源线 + 羁绊线 |
| 感情确认 | 情感线 + 羁绊线 |

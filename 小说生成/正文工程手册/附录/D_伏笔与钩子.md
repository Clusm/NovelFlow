## 一、三层次伏笔体系

### 伏笔三层次

| 层次 | 时间跨度 | 特征 | 示例 |
|------|---------|------|------|
| 近伏 | 1-3章回收 | 简单悬念 | "他不知道门后有人" |
| 中伏 | 5-20章回收 | 情节伏线 | 父亲遗信暗示幕后黑手 |
| 远伏 | 50章+回收 | 主线伏线 | 贯穿全书的终极秘密 |

### 伏笔设置原则

1. **必须回收** — 不回收的伏笔是欺骗读者
2. **提前告知** — 读者看到时要能回想"原来是这样"
3. **自然埋入** — 不能刻意，要融入正常叙事
4. **层层嵌套** — 小伏笔支撑大伏笔

### 伏笔信号词

| 信号词类型 | 示例 | 说明 |
|-----------|------|------|
| 异常描写 | "他的眼神闪了一下" | 暗示心理变化 |
| 物品出现 | "这把刀很旧" | 暗示有来历 |
| 话语暗示 | "以后你会明白的" | 制造悬念 |
| 环境细节 | "月亮是红的" | 暗示不祥 |

## 二、开篇钩子模组

### 五种开篇钩子

#### 1. 冲突型（最强）

```
开局就扔出一个冲突，让读者立刻想知道结果。
```

```
❌ 普通开局：
"今天是个好日子。"他想着。

✅ 冲突型开局：
"你敢！"他吼道，"你要是敢动她一根手指头，我就让你全家陪葬！"

门外的管家愣住了。他跟了老爷三十年，从来没见老爷这么失态过。
```

#### 2. 悬念型

```
开局抛出一个谜题，让读者想知道答案。
```

```
✅ 悬念型开局：
他死的那天，阳光很好。

村里人都说，他是自杀的。但他不这么认为。因为他亲手把那个人的脖子拧断了，那人不可能还活着。
```

#### 3. 画面型

```
开局用一个极具画面感的场景，吸引读者。
```

```
✅ 画面型开局：
血顺着刀刃滴下来，在青石板上晕开一朵一朵的红。

他站在尸体旁边，面无表情，好像刚刚只是杀了一头猪。
```

#### 4. 金句型

```
开局一句有哲理或有力量的话，吸引读者。
```

```
✅ 金句型开局：
知识是天下公器。这句话，他爹跟他说过无数遍。

直到他爹死了，他才知道这句话的分量。
```

#### 5. 逆转型

```
开局先说一个常识，再打破它。
```

```
✅ 逆转型开局：
所有人都说，他是个废物。

十六岁才起步，十九岁还是平庸。身边所有的人都在笑话他。

但他们不知道的是，他脑子里装着的东西，比整个县城的书加起来都多。
```

## 三、章末钩子写法

### 钩子五要素

```
1. 悬念感：让读者想知道接下来发生什么
2. 情绪高点：情绪达到顶峰，让读者无法放下
3. 信息差：读者知道的信息比角色多，或反之
4. 开放式：不给读者喘息的机会
5. 连接感：让读者立刻想读下一章
```

### 钩子类型分类

#### 信息型钩子

```
就在他转身的那一刻，他看到了那个东西。

一个他以为永远不会再见到的名字。
```

#### 逆转型钩子

```
他以为自己是猎人。

他错了。

从一开始，他就是猎物。
```

#### 悬念型钩子

```
"你爹——"她欲言又止。

"我爹怎么了？"他追问。

她摇摇头，眼眶红了。

"你爹不是你的亲爹。"
```

#### 情绪型钩子

```
她的声音越来越远。

他伸出手，想抓住她。

但他抓到的，只有空气。

她走了。

永远地走了。
```

## 四、钩子埋设位置

| 位置 | 效果 | 适用类型 |
|------|------|---------|
| 章末最后一句 | 最强 | 悬念/逆转/情绪 |
| 章末最后三句 | 强 | 悬念+情绪组合 |
| 章末段落 | 中等 | 信息型/过渡 |
| 章中转折处 | 较弱 | 辅助 |

---

## 五、伏笔与钩子配合

好的章节 = 近伏在当章回收 + 中伏/远伏在新埋

```
示例（第N-N+2章衔接）：

第N章埋：系统显示"检测到上古遗迹信号"
第N+1章承：陆续发现其他线索
第N+2章转+合：第一次接触遗迹（近伏回收） + 远伏：新遗迹的具体位置

这样既保证了每章有爽点（近伏回收），
又推进了主线（远伏埋设）。
```

## 六、伏笔回收检查清单

```
□ 本章埋设的近伏（1-3章内）是否计划回收？
□ 之前埋设的中伏（5-20章）是否该回收？
□ 回收时是否有足够的情绪高点？
□ 伏笔和钩子是否配合使用？
□ 读者能否在回收时产生"原来如此"的快感？
```

# 单章写作WF：详细流水线

> 本文档是单章写作的详细执行手册，配合00_主工作流使用。

---

## 执行流程图

```
① 读5份文档
    ↓
② 提取7要素
    ↓
③ 排列场景清单
    ↓
④ 一次性写完初稿
    ↓
⑤ 字数验证（≥2600字）
    ↓
⑥ AI禁词扫描
    ↓
⑦ 有问题？→ patch/扩写 → 重新验证
    ↓
⑧ 达标 → 交付
```

---

## 第一步：读5份文档

### 文档1：细纲（当前章节）

读取当前章节的细纲内容。

提取：
- 7要素：情节/爽点/钩子/冲突/金手指/情绪/伏笔
- 场景顺序
- 字数目标

### 文档2：细纲（卷纲）

读取同一文件中的卷纲部分，确认本章在卷中的位置。

### 文档3：世界观设定

读取世界观设定文件中与本章相关的设定。

### 文档4：前章

读取前一章的最后3段，确认衔接点。

### 文档5：后章（如果已写）

读取后一章的开头，确认悬念走向。

---

## 第二步：7要素自检

| 要素 | 必须？ | 本章计划 |
|------|--------|---------|
| 情节 | ✅ 必须 | 场景1/2/3... |
| 冲突 | ✅ 必须 | 障碍是什么 |
| 爽点 | 建议有 | 打脸/逆袭/收集 |
| 金手指 | 建议有 | 系统展示 |
| 情绪 | 建议有 | 愤怒/感动/热血 |
| 钩子 | ✅ 必须 | 章末悬念 |
| 伏笔 | 建议有 | 近伏/中伏/远伏 |

---

## 第三步：场景清单

在脑中排列场景顺序，用 `---` 分隔：

```
场景1：主角抵达新地点
---
场景2：与重要配角初次相遇
---
场景3：冲突爆发
---
场景4：主角面临考验
---
场景5：章末转折
```

---

## 第四步：一次性长写

**要求：**
- 一次性写完，中间不停顿
- 写完才保存文件
- 初稿目标4000-6000字（留扩写余地）

**写法：**
- 按场景清单顺序写
- 每场景写完用 `---` 分隔
- 章末必须有钩子

---

## 第五步：字数验证

```python
text = open('ChXX_标题.md').read()
total = len(text.replace('\n', ''))
print(f"总字符: {total}")
```

| 结果 | 处理 |
|------|------|
| 2600 ≤ 总字符 ≤ 4500 | ✅ 进入AI禁词扫描 |
| 总字符 < 2600 | ⚠️ 用扩写三维法补足 |
| 总字符 > 4500 | ⚠️ 删减重复描写 |

---

## 第六步：AI禁词扫描

```python
text = open('ChXX_标题.md').read()
for w in ['而是','仿佛','微微','然后','最后','然而','但是','因此','所以',
          '不过','似乎','如同','宛如','首先','次','再次','因而','因为']:
    c = text.count(w)
    if c:
        print(f"⚠️ {w}: {c}")
```

**处理：**
- 有超标项 → 进入第七步patch
- 无超标项 → 进入第八步交付

---

## 第七步：patch/扩写

### Patch规则

```python
# 示例：修复"然后"超标
text = text.replace('然后站起身', '站起身')
text = text.replace('然后转身', '转身就走')
open('ChXX_标题.md', 'w').write(text)
```

### 扩写规则（三维法）

**字数不够时扩写：**
- 感官维度：添加视觉/听觉/嗅觉/触觉细节
- 心理维度：添加回忆/联想/内心独白
- 环境维度：添加场景背景/氛围渲染

**扩写位置：** 选文件中间已有句子作锚点

---

## 第八步：交付检查清单

```
□ 字数：2600 ≤ 总字符 ≤ 4500
□ 而是 = 0
□ 因而 = 0
□ 其他AI词 ≤ 阈值
□ 情节完整
□ 章末有钩子
□ 与前章衔接顺畅
□ 文件已保存
□ 下一章可以开始
```

## 单章写作计时标准

| 步骤 | 标准时间 |
|------|---------|
| 读5份文档 | 5分钟 |
| 脑内构思场景 | 3分钟 |
| 一次性长写 | 30-45分钟 |
| 验证流水线 | 10-15分钟 |
| **总计** | **50-70分钟/章** |

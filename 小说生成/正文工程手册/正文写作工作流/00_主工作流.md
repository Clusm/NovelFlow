## v2.0 主工作流（推荐使用）

```
W1: 写作前对接 → 读v2.0大纲产物 → 生成约束卡
        ↓ 🚪门禁W1
W2: 场景级创作 → 拆场景 → 逐场景匹配wiki知识卡片 → AI写初稿
        ↓ 🚪门禁W2
W3: 章节精修 → 六维打磨（词/句/段/节奏/钩子/情绪）
        ↓ 🚪门禁W3
W4: 全维验证 → 四视角审查（事实/读者/编辑/连载一致性）
        ↓ 🚪门禁W4
W5: 修复闭环 → 修复 → 复审 → 归档
        ↓ 🚪门禁W5
   本章定稿，进入下一章
```

## v1.0 参考：旧版主工作流

```
步骤1：写作前准备（WF-01）
  读取5份文档 → 7要素自检 → 场景清单
                          ↓
步骤2：初稿写作（WF-02）
  一次性长写 → 2600-4500字 → 禁止清单自检
                          ↓
步骤3：质量验证流水线（WF-03）
  字数验证 → AI禁词扫描 → patch修复 → 扩写/删减 → 交付
                          ↓
步骤4：交付检查
  AI禁词全部达标 → 字数达标 → 下一章
```

## 一、验证流水线（5步）

```
步骤1：字数验证  →  步骤2：AI禁词扫描  →  步骤3：patch修复  →  步骤4：扩写/删减  →  步骤5：再验证交付
```

### 步骤1：字数验证

```python
text = open('ChXX_标题.md').read()
total = len(text.replace('\n', ''))
print(f"总字符: {total}")
```

| 标准 | 结果 |
|------|------|
| 2600 ≤ 字数 ≤ 4500 | ✅ 通过 |
| 字数 < 2600 | ⚠️ 需扩写 |
| 字数 > 4500 | ⚠️ 需删减 |

### 步骤2：AI禁词扫描

```python
for w in ['而是','因而','仿佛','微微','然后','最后','然而','但是','因此','所以',
          '不过','似乎','如同','宛如','首先','其次','再次','因为','于是',
          '次','不但','不仅','仿若','恰似']:
    c = text.count(w)
    if c:
        print(f"⚠️ {w}: {c}")
```

| 词汇 | 阈值 | 扫描后 |
|------|------|--------|
| 而是 | =0 | 必须是0 |
| 因而 | =0 | 必须是0 |
| 仿佛/微微/似乎/如同/宛如/仿若/恰似 | ≤2 | 全部项 |
| 然后/最后/然而/但是/因此/所以/不过/于是 | ≤2 | 全部项 |
| 首先/其次/再次/因为 | ≤2 | 全部项 |
| 次 | ≤2 | 量词用法可接受 |
| 不但/不仅 | ≤2 | 递进词 |

### 步骤3：patch修复

发现超标项后，用 `patch` 或 `execute_code` 修复。

**修复优先级：**
1. 先修复 `=0` 项（必须清零）
2. 再修复超标项（超过阈值的）
3. 最后处理边缘项（=1或=2的）

**修复原则：**
- 对话中的用法通常更自然，可保留
- 叙事中的连接词必须修复
- 扩写场景时用新内容稀释超标词

### 步骤4：扩写/删减

**字数不够时（<2600）：** 用扩写三维法补足
**字数超量时（>4500）：** 删减重复描写和冗余对话

### 步骤5：再验证

```python
# 快速验证脚本
text = open('ChXX_标题.md').read()
total = len(text.replace('\n', ''))
issues = []
for w in ['而是','仿佛','微微','然后','最后','然而','但是','因此','所以','不过',
          '似乎','如同','宛如','首先','次','再次','因而','因为']:
    c = text.count(w)
    if c > 2: issues.append(f"{w}:{c}")
print(f"字数: {total}, 问题: {issues if issues else '无'}")
```

## 二、扩写三维法（字数不够时）

### 维度一：感官描写

添加视觉、听觉、嗅觉、触觉、味觉细节。

```
原文：夜很静。他觉得很安心。
扩写：夜很静。虫鸣声从草丛里传出来，远处有狗吠声。月亮升得老高，把院子照得一片银白。他抱着膝盖坐在门槛上，凉风吹过来，带着泥土的气息。他忽然觉得很安心。
```

### 维度二：心理活动

用回忆、联想、内心独白填充。

```
原文：她比上次瘦了。
扩写：她比上次瘦了。他记得第一次见她的时候，她的脸还是圆圆的，笑起来有两个酒窝。现在她的脸颊尖了，下巴也尖了，眼睛显得更大。他心里一阵疼，想问她这半年过得怎么样，又不知道该怎么开口。
```

### 维度三：环境描写

添加场景背景、氛围渲染。

```
原文：他走进一处陌生的大门。
扩写：他走进一处陌生的大门。高墙黑瓦，门楼上挂着两个大红灯笼，在阳光下显得格外喜庆。门楣上挂着一块匾，是正楷，笔力遒劲，一看就是名家手笔。门口站着两个穿青衣的仆从，腰板挺得笔直，眼神里带着几分审视。他整了整衣冠，报上名字，仆从进去通报。他站在门口等着，手心有些出汗。
```

## 三、AI禁词修复对照表

### 高频问题词修复方案

| 问题词 | 修复方式 | 示例 |
|--------|---------|------|
| 然后 | 删掉或用动作替代 | "然后他走进门" → "他走进门" |
| 仿佛 | 删掉或直接写感官 | "仿佛刀子一样" → "像刀子一样"或删掉 |
| 微微 | 换为其他词 | "微微点头" → "轻轻点头"或"点了点头" |
| 然后 | 动作替代 | "然后坐下" → "他一屁股坐下" |
| 所以 | 因果行为替代 | "所以他很高兴" → "他高兴得原地跳了起来" |
| 最后 | 量词用法保留，叙事用法改 | "最后他走了" → "临走时" |
| 然而 | 删掉或改句式 | "然而他不知道" → "他不知道的是" |
| 但是 | 改为"只是" | "但是他很穷" → "只是他很穷" |

### Patch操作规范

**必须唯一old_string：** 选取至少前后各3个字符，确保唯一性。
**修复后立即验证：** 每patch一次，立刻运行扫描确认变化。

```python
# 正确做法
text = text.replace('他坐下，然后站起来', '他坐下，站起身')
open('file.md', 'w').write(text)
```

## 五、流水线失败处理

| 失败情况 | 处理方式 |
|----------|---------|
| patch后字数减少过多 | 先扩写再patch |
| AI词超标太多（>10处） | 重新写该段落 |
| 字数一直不够（<2000） | 重新写整章 |
| 修复后产生新问题 | 重新扫描并修复 |

## 一、AI禁词超标修复

### 修复优先级

```
第1优先级：=0项（而是、因而）必须清零
第2优先级：超标项（出现次数 > 阈值）
第3优先级：边缘项（出现次数 = 阈值）
```

### 高频超标词修复对照

| 问题词 | 阈值 | 常见出现场景 | 修复方案 |
|--------|------|------------|---------|
| 然后 | ≤2 | 叙事顺序 | 删除，或用具体动作替代 |
| 微微 | ≤2 | 表情描写 | 改为"轻轻"/"浅浅"/"淡淡" |
| 仿佛 | ≤2 | 比喻 | 删除，直接写感官 |
| 所以 | ≤2 | 因果解释 | 用行为替代因果说明 |
| 最后 | ≤2 | 叙事顺序 | 改为"临走时"/"结束时" |
| 然而 | ≤2 | 转折 | 改为"只是"/"没想到" |
| 但是 | ≤2 | 转折 | 改为"只是"/"不过" |
| 因为 | ≤2 | 解释原因 | 删除解释，直陈结果 |
| 次 | ≤2 | 量词用法 | 确认是量词则保留 |
| 似乎 | ≤2 | 揣测 | 改为确定语气 |

### 批量修复脚本

```python
text = open('ChXX_标题.md').read()

# 批量修复
replacements = [
    ('然后站起身', '站起身'),
    ('然后转身', '他转身'),
    ('他坐下，然后站起来', '他站起身'),
    ('嘴角微微扬起', '嘴角轻轻扬起'),
    ('眉头微微一皱', '眉头轻轻皱起'),
    ('仿佛', ''),  # 直接删除（谨慎使用）
    ('然而', '只是'),
    ('但是', '只是'),
]

for old, new in replacements:
    count = text.count(old)
    if count:
        print(f"修复: {old} → {new} ({count}处)")
        text = text.replace(old, new)

open('ChXX_标题.md', 'w').write(text)
print("✅ 修复完成")
```

### patch操作规范

1. **必须确保old_string唯一** — 选取前后各3个字符
2. **修复后立即验证** — 每patch一次，立刻扫描
3. **大段patch用execute_code** — 避免工具截断

```python
# execute_code patch示例
text = open('Ch1_新标题.md').read()
old = '''他心里一沉，仿佛有什么东西压在胸口，让他喘不过气来。'''
new = '''他心里一沉，像是被什么攥住了喉咙。'''
text = text.replace(old, new)
open('Ch1_新标题.md', 'w').write(text)
```

## 二、字数不够扩写

### 扩写三维法

**维度一：感官描写**
```python
# 原文
夜很静。他觉得很安心。

# 扩写
夜很静。虫鸣声从草丛里传出来，远处有狗吠声。月亮升得老高，把院子照得一片银白。他抱着膝盖坐在门槛上，凉风吹过来，带着泥土和青草的气息。远处的山峦黑黢黢的，像是一头躺下的大牛。他深吸一口气，觉得心里很踏实，很安心。
```

**维度二：心理活动**
```python
# 原文
她比上次瘦了。

# 扩写
她比上次瘦了。他记得第一次见她的时候，她的脸还是圆圆的，笑起来有两个浅浅的酒窝，眼睛亮晶晶的，像是装着两颗星星。现在她的脸颊尖了，下巴尖了，眼睛显得更大更深。他心里一阵疼，想问她这半年过得怎么样，想问她有没有被人欺负，想问她有没有想他。但话到嘴边，又咽了回去。
```

**维度三：环境/氛围**

```
原文：他走进一处陌生的大门。
扩写：他走进一处陌生的大门。高墙黑瓦，门楼上挂着两个大红灯笼，在阳光下显得格外喜庆。门楣上挂着一块匾，是正楷，笔力遒劲。门口站着两个穿青衣的仆从，腰板挺得笔直。他整了整衣冠，报上名字，仆从进去通报。他站在门口等着，手心有些出汗。
```

### 扩写位置选择

| 位置 | 适合扩写内容 |
|------|------------|
| 章首 | 钩子展开、环境描写 |
| 章中 | 冲突展开、心理活动 |
| 章末 | 悬念展开、情绪高点 |
| 场景切换处 | 环境过渡、心理独白 |

### 扩写后验证

```python
text = open('ChXX_标题.md').read()
total = len(text.replace('\n', ''))
print(f"扩写后字数: {total}")
# 确认≥2600
```

## 三、字数超量删减

### 删减原则

1. **删重复描写** — 同一个画面出现两次
2. **删冗余对话** — 说过的话不再重复
3. **删解释性文字** — "这是因为……"类解释
4. **合并场景** — 两个相似场景合并

### 删减检查点

```python
# 检查重复段落
lines = text.split('\n')
from collections import Counter
# 找重复的连续3行
for i in range(len(lines)-2):
    segment = '\n'.join(lines[i:i+3])
    if text.count(segment) > 1:
        print(f"重复段落: 行{i+1}-{i+3}")
```

## 四、章节错乱诊断与修复

### 时间线错乱

**现象：** 后章节的时间反而早于前章节

**诊断：**
1. 提取所有章节的日期/时间关键词
2. 按章节顺序排列
3. 检查是否有倒退

**修复：** 调整章节顺序，或修改时间描述

### 编号错乱

**现象：** 文件名编号与内容不对应

**诊断：**
```python
# 检查ChN文件内容是否确实是第N章
import re
for i in range(1, 100):
    filename = f'Ch{i}_*.md'
    # 提取文件中的"第N章"
    # 匹配则为正确，不匹配则有问题
```

### 内容覆盖

**现象：** 两个章节写到同一个文件里

**诊断：** 文件特别大（>8000字），或出现两个"第X章"标题

**修复：** 按场景分隔符`---`拆分，分别验证

## 五、patch失败处理

| 失败情况 | 原因 | 解法 |
|----------|------|------|
| old_string不唯一 | 多次出现相同文字 | 加前后文确保唯一 |
| patch后字数骤降 | 误删重要段落 | 重新读文件，从备份恢复 |
| 反复patch总失败 | 段落结构有问题 | 整段重写 |
| AI词清不干净 | 替换方式不对 | 用新内容稀释，而非替换 |

## 六、快速修复脚本模板

```python
text = open('ChXX_标题.md').read()
original_len = len(text.replace('\n', ''))

# AI词修复
fixes = [
    ('然后', ''),
    ('微微', '轻轻'),
    ('仿佛', ''),
]
for old, new in fixes:
    text = text.replace(old, new)

# 扩写（插入到已知锚点）
anchor = '她比上次瘦了。'
insert = '\n\n他心里一阵疼，想问她这半年过得怎么样，又不知道该怎么开口。'
if insert not in text:
    text = text.replace(anchor, anchor + insert)

open('ChXX_标题.md', 'w').write(text)
final_len = len(text.replace('\n', ''))
print(f"修复前: {original_len}字 → 修复后: {final_len}字")
```

# WF-06-W1：写作前对接

> 正文工程手册 v2.0 · 写作前对接阶段
> 上一阶段：大纲定稿（大纲手册 Phase 7 通过） | 下一阶段：[W2 场景级创作](WF-06_W2_场景级创作.md)
> 版本：v2.0 | 最后更新：2026-04-30

---

**目标**：从v2.0大纲产物中提取本章所有前提条件，生成一份"写作约束卡"——AI写本章前的必读文件。

**AI角色**：信息提取 + 约束组装（不参与创作）

**耗时**：约2分钟

**AI使用的wiki知识**：
- `倒果为因写作法` — 确认本章落点
- `开篇钩子模组` — 确认开头钩子公式
- `伏笔设计体系` — 标注埋设/回收操作
- `AI大纲一致性管理` — 状态约束注入
- `去AI化写作` — 禁区词预标注

---

## Step 1.1：读取v2.0大纲产物

AI写作本章前，按以下顺序读取5份v2.0大纲文件：

| 序号 | 读取文件 | 提取内容 |
|------|---------|---------|
| 1 | `06_第N卷细纲_ChXX.md` | 本章11字段：落点/倒推铺垫/开头钩子类型+内容/四段情节点(起承转合)/章末钩子类型+内容/伏笔操作/爽感类型/去AI味预标注/状态追踪核对项 |
| 2 | `04_第N卷_剧情块.md` | 本章所属剧情块（块名/块内位置/核心事件）、本章在块中处于建置期/推进期/高潮期/收束期 |
| 3 | `05_第N卷_状态追踪.md` | 主角状态快照（实力/身体限制/心理/位置/金手指/可用资源）、女主/反派状态、本卷约束清单、需回收的旧伏笔 |
| 4 | `02b_世界观核心设定.md` | 相关设定引用（力量体系等级/金手指规则/势力信息/世界观铁律） |
| 5 | `正文/第N卷/Ch(N-1)_xxx.md` | 前章结尾（读取最后5段，提取衔接点：时间/空间/情绪状态）+ 提取结尾特征签名（最后3句话原文、钩子类型、结尾句式结构模式、结尾关键意象词），写入约束卡"前章结尾特征"，供W4做相邻章节结束语去重 |

---

## Step 1.2：AI生成「本章写作约束卡」

AI根据以上5份输入，自动组装生成约束卡。

**产出物**：每章开始前生成一份，可保存为独立文件或直接注入AI上下文。

### 约束卡模板

```markdown
# ChXX：[章节标题] — 写作约束卡

## 1. 本章落点（来自细纲 + 倒果为因）
读者看完这章最想记住的画面：[从细纲"本章落点"字段提取]

## 2. 本章在剧情块中的位置（来自剧情块）
块名：[XXX] | 块总章数：N章 | 本章是第X/N章
当前阶段：[建置期 / 推进期 / 高潮期 / 收束期]
本阶段要求：[建置期=铺垫蓄势、推进期=冲突升级、高潮期=事件解决+爽点爆发、收束期=情绪回落+钩子]

## 3. 本章关键约束（来自状态追踪 + 世界观）
- ⚠️ 主角当前身体限制：[从状态追踪读取，如"右臂受伤未愈，ChXX前不能写右手全力战斗"]
- ⚠️ 主角当前境界：[从状态追踪读取]
- ⚠️ 主角当前位置：[从状态追踪读取]
- ⚠️ 金手指限制：[从状态追踪读取，如"顿悟系统每日触发上限3次"]
- ⚠️ 世界观铁律：[从02b读取，如"秀才境界不可御空飞行"]
- ⚠️ 锁死的设定：[本章涉及的势力/人物关系/物品等已确定的设定]

## 4. 开头钩子（来自细纲）
类型：[冲突前置 / 悬念前置 / 人设前置]
具体内容：[从细纲读取]
选择理由：[从细纲读取]

## 5. 本章情节点（来自细纲）
起（约500字）：[从细纲读取，具体写什么]
承（约800字）：[从细纲读取，具体写什么]
转-高潮（约1000字）：[从细纲读取，具体写什么 + 爽感类型 + 写法要点 + 旁观者反应层次设计]
合（约500字）：[从细纲读取，具体写什么]

## 6. 章末钩子（来自细纲）
类型：[反转式 / 选择式 / 悬念式 / 情感式]
具体内容：[从细纲读取]

## 7. 本章伏笔操作（来自细纲 + 伏笔设计体系）
### 本埋设
| ID | 内容 | 承诺回收章 |
|----|------|----------|
| F0xx | [从细纲读取] | ChXX |

### 本回收
| ID | 内容 | 回收方式 |
|----|------|---------|
| F0xx | [从细纲读取] | [从细纲读取] |

### 即将到期提醒（来自状态追踪）
| ID | 内容 | 剩余章数 | 状态 |
|----|------|---------|------|
| F0xx | [内容] | N章 | ⚠️请在本章或下章回收 |

## 8. 去AI味预标注（来自细纲 + 去AI化写作）
- 🚫 本章禁用词：[从细纲读取，如"璀璨""心中涌起一股""眼眸中闪过"]
- 🚫 句式提醒：[从细纲读取，如"诗会场景禁止使用'首先其次最后'"]
- 🚫 通用禁区始终生效：[24项AI禁词 + "嘴角勾起一抹冷笑""不由得""蓦然回首"]

## 9. 前章衔接点（来自正文）
- 前章结尾的时间/空间/情绪状态：[从Ch(N-1)最后5段提取]
- 前章章末钩子：[从Ch(N-1)提取]
- 本章必须承接的点：[如"前章结尾主角刚踏入城门，本章开头必须在城市场景"]

## 10. 前章结尾特征签名 ⭐新增（供W4结束语去重比对）
- 最后3句话原文：[从Ch(N-1)逐句提取]
- 结尾句式结构模式：[如"动作句→环境句→留白句" / "对话收束→内心独白→画面定格"]
- 结尾钩子类型：[反转式/选择式/悬念式/情感式/信息差式]
- 结尾关键意象词：[提取1-3个标志性词汇，如"门/夜色/脚步声"]
- ⚠️ 本章结尾禁止：与上章结尾使用相同的句式结构模式 + 相同的关键意象词组合（如不可连续两章都以"他看着她，XXX"句式收尾）

## 11. 字数目标
初稿目标：[默认2600-4500字，特殊剧情章可延至5000字]
```

## Step 1.3：约束卡完整性自检

AI生成约束卡后，逐项检查：

| 检查项 | 自检 |
|--------|------|
| 落点字段是否从细纲成功提取？ | □ |
| 状态约束（身体限制/境界/位置/金手指限制）是否从状态追踪成功填充？ | □ |
| 开头钩子类型+内容是否从细纲成功提取？ | □ |
| 四段情节点是否完整（起承转合都有内容）？ | □ |
| 章末钩子类型+内容是否从细纲成功提取？ | □ |
| 伏笔操作（埋设+回收）是否完整记录？ | □ |
| 去AI味预标注是否从细纲成功提取？ | □ |
| 前章衔接点是否从Ch(N-1)成功提取？ | □ |
| 前章结尾特征签名（最后3句话/句式模式/钩子类型/关键意象词）是否成功提取？ ⭐新增 | □ |
| 状态追踪核对项（□主角状态一致 □无违反铁律 □人设无OOC）是否从细纲成功提取？ | □ |

## 🚪 门禁W1：约束卡评审

**谁来审**：作家本人（或AI辅助快速扫描）

| # | 检查项 | 不通过的表现 |
|---|--------|------------|
| 1 | 本章落点清晰吗？ | 看了不知道这章要达成什么效果 |
| 2 | 状态约束是否完整？ | 主角有伤但约束卡没标注 |
| 3 | 情节点四段都有具体内容吗？ | 某段是空的或只有一句"待定" |
| 4 | 前章衔接流畅吗？ | 本章开头和前章结尾明显断裂 |
| 5 | 伏笔操作（埋+回）都有记录吗？ | 细纲标注了伏笔但约束卡没列出 |
| 6 | 去AI味预标注到位吗？ | 本章明显会触发某个禁区词但没标注 |

**不通过怎么办**：回到对应的大纲文件补充缺失信息，重新生成约束卡。

**通过标准**：全部 ✅ → 进入 [W2 场景级创作](WF-06_W2_场景级创作.md)。

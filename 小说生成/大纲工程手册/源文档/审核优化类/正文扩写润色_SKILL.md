# 网文章节扩写润色 workflow

## 触发条件
网文初稿章节字数不足（<2500字/章目标），需要扩写润色时使用。

## 核心概念

**扩写 ≠ 重写**：保留核心情节和节奏，只在关键节点补充细节。
**润色 = 去AI化 + 扩写**：在扩写基础上清理AI痕迹、替换机械表达。

## 工具准备
`去AI化写作.md`（知识库路径：`~/ainovel/wiki/concepts/去AI化写作.md`）——现已完成全面扩充，包含：AI检测原理（6大维度）、高频禁区词表（形容词/连接词/情绪表达三类）、句式改造4步法、逻辑层面改造、情感物化锚点表、五感写作、十二组实战清理对比示例。扩写前务必先读一遍。
- `叙事技巧体系.md`（知识库路径：`~/ainovel/wiki/concepts/叙事技巧体系.md`）

## 标准扩写操作步骤

### 第1步：加载原文
用 `read_file` 读取初稿，测量当前字数（`wc -m`）。

### 第2步：逐章扩写
每章按以下维度补充（不需要全部用，选适用的）：

**感官细节**（最优先）：
- 嗅觉：烟味/汽油/汗水/花香/霉味/炸串香
- 触觉：铁锈/黏糊/凉意/烫/扎/硌
- 味觉：泡面调料包/廉价啤酒/烟的苦
- 听觉：喇叭声/键盘声/蝉叫/脚步

**环境深度**：
- 场景中的次要人物（小贩/老人/孩子）
- 天气/光线/时间（光线角度决定情绪）
- 物品细节（窗帘/门把手/地上的裂缝）

**心理深度**：
- 主角沉默时的内心想法（不直接说"他很悲伤"，用动作/眼神/呼吸暗示）
- 回忆片段（前世vs今生对比）
- 计算/盘算（数字/时间线/资产）

**情绪钩子强化**：
- 章节结尾留悬念或情绪高点
- 避免"他心想/他发誓/他决定"等直白表述，改用无声动作

### 第3步：替换AI痕迹
对照 `去AI化写作.md` 中的"AI高频禁区词表"，逐一替换：

| 禁区词 | 替换建议 |
|--------|----------|
| 蓦然回首 | 删除或改写 |
| 不由得/不由自主/下意识 | 删掉，保留动作本身 |
| 此刻/与此同时 | 删掉或用具体时间/空间代替 |
| 眼眸中闪过 | 用具体眼神描写替代 |
| 嘴角微微上扬 | 用"笑了一下/笑了声/笑容很淡"替代 |
| 心中暗暗想着 | 直接删掉，让行动说话 |
| 轻轻地/缓缓地/渐渐 | 删掉"地"，保留动作 |
| 微微一笑/轻轻落在 | 改为具体动作描写 |
| 微微皱起/缓缓抬起 | 用更真实的反应替代 |

### 第4步：铁律检查
- **删"像X一样"比喻**：AI爱用"像一只xxx""仿佛xxx"，检查并删除
- **删解释尾巴**：句子里有"因为/所以/虽然/但是"连词且是解释性内容的，压缩或删除
- **删过度心理描写**：超过3行的连续内心独白，用无声动作替代

### 第5步：字数验证
`wc -m Ch*.md` 确认每章达到目标（2500-4000字）。

## 文件命名规范
```
正文/第X卷/ChXX_章标题.md
例：/home/tao/小说流/她还在读大三_万亿帝国/正文/第一卷/Ch01_干杯祝你们百年好合.md
```

## 已知坑
- 扩写后仍需保留情节核心，不能因加细节而拖慢节奏
- 环境描写不超过3行，要精准（一个物品/一个动作/一个声音）
- 心理描写用具体意象替代，不要直接说感受

# 附录B：核心概念术语表

## 大纲结构术语

| 术语 | 定义 |
|------|------|
| **细纲** | 章节级别的详细大纲，含7要素（情节/冲突/爽点/钩子等） |
| **分卷大纲** | 各卷的整体结构概述，通常含三幕/四幕分界 |
| **主大纲** | 全书章节总览列表 |
| **7要素 | 每个章节必须包含的7个结构要素 |
| **名场面** | 传唱级经典场景，用【】标记 |

---

## 7要素详解

| 要素 | 说明 | 示例 |
|------|------|------|
| **情节主线** | 5个情节点（含起承转合） | 主角穿越→发现系统→第一次战斗→ |
| **金手指应用** | 本段金手指如何触发/升级 | 系统首次激活，获得新手大礼包 |
| **冲突对手** | 对手是谁+筹码+弱点+对策 | 反派张三大少，筹码是家族势力，弱点是轻敌 |
| **爽点位置** | 第X段+技巧类型 | 第3段：打脸原话+表情变化+围观反应 |
| **情绪价值** | 主导情绪+爽感模组 | 狂喜/碾压/甜蜜/复仇 |
| **伏笔埋设** | 短线/中线/长线伏笔 | 短线：神秘人物出场；长线：身世之谜 |
| **章末钩子** | 留下什么悬念 | 下一章揭晓真相 |

---

## 爽感类型分类

| 类型 | 说明 |
|------|------|
| **规则碾压** | 理科知识降维打击 |
| **打脸** | 当场还击，不过夜 |
| **传承/代际** | 古人认可，跨越时空 |
| **势力扩张** | 小弟变多，势力变大 |
| **名场面** | 经典复刻/诗词引发天象 |
| **甜宠** | 感情线撒糖 |
| **逆袭** | 草根战胜天才 |
| **知识融合** | 跨学科组合创造奇迹 |
| **学术声望** | 积累被认可 |

---

## 伏笔类型

| 类型 | 回收标准 |
|------|---------|
| **核心伏笔** | 50章内回收，影响主线 |
| **单元伏笔** | 本卷结尾前回收，单卷内 |
| **氛围伏笔** | 全文结尾前回收，增加神秘感 |
| **关键伏笔** | 150章内回收（身世/系统来源/文道断层） |

---

## 质量评级

| 评级 | 说明 |
|------|------|
| **P0** | 影响留存/完读率，必须立即修复 |
| **P1** | 影响精品率，30天内修复 |
| **P2** | 锦上添花，可选修复 |
| **S级** | 历史级名场面/高潮 |
| **A/B/C级** | 爽点密度评级 |

---

## 问题类型

| 问题类型 | 典型表现 |
|---------|---------|
| **S级过密** | 某10章区间有5+个S级，读者审美疲劳 |
| **节奏断层** | 连续S级无缓冲，或持续压抑>5章 |
| **伏笔断裂** | 埋设后未回收 |
| **章节重复** | 同一章节号出现2-3个副本 |
| **时间线矛盾** | 年份倒退或事件顺序错乱 |
| **格式不统一** | 各卷字段名不一致 |

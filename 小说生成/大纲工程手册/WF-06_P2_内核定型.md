## Step 2.1：AI生成「题材流派定位书」

**AI使用的wiki知识**：
- `传统升级流vs番茄风` — 节奏/结构/平台匹配
- `系统流vs面板流vs加点流` — 金手指定位
- `扮猪吃虎流vs穿书碾压流` — 爽感机制选择
- `修仙2.0` / `克苏鲁诡异流` / `加点面板流` — 各流派核心特征
- `2025网文十大关键词` — 市场趋势对齐
- `新人选平台指南` — 平台策略

**产出物**：`02a_题材流派定位书.md`

**必须包含的章节**：

1. **流派三角定位**：核心流派 + 目标平台 + 读者画像的三角图
2. **流派特征诊断表**：从wiki提取该流派的标准特征，逐项标注"采用/不采用/调整"，并给出调整说明
3. **差异化分析**：与同类型典型做法的对比表
4. **平台匹配策略**：目标平台的核心指标 + 本书对位 + 差距 + 应对策略
5. **2025市场趋势对齐**：十大关键词中与本书关联度评估 + 可融入方向

## Step 2.2：AI生成「核心创作纲领」

**AI使用的wiki知识**：
- `爽感四大基石` — 核心爽点类型判定
- `读者情绪地图` — 目标读者群情绪对齐
- `读者心理学` — 底层心理机制（期待感操控/代入感破坏/叙事闭合渴望）
- `倒果为因写作法` — 全书级落点设计
- `网文九线写法` — 多线预判

**产出物**：`01_核心创作纲领.md`

**必须包含的章节**：

1. **一句话卖点**（20字以内）
2. **基调关键词**（3-5个形容词）
3. **核心爽点类型与权重**：主类型(40%) + 辅类型1(30%) + 辅类型2(20%) + 点缀(10%)，含分布密度要求
4. **读者情绪预设目标**：驱动追读的四种核心情绪（紧张感/期待感/反差感/代入感）+ 读者渴望 → 情绪层级 → 本书满足方式 → 体现位置
5. **读者心理学策略**：利用叙事闭合渴望设计悬念链（每3-5章释放部分信息但不完全揭露）；利用虚拟危险的安全边界制造紧张而不致郁；设计期待感操控——"读者知道主角有底牌但还没亮"
6. **全书三级落点**（倒果为因·全书级）：终极落点 + 每卷大高潮落点画面
7. **九线预判**：9条线的全书规划 + 关键节点的预分配
8. **创作禁区**（至少3条）

## Step 2.3：AI生成「人物体系设计」⭐新增

**AI使用的wiki知识**：
- `人物塑造方法` — 五层人物构建法（表层/能力层/性格层/动机层/弧光层）+ 人物塑造三要素（欲望-能力-缺陷）
- `详细人设模板` — 三维模型（生理层/社会层/心理层）+ 核心人设公式（初始身份×反差×核心执念）

**产出物**：`02c_人物体系设计.md`

**必须包含的章节**：

1. **主角五层构建**：
   - 表层：外貌特征 + 标志性动作 + 口头禅（至少2个）
   - 能力层：初始实力等级 + 擅长领域 + 天赋方向 + 弱点/短板
   - 性格层：核心性格（3个形容词）+ 表层性格（别人眼中的TA）+ 深层性格（真实的TA）+ 核心价值观
   - 动机层：核心欲望（最想要什么）+ 最大恐惧（最怕失去什么）+ 行为动机链条
   - 弧光层：起点状态 → 关键转折事件(1-3个) → 变化过程(抗拒→动摇→反复→接受) → 终点状态

2. **核心配角功能设计**（≥4个配角）：
   | 角色名 | 功能定位 | 与主角关系 | 性格标签(3个) | 核心欲望 | 弧光简要 |
   |--------|---------|-----------|-------------|---------|---------|

3. **反派体系**：
   - 主要反派(≥2个)：动机必须合理——反派是自己故事中的"主角"
   - 反派的五层构建（简版：表层/能力层/性格层/动机层）
   - 反派与主角的对抗升级阶梯（每一阶段抗什么）

4. **人物关系网络**：
   - 关系钩子设计（师徒/兄弟/主仆/恋人/仇敌中至少3类）
   - 关系变化时间线（初识→冲突→和解/决裂→最终关系）

5. **人物弧光路径图**（主角+核心配角）：
   - 每卷的角色变化目标
   - 推动弧光的关键场景分布

## Step 2.4：AI生成「世界观核心设定」

**AI使用的wiki知识**：
- `08-网文世界观创世师` — 世界观设计框架
- `金手指设计原理` — 系统/金手指的限制、代价
- `加点面板流` — 属性面板设计原则（若选面板流）
- `系统流vs面板流vs加点流` — 金手指定位

**产出物**：`02b_世界观核心设定.md`

**必须包含的章节**：

1. **力量体系**：等级表（名称+核心能力+社会地位+突破条件+典型寿命）
2. **金手指/核心能力**：名称/来源/觉醒时机/核心规则(3条)+限制代价(≥2条)/进化路径表
3. **属性面板设计**（若选面板流）：属性表（名称+简称+作用+初始值+上限）
4. **世界格局**：结构选择 + 势力表(≥3个，含定位+利益+与主角关系+内部矛盾) + 核心社会矛盾
5. **核心铁律**（AI大纲生成时绝对不可违反，≥3条）

## 🚪 门禁2：内核评审

| # | 检查项 | 不通过的表现 |
|---|--------|------------|
| 1 | 一句话卖点有吸引力？ | 自己都无感 |
| 2 | 流派特征诊断表无"假采用"？ | 选了但没真正用上的特征 |
| 3 | 差异化够明显？ | 和同类作品几乎一样 |
| 4 | 主角五层构建完整吗？ | 动机层/弧光层空白或敷衍 |
| 5 | 核心配角有独立动机和弧光吗？ | 配角纯工具化 |
| 6 | 反派动机合理吗？ | 反派纯坏，无内在逻辑 |
| 7 | 力量体系有层次感吗？ | 等级之间读者无法感知区别 |
| 8 | 金手指有限制吗？ | 没有限制 = 没有冲突可能 |
| 9 | 核心铁律够硬吗？ | 太软，不足以防止AI跑偏 |

**不通过怎么办**：回到对应 Step 修改，直到全部 ✅。

**通过标准**：全部 ✅ → 进入 [Phase 3](WF-06_P3_分卷骨架.md)。
